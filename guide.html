📖 Quick Start Guide

🎯 How to Use This Website

This website is designed to teach optimization algorithms through hands-on experiments. Here's how to get the most out of it:

1️⃣ Browse Experiments
Start at the Home Page:
Click on any course outcome (CO1-CO5) section to see experiments.
Choose Your Level:
CO1-CO2: Beginner-friendly foundational methods
CO3-CO4: Intermediate techniques
CO5: Advanced optimization methods
2️⃣ Read an Experiment

Each experiment page contains these sections:

📌 Aim:
Describes what you'll learn and what problem you're solving
📝 Description:
Explains the concept in plain language
🔧 Algorithm:
Lists the step-by-step procedure
📐 Formulas:
Shows the mathematical equations you'll use
📊 Mathematical Procedure:
Works through a complete example calculation
💻 Python Program:
Provides working, easy-to-understand code
📤 Expected Output:
Shows what results you should get
3️⃣ Run the Code
Install Dependencies:
pip install numpy scipy scikit-learn cvxpy pulp
Copy the Code:
Select and copy the Python code from the program section
Run in a Terminal or IDE:
Paste into Python, Jupyter, or your favorite IDE and run
Observe the Output:
Compare with the expected output section
4️⃣ Experiment & Learn
Modify Parameters:
Try different learning rates, initial values, or datasets
Work Through Math:
Manually calculate one iteration to verify understanding
Add Visualizations:
Use matplotlib to plot convergence or optimization surfaces
Implement Variations:
Try different algorithms on the same problem

💡 Tips for Success
1. Start Simple: Understand gradient descent before moving to complex methods
2. Don't Skip the Math: Work through calculations - it reinforces understanding
3. Run Every Code Example: Seeing working code is different from reading it
4. Modify and Experiment: Change one parameter and observe the effect
5. Create Visualizations: Plot functions, convergence curves, and optimization paths
6. Connect Concepts: Notice how different algorithms relate to each other
❓ Common Questions
Q: What if I don't understand the math?
A: Start with the Algorithm section (step-by-step), then read Mathematical Procedure (worked examples).
Q: Can I run the code without installing libraries?
A: Some basic experiments yes (like Exp 1), but most require numpy. Try Google Colab for free online Python!
Q: Should I memorize all the formulas?
A: No! Focus on understanding concepts. The formulas are references you can look up.
Q: In what order should I do the experiments?
A: Follow CO1 → CO5 progression. They build on each other conceptually.
Q: How long does each experiment take?
A: 30-60 minutes for reading, understanding, and running code. Practice modifying takes more time!
📊 Key Concepts Checklist

By the end of all experiments, you should understand:

  • ☐ How gradient descent works and why it converges
  • ☐ Difference between batch, mini-batch, and stochastic gradient descent
  • ☐ How to handle constraints in optimization
  • ☐ Purpose of regularization (L1, L2, Elastic Net)
  • ☐ Advanced methods: Newton, BFGS, Interior Point
  • ☐ Distributed optimization basics
  • ☐ How to implement optimization algorithms in Python
  • ☐ When to use which algorithm for which problem
