Getting Started

This website is designed for hands-on learning of optimization algorithms. Follow these steps to maximize your learning experience:

  1. Browse Outcomes: Start at the Home page and select a Course Outcome (CO1-CO5).
  2. Review Theory: Read the Aim and Description to understand the problem context.
  3. Study the Algorithm: Follow the procedural steps before looking at the code.
  4. Execute Code: Copy the provided simple Python programs into your local environment.
Prerequisites

Ensure you have Python installed along with the following libraries:

pip install numpy scipy scikit-learn cvxpy pulp
Learning Progression
  • Foundations (CO1): Understanding gradients and Newton's method.
  • Constraints (CO2): Solving problems with bounds and linear constraints.
  • Non-Smooth (CO3): Handling absolute values and barrier functions.
  • Distributed (CO4): Implementing large-scale optimization simulation.
  • Advanced (CO5): Exploring Quasi-Newton and Conjugate methods.
