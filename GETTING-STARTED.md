## 🎯 Using the Website

### To View an Experiment:
1. Find the course outcome section (e.g., CO1)
2. Click on any experiment card
3. Read through the sections:
   - **📌 Aim**: What you'll learn
   - **📝 Description**: Explanation
   - **🔧 Algorithm**: Steps to follow
   - **📐 Formulas**: Equations
   - **📊 Procedure**: Example calculation
   - **💻 Program**: Python code to run
   - **📤 Output**: Expected results

### Navigation:
- **Previous/Next buttons**: Move between experiments
- **Home button**: Return to main page
- **Breadcrumbs**: Show your location

## 💻 Running the Python Code

### Install Requirements (First Time Only):

### To Run Code:
1. Copy the Python code from the **💻 Simple Python Program** section
