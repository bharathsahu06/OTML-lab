Laboratory Procedures Manual

Official list of experiments and mathematical formulations for the OTML course.

CO1: Basic Optimization Methods

Exp 01: Gradient Descent

Implementation of descent for \( f(x) = x^2 \) using samples \( x=[1,2,3,4] \) and \( y=[1,4,9,16] \).

Exp 02: Newton's Method

Optimization of \( f(x) = x^4 - 3x^2 + 2 \) using second-order curvature information.

Exp 03: Stochastic Gradient Descent (SGD)

Single-sample updates for \( f(x) = x^2 \) using dataset \( x=[1,2,3,4], y=[1,4,9,16] \).

CO2: Constrained Optimization

Exp 04: Minimize \( f(x,y)=x^2+y^2 \)

Subject to \( x+y \ge 1; x,y \ge 0 \) using CVXPY.

Exp 05: Linear Objective Optimization

Subject to \( x+y \ge 4, x,y \ge 0 \) for \( f(x,y)=3x+2y \) using CVXPY.

Exp 06: Quadratic Objective Optimization

Subject to \( x_1+x_2 \ge 1, x_1,x_2 \ge 0 \) for \( f(x)=x_1^2+x_2^2 \) using CVXPY.

Exp 07: LP with and without PuLP

Implementation of linear programming for \( z = 3x + 2y \) with manual and automated verification.

CO3: Advanced Methods

Exp 08: Sub-gradient Method (Dataset 1)

L1-regularized linear regression for dataset \( (1,2,5), (2,1,6), (3,4,7) \).

Exp 09: Sub-gradient Method (Dataset 2)

L1-regularized linear regression for dataset \( (2,3,8), (1,4,7), (3,2,9) \).

Exp 10: Interior Point Method (1D)

Minimizing \( x^2 \) subject to \( x \ge 1 \) using barrier functions.

Exp 11: Interior Point Method (2D)

Minimizing \( x+y \) subject to \( x,y \ge 1 \).

CO4: Distributed and Batch Methods

Exp 12: Mini-batch GD for \( y = 2x + 1 \)

Processing batches of size 2 for linear regression samples.

Exp 13: Distributed GD Simulation (6 points)

Simulating 3 workers processing data for the model \( y = 2x + 1 \).

Exp 14: Mini-batch GD for \( y = 2x + 3 \)

Linear regression for \( x=[1,2,3,4], y=[5,7,9,11] \).

Exp 15: Distributed GD Simulation (4 points)

Simulating 2 workers processing data for the model \( y = 2x + 1 \).

CO5: Advanced Regularization

Exp 16: Regularization Techniques

Implementation of L1 (Lasso), L2 (Ridge), and Elastic Net for feature shrinkage.

Exp 17: Grouped Sparsity

Demonstrating block sparsity on pre-defined feature groups using Group Lasso.

Exp 18: Conjugate Gradient Method

Solving \( Ax = b \) for symmetric, positive definite matrices.

Exp 19: BFGS Quasi-Newton Update

Minimizing \( f(x_1,x_2) = x_1^2 + 2x_2^2 \) using gradient-based Hessian approximation.
