Overview

This page summarizes the mathematical formulations and algorithms for all 19 experiments conducted in the OTML Lab. Each section reflects the verified procedures from the official laboratory manual.

CO1: Gradient Based Optimization

Exp 01: OLS Linear Regression

Computes \( w = (X^T X)^{-1} X^T y \). Optimal for datasets where the model perfectly matches observations.

Exp 02: Gradient Descent

Iteratively updates parameters: \( \theta = \theta - \eta \nabla J(\theta) \). Uses full-batch gradients.

Exp 03: Newton's Method

Second-order optimization using Hessian information: \( x_{k+1} = x_k - [Hf(x_k)]^{-1} \nabla f(x_k) \).

CO2: Stochastic and Constrained Optimization

Exp 04: Stochastic Gradient Descent (SGD)

Updates weights using a single random sample per iteration, enabling faster data processing.

Exp 05: Linear & Quadratic Programming

Solves optimization problems with linear constraints using the Simplex method (LP) and KKT conditions (QP).

Exp 06: CVXPY and PuLP Solvers

Demonstrates higher-level modeling libraries for convex optimization and linear programming.

CO3: Non-Smooth and Interior Point Methods

Exp 07: Sub-gradient Method

Addresses non-differentiable objectives (like L1-norm) by using any vector in the sub-differential set.

Exp 08: Interior Point Method

Handles constraints by converting them into log-barrier functions: \( \phi(x; \mu) = f(x) - \mu \sum \ln(-g_i(x)) \).

CO4: Scalable and Distributed Optimization

Exp 09: Mini-batch Gradient Descent

The standard training method for deep learning, using subsets of data for update stability.

Exp 10: Distributed Gradient Descent

Parallelizes computation across workers. Gradients are computed locally and averaged globally.

Exp 11: Elastic Net Regularization

Combines L1 and L2 penalties: \( J(w) = MSE + \lambda_1 \|w\|_1 + \lambda_2 \|w\|_2^2 \).

Exp 12: Group Lasso

Promotes block-sparsity, allowing features to be toggled on/off in pre-defined groups.

CO5: Advanced Topics

Exp 13: Parameter Averaging

Distributed technique where workers perform multiple local updates before models are averaged.

Exp 14: Shifted Linear Systems

Optimization of models with non-zero intercepts (\( y = mx + c \)) using bias terms.

Exp 15: System Staleness

Analyzes the impact of asynchronous communication and stragglers in distributed environments.

Exp 18: Conjugate Gradient Method

Iteratively solves \( Ax=b \) in \( n \) steps by searching in A-conjugate directions.

Exp 19: BFGS Quasi-Newton

Learns a local approximation of the inverse Hessian without computing second-order derivatives.
