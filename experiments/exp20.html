Experiment 20: Implementing CNN (Cat vs Dog)

1) Aim
To build a simple Convolutional Neural Network (CNN) for binary classification of cat and dog images from the CIFAR-10 dataset.
2) Description

A Convolutional Neural Network (CNN) is a deep learning architecture specifically designed to process and analyze image data by learning spatial patterns such as edges, textures, and object parts. Unlike traditional machine learning approaches that require manual feature engineering, CNNs automatically extract meaningful visual features directly from pixel values through trainable convolutional filters.

In this experiment, the CIFAR-10 dataset, which contains 60,000 32×32 color images across 10 classes, is filtered to keep only two classes: cat (class 3) and dog (class 5), converting the task into binary classification. The input images are normalized by dividing by 255.0, ensuring pixel values fall in the range [0, 1], which makes training more stable and faster. The network architecture applies convolution operations to detect local patterns, max-pooling layers to reduce spatial dimensions while retaining important feature activations, and fully connected layers to combine extracted features for final decision making.

The final output layer uses a sigmoid activation function, which produces a probability value between 0 and 1. A value closer to 1 indicates the image is likely a dog, while a value closer to 0 indicates a cat. During training, the model minimizes binary cross-entropy loss using the Adam optimizer, and performance is measured with accuracy on validation and test data. This architecture demonstrates the complete CNN workflow: data preprocessing, feature extraction, classification, evaluation, and single-image inference, making it ideal for beginners learning deep learning concepts.

3) Network Architecture

CNN Model Structure:

  1. Input Layer: 32×32×3 (height × width × channels for RGB images)
  2. Conv2D Layer 1: 32 filters, 3×3 kernel with ReLU activation
  3. MaxPooling2D Layer 1: 2×2 pool size for spatial dimension reduction
  4. Conv2D Layer 2: 64 filters, 3×3 kernel with ReLU activation
  5. MaxPooling2D Layer 2: 2×2 pool size
  6. Flatten Layer: Converts 2D feature maps to 1D vector
  7. Dense Layer 1: 64 neurons with ReLU activation
  8. Dense Layer 2 (Output): 1 neuron with Sigmoid activation for binary classification

Compilation Details:

  • Optimizer: Adam
  • Loss Function: Binary Crossentropy
  • Metrics: Accuracy
4) Important Formulas

Sigmoid Activation (Output Layer):

$$\sigma(z) = \frac{1}{1 + e^{-z}}$$

Binary Cross-Entropy Loss:

$$L = -\frac{1}{n}\sum_{i=1}^{n} \left[ y_i \log(\hat{y}_i) + (1-y_i) \log(1-\hat{y}_i) \right]$$

where $y_i$ is the true label (0 for cat, 1 for dog) and $\hat{y}_i$ is the predicted probability.

ReLU Activation (Hidden Layers):

$$\text{ReLU}(z) = \max(0, z)$$

Prediction Rule:

$$\text{Predicted Class} = \begin{cases} \text{Dog} & \text{if } \hat{y} \geq 0.5 \\ \text{Cat} & \text{if } \hat{y} < 0.5 \end{cases}$$
5) Algorithm (Step by Step)
  1. Load Data: Load CIFAR-10 dataset and extract cat and dog images. Normalize by scaling pixel values to [0, 1].
  2. Convert Labels: Create binary labels where 0 = cat and 1 = dog.
  3. Build Model: Construct the CNN with specified layers and architecture.
  4. Compile Model: Configure optimizer (Adam), loss function (Binary Crossentropy), and metrics (Accuracy).
  5. Train Model: Fit the model on training data with 20% validation split for 5 epochs, using batch size 64.
  6. Evaluate Model: Test the trained model on test set to get final accuracy.
  7. Single Image Inference: Load a user-provided image, preprocess it, and predict whether it is a cat or dog with confidence score.
6) Implementation Overview

Main Code Steps:

# 1. Load and prepare CIFAR-10 data (cat and dog only)
(x_train, y_train), (x_test, y_test) = tf.keras.datasets.cifar10.load_data()
keep_train = np.isin(y_train, [3, 5]).ravel()
keep_test = np.isin(y_test, [3, 5]).ravel()
x_train, x_test = x_train[keep_train] / 255.0, x_test[keep_test] / 255.0
y_train, y_test = (y_train[keep_train] == 5).astype("int32"), 
                  (y_test[keep_test] == 5).astype("int32")

# 2. Build CNN model
model = Sequential([
    layers.Input((32, 32, 3)),
    layers.Conv2D(32, 3, activation="relu"), 
    layers.MaxPooling2D(),
    layers.Conv2D(64, 3, activation="relu"), 
    layers.MaxPooling2D(),
    layers.Flatten(), 
    layers.Dense(64, activation="relu"),
    layers.Dense(1, activation="sigmoid")
])

# 3. Compile and train
model.compile(optimizer="adam", loss="binary_crossentropy", metrics=["accuracy"])
history = model.fit(x_train, y_train, epochs=5, batch_size=64, 
                    validation_split=0.2, verbose=1)

# 4. Evaluate on test set
loss, acc = model.evaluate(x_test, y_test, verbose=0)
print(f"Test Accuracy: {acc:.4f}")

# 5. Predict on custom image
img = load_img(image_path, target_size=(32, 32))
arr = img_to_array(img) / 255.0
arr = np.expand_dims(arr, axis=0)
prob_dog = float(model.predict(arr, verbose=0)[0][0])
pred_label = "Dog" if prob_dog >= 0.5 else "Cat"
print(f"Prediction: {pred_label} (Dog probability: {prob_dog:.4f})")
