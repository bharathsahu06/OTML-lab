Experiment 20: Implementing CNN (Cat vs Dog)

1) Aim
To build a simple Convolutional Neural Network (CNN) for binary classification of cat and dog images from the CIFAR-10 dataset.
2) Description

A Convolutional Neural Network (CNN) is a special type of artificial brain that is very good at understanding pictures. Just like humans look at pictures and recognize objects, CNNs learn to find patterns in images automatically without needing someone to tell them what to look for.

In this experiment, we use a dataset called CIFAR-10, which contains thousands of small pictures (32-by-32 pixels) in color. We pick only the cat and dog pictures from this dataset. Before the CNN sees these pictures, we make them all the same brightness level so the learning process is easier and faster.

The CNN works by passing images through several layers. First, it looks for simple patterns like lines and corners (convolution layer). Then it shrinks the image to keep only the most important information (pooling layer). After doing this a few times, it flattens the image information and uses regular brain-like layers to decide: "Is this a cat or a dog?" Finally, it gives a score between 0 and 1, where 0 means "definitely a cat" and 1 means "definitely a dog".

4) How the Network Makes Decisions

Prediction Score: The final layer gives a number between 0 and 1. If the score is closer to 0.5 or below, the image is a cat. If it's above 0.5, the image is a dog.

How the Network Learns: During training, the network looks at many cat and dog pictures. For each picture, it makes a guess and then checks if the guess was right or wrong. If wrong, it adjusts itself a little bit to make better guesses next time. It repeats this process many times until it gets very good at recognizing cats and dogs.

What Each Layer Does:

  • Convolution Layer: Acts like a detective looking for clues. It moves a small window across the image to find patterns like whiskers, paws, ears, etc.
  • ReLU (Activation): A rule that says "if the pattern signal is negative, ignore it. If it's positive, keep it." This helps the network focus on useful patterns.
  • Pooling Layer: Like taking a photo from far away - you lose small details but keep the important shapes. This makes the network faster and stronger.
  • Dense Layer: All the patterns found are combined together with weights (importance scores) to make the final decision.
  • Sigmoid Output: The final answer: a number between 0 (definitely cat) and 1 (definitely dog).
5) Simple Steps (How It Works)
  1. Get the Pictures: Download the CIFAR-10 dataset which has thousands of small pictures. Pick only the cat and dog pictures.
  2. Prepare Pictures: Make all pictures the same brightness level (0 to 1) so learning is easier.
  3. Build the Brain: Create the CNN with all its layers as described above.
  4. Start Learning: Show the network 5 batches of pictures. For each batch, it makes guesses and fixes mistakes.
  5. Check the Score: After training, test the network on pictures it has never seen before to see how accurate it is.
  6. Use It! Give the network a new picture of a cat or dog and it will tell you what it is.

Full Implementation - All Code Cells:

Cell 1: Import Required Libraries

import numpy as np
import matplotlib.pyplot as plt
import tensorflow as tf
from tensorflow.keras import layers, Sequential

Cell 2: Load and Prepare Data (Extract Cat vs Dog from CIFAR-10)

# 1) Load data and keep only cat(3), dog(5)
(x_train, y_train), (x_test, y_test) = tf.keras.datasets.cifar10.load_data()

keep_train = np.isin(y_train, [3, 5]).ravel()
keep_test = np.isin(y_test, [3, 5]).ravel()

x_train, x_test = x_train[keep_train] / 255.0, x_test[keep_test] / 255.0
y_train, y_test = (y_train[keep_train] == 5).astype("int32"), (y_test[keep_test] == 5).astype("int32")

print(x_train.shape, y_train.shape)
print(x_test.shape, y_test.shape)

Output: Filtered dataset shapes containing only cat and dog images

# 2) Show a few images

Cell 4: Build and Compile the CNN Model

# 3) Small CNN model
model = Sequential([
    layers.Input((32, 32, 3)),
    layers.Conv2D(32, 3, activation="relu"), 
    layers.MaxPooling2D(),
    layers.Conv2D(64, 3, activation="relu"), 
    layers.MaxPooling2D(),
    layers.Flatten(), 
    layers.Dense(64, activation="relu"),
    layers.Dense(1, activation="sigmoid")
])

model.compile(optimizer="adam", loss="binary_crossentropy", metrics=["accuracy"])
model.summary()

Constructs the CNN architecture and displays layer information with parameter counts

Cell 5: Train the Model on Training Data and Evaluate on Test Data

# 4) Train + test
history = model.fit(x_train, y_train, epochs=5, batch_size=64, validation_split=0.2, verbose=1)
loss, acc = model.evaluate(x_test, y_test, verbose=0)
print(f"Test Accuracy: {acc:.4f}")

Trains for 5 epochs with batch size 64 and 20% validation split. Reports test accuracy.
