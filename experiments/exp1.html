Experiment 1: Gradient Descent

1) Aim
Implement Gradient Descent using a linear regression model for the given dataset:
  • Input $x=[3,4,5,6,7]$
  • Target $y=[4,5,6,7,8]$
2) Description

Gradient Descent is a foundational optimization algorithm that iteratively minimizes a loss function by moving in the direction opposite to the gradient. The gradient represents the direction of steepest increase, so moving opposite to it leads toward minima. This method is the backbone of modern machine learning because it scales well to high-dimensional problems and large datasets.

The core insight is that the gradient encodes both the direction and magnitude of change needed to reduce the loss. At each iteration, parameters are adjusted by a small step (controlled by the learning rate) in the negative gradient direction. This process continues until convergence, meaning the loss stabilizes and significant improvements become negligible.

Gradient Descent's effectiveness depends on problem structure, learning rate selection, and initial conditions. It works well for smooth, differentiable loss functions and is computationally efficient since it only requires first-order derivatives (gradients).

3) Important Formulas

Model (Linear Regression):

$$\hat{y}_i=w_0+w_1 x_i$$

Loss (Mean Squared Error):

$$J(w_0,w_1)=\frac{1}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)^2$$

Gradients:

$$\frac{\partial J}{\partial w_0}=\frac{2}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)$$ $$\frac{\partial J}{\partial w_1}=\frac{2}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)x_i$$

Update rule:

$$w_0\leftarrow w_0-\alpha\frac{\partial J}{\partial w_0}, \quad w_1\leftarrow w_1-\alpha\frac{\partial J}{\partial w_1}$$
4) Algorithm (Step by Step)
  1. Initialize weights $w_0=0, w_1=0$ and set hyperparameters ($\alpha$, tolerance, max iterations).
  2. Compute predictions $\hat{y}_i=w_0+w_1 x_i$.
  3. Compute loss $J(w_0,w_1)$ (Mean Squared Error).
  4. Compute gradients $\frac{\partial J}{\partial w_0}$ and $\frac{\partial J}{\partial w_1}$.
  5. Update $w_0$ and $w_1$ simultaneously and repeat until convergence.
5) Mathematical Procedure (Complete Step-by-Step)

Given: $x=[3,4,5,6,7]$, $y=[4,5,6,7,8]$, model $\hat{y}_i=w_0+w_1 x_i$

1. Compute predictions: $$\hat{y}_i=w_0+w_1 x_i$$
2. Compute errors: $$e_i=\hat{y}_i-y_i$$
3. Compute loss (Mean Squared Error): $$J(w_0,w_1)=\frac{1}{n}\sum_{i=1}^{n}(e_i)^2$$
4. Compute gradients and update: $$\frac{\partial J}{\partial w_0}=\frac{2}{n}\sum_{i=1}^{n}(e_i),\quad \frac{\partial J}{\partial w_1}=\frac{2}{n}\sum_{i=1}^{n}(e_i x_i)$$ $$w_0\leftarrow w_0-\alpha\frac{\partial J}{\partial w_0},\quad w_1\leftarrow w_1-\alpha\frac{\partial J}{\partial w_1}$$
6) Given Input
  • $x=[3,4,5,6,7]$
  • $y=[4,5,6,7,8]$
  • Initial $w_0=0, w_1=0$
  • Learning rate $\alpha=0.01$
  • Tolerance $=10^{-6}$
  • Max iterations $=100$
7) Python Implementation & Output
import numpy as np # Data x = np.array([3, 4, 5, 6, 7], dtype=float) y = np.array([4, 5, 6, 7, 8], dtype=float) # Initial weights w0 = 0.0 w1 = 0.0 # Hyperparameters lr = 0.01 tol = 1e-6 max_iter = 100 n = len(x) print("\nIter\t w0\t\t w1\t\t Loss") for i in range(1, max_iter + 1): # Predictions y_pred = w0 + w1 * x errors = y_pred - y # Loss loss = np.mean(errors**2) # Gradients grad_w0 = 2 * np.mean(errors) grad_w1 = 2 * np.mean(errors * x) # Update w0_new = w0 - lr * grad_w0 w1_new = w1 - lr * grad_w1 print(f"{i}\t {w0_new:.6f}\t {w1_new:.6f}\t {loss:.6f}") if abs(w0_new - w0) < tol and abs(w1_new - w1) < tol: w0 = w0_new w1 = w1_new break w0 = w0_new w1 = w1_new print("\nLinear Model: y = w0 + w1*x") print(f"\nOptimal w0 = {round(w0, 6)}") print(f"Optimal w1 = {round(w1, 6)}") print(f"Final Loss = {round(loss, 6)}")
