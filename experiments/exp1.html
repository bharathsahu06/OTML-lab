Experiment 1: Gradient Descent Optimization

📌 Aim

To understand and implement the gradient descent algorithm to find the minimum of the function f(x) = x². The method shows how repeated movement in the direction of the negative gradient leads toward the minimum of the objective function.

📝 Description

Gradient descent is a first-order optimization method used to minimize a function by repeatedly moving in the direction of steepest decrease. At every iteration, the algorithm evaluates the gradient of the function at the current point and uses that information to decide how the variable should be updated.

For f(x) = x^2, the gradient is 2x, so the update rule keeps pulling x toward 0, which is the global minimum. This simple objective clearly shows how gradients, learning rate, and convergence are connected in iterative optimization.

  • The gradient tells how the function changes at the current point
  • The learning rate controls how large each update step is
  • Repeated updates gradually reduce the objective value until the algorithm converges
🔧 Algorithm
Step 1: Initialize x to a random value (e.g., x = 5.0)
Step 2: Set learning rate α (e.g., α = 0.01)
Step 3: Calculate gradient: g = ∂f/∂x = 2x
Step 4: Update x: x = x - α × g
Step 5: Repeat steps 3-4 until x converges (change is very small)
Step 6: Return the final x value (minimum point)
📐 Formulas
Objective Function:
f(x) = x²
Gradient (Derivative):
∂f/∂x = 2x
Update Rule:
x(n+1) = x(n) - α × ∂f/∂x
where α is the learning rate (step size)
📊 Mathematical Procedure (Step by Step)

Example Calculation:

Let's do gradient descent with initial x₀ = 5, learning rate α = 0.1

Iteration 0:
x₀ = 5.0
f(5) = 5² = 25
∇f = 2 × 5 = 10
Iteration 1:
x₁ = 5.0 - 0.1 × 10 = 5.0 - 1.0 = 4.0
f(4) = 4² = 16
∇f = 2 × 4 = 8
Iteration 2:
x₂ = 4.0 - 0.1 × 8 = 4.0 - 0.8 = 3.2
f(3.2) = 3.2² = 10.24
∇f = 2 × 3.2 = 6.4
Iteration 3:
x₃ = 3.2 - 0.1 × 6.4 = 3.2 - 0.64 = 2.56
f(2.56) = 2.56² = 6.5536
∇f = 2 × 2.56 = 5.12

... (continuing this process)

Final Iteration (after ~100 iterations):
x ≈ 0.0 (very close to 0)
f(x) ≈ 0 (minimum value)
💻 Simple Python Program

Easy to Understand Code:

# Gradient Descent for f(x) = x² def gradient_descent(x_start, learning_rate, iterations): x = x_start # Start point for i in range(iterations): gradient = 2 * x # Calculate gradient (derivative of x²) x = x - learning_rate * gradient # Update x # Print progress every 10 iterations if i % 10 == 0: f_value = x ** 2 print(f"Iteration {i}: x = {x:.4f}, f(x) = {f_value:.4f}") return x # Run the algorithm final_x = gradient_descent(5.0, 0.1, 100) print(f"\nFinal minimum at x = {final_x:.6f}") print(f"Minimum value f(x) = {final_x**2:.6f}")
