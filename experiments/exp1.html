Experiment 1: Gradient Descent Implementation

Home / Experiment 1
Aim

To understand and implement the Gradient Descent algorithm for optimizing the function \( f(x) = x^2 \) using the dataset \( x = [1, 2, 3, 4] \) and \( y = [1, 4, 9, 16] \).

Description

Gradient Descent is an iterative optimization algorithm used to find the local minimum of a function. In this experiment, we minimize the mean squared error between the predictions of our model \( f(x) = x^2 \) and the actual labels \( y \).

\( f(x) = x^2 \)
Gradient: \( \nabla f(x) = 2x \)
Algorithm
  1. Initialize the parameter \( x \) to a random value (e.g., 5.0).
  2. Set the learning rate \( \alpha = 0.1 \).
  3. Repeat for \( N \) iterations:
    • Compute the gradient: \( g = 2x \).
    • Update the parameter: \( x = x - \alpha \cdot g \).
  4. Observe the convergence to the global minimum at \( x = 0 \).
Simple Python Program
import numpy as np

# Initial values
x = 5.0
learning_rate = 0.1
epochs = 20

print("k | x_k | f(x_k)")
print("-" * 15)

for k in range(epochs):
    grad = 2 * x
    x = x - learning_rate * grad
    print(f"{k} | {x:.4f} | {x**2:.4f}")
