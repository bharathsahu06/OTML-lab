Gradient Descent Optimization

🎯 Aim

To implement and analyze the convergence properties of the Gradient Descent algorithm for minimizing a fundamental convex objective function \( f(x) = x^2 \).

📝 Problem Statement

Implement a gradient descent routine to find the global minimum of the function \( f(x) = x^2 \). Given a dataset \( x = [1, 2, 3, 4], y = [1, 4, 9, 16] \), verify that the data points lie on the objective curve and observe the iterative reduction in value starting from an initial point \( x_0 = 5.0 \).

📐 Mathematical Procedure
Part A: Model Derivation

Verify data consistency for the relationship \( y = x^2 \):

  • At \( x = 1 \): \( y = 1^2 = 1 \) 🗸
  • At \( x = 2 \): \( y = 2^2 = 4 \) 🗸
  • At \( x = 3 \): \( y = 3^2 = 9 \) 🗸
  • At \( x = 4 \): \( y = 4^2 = 16 \) 🗸
Part B: Iterative Update Rule

The derivative (gradient) of the objective \( f(x) = x^2 \) is:

\( f'(x) = 2x \)

The parameter update rule with learning rate \( \eta \) is:

\( x_{k+1} = x_k - \eta \cdot f'(x_k) = x_k - \eta(2x_k) \)
Part C: Manual Calculation ( \(\eta = 0.1\), \(x_0 = 5.0\) )
  • Iteration 0: \( x_1 = 5.0 - 0.1(2 \cdot 5.0) = 4.0 \)
  • Iteration 1: \( x_2 = 4.0 - 0.1(2 \cdot 4.0) = 3.2 \)
  • Iteration 2: \( x_3 = 3.2 - 0.1(2 \cdot 3.2) = 2.56 \)
💻 Python Implementation
# Gradient Descent for f(x) = x^2 import numpy as np # Hyperparameters x = 5.0 alpha = 0.1 epochs = 20 print(f"Initial state: x = {x}") for k in range(epochs): # x = x - alpha * gradient (where gradient = 2 * x) x = x * (1 - 2 * alpha) if k % 5 == 0: print(f"Iteration {k+1}: x = {x:.8f}") print(f"\nFinal x: {x:.8f}")
📊 Expected Output

Final x: 0.00000000 (Approaches 0)
✅ Result & Verification
The parameter \( x \) successfully converges to the theoretical minimum at 0. Since the contraction factor \( (1 - 2\eta) = 0.8 \) is less than 1, geometric convergence is guaranteed for this convex function.
