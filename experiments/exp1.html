Experiment 1: Gradient Descent

1) Aim
Understand and implement Gradient Descent to learn $w$ in $f(x)=w x^2$ for the given data:
  • Input $x=[1,2,3,4]$
  • Target $y=[1,4,9,16]$
2) Description

Gradient Descent is a foundational optimization algorithm that iteratively minimizes a loss function by moving in the direction opposite to the gradient. The gradient represents the direction of steepest increase, so moving opposite to it leads toward minima. This method is the backbone of modern machine learning because it scales well to high-dimensional problems and large datasets.

The core insight is that the gradient encodes both the direction and magnitude of change needed to reduce the loss. At each iteration, parameters are adjusted by a small step (controlled by the learning rate) in the negative gradient direction. This process continues until convergence, meaning the loss stabilizes and significant improvements become negligible.

Gradient Descent's effectiveness depends on problem structure, learning rate selection, and initial conditions. It works well for smooth, differentiable loss functions and is computationally efficient since it only requires first-order derivatives (gradients).

3) Important Formulas

Model:

$$\hat{y}_i=w x_i^2$$

Loss (Mean Squared Error):

$$J(w)=\frac{1}{n}\sum_{i=1}^{n}(w x_i^2-y_i)^2$$

Gradient w.r.t. $w$:

$$\frac{dJ}{dw}=\frac{2}{n}\sum_{i=1}^{n}(w x_i^2-y_i)\,x_i^2$$

Update rule:

$$w\leftarrow w-\alpha\frac{dJ}{dw}$$
4) Algorithm (Step by Step)
  1. Read an initial value for $w$ and set hyperparameters ($\alpha$, tolerance, max iterations).
  2. Compute predictions $\hat{y}=w x^2$.
  3. Compute loss $J(w)$ (Mean Squared Error).
  4. Compute gradient $\frac{dJ}{dw}$.
  5. Update $w\leftarrow w-\alpha\frac{dJ}{dw}$ and repeat until convergence.
5) Mathematical Procedure (Complete Step-by-Step)

Given: $x=[1,2,3,4]$, $y=[1,4,9,16]$, model $\hat{y}=w x^2$

1. Compute predictions: $$\hat{y}_i=w x_i^2$$
2. Compute errors: $$e_i=\hat{y}_i-y_i$$
3. Compute loss (Mean Squared Error): $$J(w)=\frac{1}{n}\sum_{i=1}^{n}(e_i)^2$$
4. Compute gradient and update: $$\frac{dJ}{dw}=\frac{2}{n}\sum_{i=1}^{n}(w x_i^2-y_i)\,x_i^2,\qquad w\leftarrow w-\alpha\frac{dJ}{dw}$$
6) Given Input
  • $x=[1,2,3,4]$
  • $y=[1,4,9,16]$
  • Initial $w$ (user input)
  • Learning rate $\alpha=0.01$
  • Tolerance $=10^{-6}$
  • Max iterations $=100$
7) Python Implementation & Output
import numpy as np # Data x = np.array([1, 2, 3, 4], dtype=float) y = np.array([1, 4, 9, 16], dtype=float) # Initial weight w = float(input("Enter initial w: ")) # Hyperparameters lr = 0.01 tol = 1e-6 max_iter = 100 print("\nIter\t w\t\t Loss") for i in range(1, max_iter + 1): y_pred = w * x**2 error = y_pred - y loss = np.mean(error**2) # Gradient (manually derived) grad = 2 * np.mean(error * x**2) w_new = w - lr * grad print(f"{i}\t {w_new:.6f}\t {loss:.6f}") if abs(w_new - w) < tol: w = w_new break w = w_new print("f(x) = w * x^2") print("\nOptimal w =", round(w, 6)) print("Final Loss =", round(loss, 6))
