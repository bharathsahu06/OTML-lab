Experiment 01: Linear Regression using OLS

Home / Experiment 01
Aim

To implement Linear Regression using the Ordinary Least Squares (OLS) method to find the best-fit line for a given dataset.

Description

Ordinary Least Squares (OLS) is a type of linear least squares method for estimating the unknown parameters in a linear regression model. Standard OLS regression minimizes the sum of squared vertical distances (residuals) between the observed responses in the dataset and the responses predicted by the linear approximation.

Dataset:

  • X: [1, 3, 5, 7, 9]
  • Y: [3, 8, 13, 20, 25]
Algorithm
  1. Initialize the dataset of x and y values.
  2. Calculate the number of elements \( n \).
  3. Compute \( \sum x \), \( \sum y \), \( \sum xy \), and \( \sum x^2 \).
  4. Calculate the slope \( m \) and intercept \( b \) using the OLS formulas.
  5. Iterate through the data to calculate predicted values and squared errors.
  6. Calculate Total Error and Mean Squared Error (MSE).
Formulas
Slope \( m = \frac{n\sum(xy) - (\sum x)(\sum y)}{n\sum(x^2) - (\sum x)^2} \)
Intercept \( b = \frac{\sum y - m(\sum x)}{n} \)
Mean Squared Error (MSE) \( = \frac{1}{n}\sum(y - \hat{y})^2 \)
Mathematical Procedure

Based on the dataset \( x=[1, 3, 5, 7, 9], y=[3, 8, 13, 20, 25] \):

  • \( n = 5 \)
  • \( \sum x = 1+3+5+7+9 = 25 \)
  • \( \sum y = 3+8+13+20+25 = 69 \)
  • \( \sum xy = (1*3)+(3*8)+(5*13)+(7*20)+(9*25) = 3+24+65+140+225 = 457 \)
  • \( \sum x^2 = 1+9+25+49+81 = 165 \)

Calculation for \( m \):

\( m = \frac{5(457) - (25)(69)}{5(165) - (25)^2} = \frac{2285 - 1725}{825 - 625} = \frac{560}{200} = 2.8 \)

Calculation for \( b \):

\( b = \frac{69 - 2.8(25)}{5} = \frac{69 - 70}{5} = \frac{-1}{5} = -0.2 \)

Simple Python Program
# Linear Regression using OLS
def linear_regression(x, y):
    n = len(x)
    sum_xy, sum_x, sum_y, sum_xs = 0, 0, 0, 0
    for i in range(n):
        sum_xy += x[i] * y[i]
        sum_x += x[i]
        sum_y += y[i]
        sum_xs += x[i]**2
    
    m = ((n * sum_xy) - (sum_x * sum_y)) / (n * sum_xs - sum_x**2)
    b = (sum_y / n) - (m * (sum_x) / n)
    return m, b

def mse(x, y, m, b):
    n = len(x)
    total_error = 0
    for i in range(n):
        pred = m * x[i] + b
        total_error += (y[i] - pred)**2
    return total_error / n, total_error

x = [1, 3, 5, 7, 9]
y = [3, 8, 13, 20, 25]

m, b = linear_regression(x, y)
mse_val, total_err = mse(x, y, m, b)

print(f"Slope: {m}")
print(f"Intercept: {b:.2f}")
print(f"Total Error: {total_err:.2f}")
print(f"MSE: {mse_val:.2f}")
Expected Output
Slope: 2.8
Intercept: -0.20
Total Error: 1.20
MSE: 0.24
