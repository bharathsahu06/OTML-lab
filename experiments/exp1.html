Experiment 1: Linear Regression (Gradient Descent & OLS)

1) Aim
Solve linear regression for the given dataset using two methods:
  • 1A) Gradient Descent Method: Iterative optimization approach
  • 1B) Ordinary Least Squares (OLS) Method: Closed-form analytical solution

Dataset:

  • Input $x=[3,4,5,6,7]$
  • Target $y=[4,5,6,7,8]$
2) Description

1A) Gradient Descent Method: Gradient Descent is a foundational optimization algorithm that iteratively minimizes a loss function by moving in the direction opposite to the gradient. The gradient represents the direction of steepest increase, so moving opposite to it leads toward minima. This method is the backbone of modern machine learning because it scales well to high-dimensional problems and large datasets.

The core insight is that the gradient encodes both the direction and magnitude of change needed to reduce the loss. At each iteration, parameters are adjusted by a small step (controlled by the learning rate) in the negative gradient direction. This process continues until convergence, meaning the loss stabilizes and significant improvements become negligible.

Gradient Descent's effectiveness depends on problem structure, learning rate selection, and initial conditions. It works well for smooth, differentiable loss functions and is computationally efficient since it only requires first-order derivatives (gradients).

1B) Ordinary Least Squares (OLS) Method: OLS is a direct closed-form solution to linear regression that provides the optimal parameters in a single computation, without iteration. It minimizes the sum of squared residuals by solving a system of linear equations derived from setting the gradient to zero (first-order optimality condition).

The OLS approach is statistically elegant: it directly computes the parameter values that minimize the Mean Squared Error (MSE) loss. For linear regression problems, OLS provides the exact optimal solution instantly, with no hyperparameter tuning required (unlike Gradient Descent which needs learning rate and convergence criteria). The trade-off is that OLS requires matrix inversion, which becomes computationally expensive in very high dimensions.

Both methods arrive at identical solutions for linear regression, but they differ in approach: Gradient Descent is iterative and general (applicable to non-convex problems), while OLS is direct and specific to convex problems like linear regression.

3) Important Formulas

1A) Gradient Descent Formulas:

Model (Linear Regression):

$$\hat{y}_i=w_0+w_1 x_i$$

Loss (Mean Squared Error):

$$J(w_0,w_1)=\frac{1}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)^2$$

Gradients:

$$\frac{\partial J}{\partial w_0}=\frac{2}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)$$ $$\frac{\partial J}{\partial w_1}=\frac{2}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)x_i$$

Update rule:

$$w_0\leftarrow w_0-\alpha\frac{\partial J}{\partial w_0}, \quad w_1\leftarrow w_1-\alpha\frac{\partial J}{\partial w_1}$$

1B) Ordinary Least Squares (OLS) Formulas:

Matrix form: Add a column of 1s to create design matrix $X$, then solve:

$$X = \begin{bmatrix} 1 & x_1 \\ 1 & x_2 \\ \vdots & \vdots \\ 1 & x_n \end{bmatrix}, \quad \mathbf{y} = \begin{bmatrix} y_1 \\ y_2 \\ \vdots \\ y_n \end{bmatrix}$$

Normal equation (closed-form solution):

$$\mathbf{w}^* = (X^T X)^{-1} X^T \mathbf{y}$$

where $\mathbf{w} = [w_0, w_1]^T$ contains the intercept and slope.

4) Algorithm (Step by Step)

1A) Gradient Descent Algorithm:

  1. Initialize weights $w_0=0, w_1=0$ and set hyperparameters ($\alpha$, tolerance, max iterations).
  2. Compute predictions $\hat{y}_i=w_0+w_1 x_i$.
  3. Compute loss $J(w_0,w_1)$ (Mean Squared Error).
  4. Compute gradients $\frac{\partial J}{\partial w_0}$ and $\frac{\partial J}{\partial w_1}$.
  5. Update $w_0$ and $w_1$ simultaneously and repeat until convergence.

1B) Ordinary Least Squares (OLS) Algorithm:

  1. Construct design matrix $X$ by adding a column of 1s to the feature values.
  2. Compute the Normal Equation: $\mathbf{w}^* = (X^T X)^{-1} X^T \mathbf{y}$
  3. Extract $w_0$ and $w_1$ from the optimal weights vector $\mathbf{w}^*$.
  4. Compute final loss using the optimal parameters (done in single step, no iteration needed).
5) Mathematical Procedure (Complete Step-by-Step)

1A) Gradient Descent Procedure:

Given: $x=[3,4,5,6,7]$, $y=[4,5,6,7,8]$, model $\hat{y}_i=w_0+w_1 x_i$

1. Compute predictions: $$\hat{y}_i=w_0+w_1 x_i$$
2. Compute errors: $$e_i=\hat{y}_i-y_i$$
3. Compute loss (Mean Squared Error): $$J(w_0,w_1)=\frac{1}{n}\sum_{i=1}^{n}(e_i)^2$$
4. Compute gradients and update: $$\frac{\partial J}{\partial w_0}=\frac{2}{n}\sum_{i=1}^{n}(e_i),\quad \frac{\partial J}{\partial w_1}=\frac{2}{n}\sum_{i=1}^{n}(e_i x_i)$$ $$w_0\leftarrow w_0-\alpha\frac{\partial J}{\partial w_0},\quad w_1\leftarrow w_1-\alpha\frac{\partial J}{\partial w_1}$$

1B) Ordinary Least Squares (OLS) Procedure:

Given: $x=[3,4,5,6,7]$, $y=[4,5,6,7,8]$

1. Construct design matrix $X$ (add column of 1s): $$X = \begin{bmatrix} 1 & 3 \\ 1 & 4 \\ 1 & 5 \\ 1 & 6 \\ 1 & 7 \end{bmatrix}, \quad \mathbf{y} = \begin{bmatrix} 4 \\ 5 \\ 6 \\ 7 \\ 8 \end{bmatrix}$$
2. Compute $X^T X$ and $X^T \mathbf{y}$: $$X^T X = \begin{bmatrix} 5 & 25 \\ 25 & 135 \end{bmatrix}, \quad X^T \mathbf{y} = \begin{bmatrix} 30 \\ 160 \end{bmatrix}$$
3. Compute $(X^T X)^{-1}$: $$(X^T X)^{-1} = \frac{1}{675-625} \begin{bmatrix} 135 & -25 \\ -25 & 5 \end{bmatrix} = \begin{bmatrix} 2.7 & -0.5 \\ -0.5 & 0.1 \end{bmatrix}$$
4. Solve for optimal weights: $$\mathbf{w}^* = (X^T X)^{-1} X^T \mathbf{y} = \begin{bmatrix} 1 \\ 1 \end{bmatrix}$$ $$w_0^* = 1, \quad w_1^* = 1$$

Common Data:

1A) Gradient Descent Hyperparameters:

  • Learning rate $\alpha=0.01$

1B) OLS:

7) Python Implementation & Output

1A) Method 1: Gradient Descent

import numpy as np # Data x = np.array([3, 4, 5, 6, 7], dtype=float) y = np.array([4, 5, 6, 7, 8], dtype=float) # Initial weights w0 = 0.0 w1 = 0.0 # Hyperparameters lr = 0.01 tol = 1e-6 max_iter = 100 n = len(x) print("=== GRADIENT DESCENT METHOD ===") print("\nIter\t w0\t\t w1\t\t Loss") for i in range(1, max_iter + 1): # Predictions y_pred = w0 + w1 * x errors = y_pred - y # Loss loss = np.mean(errors**2) # Gradients grad_w0 = 2 * np.mean(errors) grad_w1 = 2 * np.mean(errors * x) # Update w0_new = w0 - lr * grad_w0 w1_new = w1 - lr * grad_w1 print(f"{i}\t {w0_new:.6f}\t {w1_new:.6f}\t {loss:.6f}") if abs(w0_new - w0) < tol and abs(w1_new - w1) < tol: w0 = w0_new w1 = w1_new break w0 = w0_new w1 = w1_new print("\nGradient Descent Results:") print(f"Optimal w0 = {round(w0, 6)}") print(f"Optimal w1 = {round(w1, 6)}") print(f"Final Loss = {round(loss, 6)}") print(f"Model: y = {w0:.1f} + {w1:.1f}*x")
