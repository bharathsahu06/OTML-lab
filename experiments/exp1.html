Experiment 1: Gradient Descent

1) Aim
Understand and implement Gradient Descent for the given data:
  • Input $x=[1,2,3,4]$
  • Target $y=[1,4,9,16]$
2) Description

Gradient Descent is an iterative optimization method used to minimize a loss function. At each iteration, the parameters are moved in the opposite direction of the gradient because the gradient points toward the steepest increase of the loss.

For this experiment, we fit a simple linear model:

$$\hat{y}=wx+b$$

Here, $w$ represents the slope and $b$ represents the intercept. The objective is to find values of $w$ and $b$ that reduce the Mean Squared Error between predicted and actual values.

Even though the target values follow a quadratic trend, we use a linear model to demonstrate how Gradient Descent updates parameters $w$ and $b$ step by step. This clearly shows the learning process: prediction, error calculation, gradient computation, and parameter update repeated until the loss becomes stable.

3) Important Formulas

Model:

$$\hat{y}_i=wx_i+b$$

Loss (Mean Squared Error):

$$J(w,b)=\frac{1}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)^2$$

Gradients:

$$\frac{\partial J}{\partial w}=\frac{2}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)x_i, \qquad \frac{\partial J}{\partial b}=\frac{2}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)$$

Update rules:

$$w \leftarrow w-\alpha\frac{\partial J}{\partial w}, \qquad b \leftarrow b-\alpha\frac{\partial J}{\partial b}$$

where $\alpha$ is the learning rate.

4) Algorithm (Step by Step)
  1. Initialize $w=0$, $b=0$, choose learning rate $\alpha$, and number of iterations.
  2. Compute predictions $\hat{y}=wx+b$.
  3. Compute loss $J(w,b)$.
  4. Compute gradients $\frac{\partial J}{\partial w}$ and $\frac{\partial J}{\partial b}$.
  5. Update $w$ and $b$ using Gradient Descent rules.
  6. Repeat Steps 2-5 until iterations are completed.
  7. Report final parameters, final loss, and predicted output.
5) Mathematical Procedure (Complete Step-by-Step)

Given:

  • $x=[1,2,3,4]$, $y=[1,4,9,16]$, $n=4$
  • Initial $w_0=0$, $b_0=0$, learning rate $\alpha=0.01$
1. Compute initial predictions: $$\hat{y}=w_0x+b_0=[0,0,0,0]$$
2. Compute errors: $$e=\hat{y}-y=[-1,-4,-9,-16]$$
3. Compute gradient w.r.t. $w$: $$\frac{\partial J}{\partial w}=\frac{2}{4}[(-1)(1)+(-4)(2)+(-9)(3)+(-16)(4)] = \frac{1}{2}(-100)=-50$$
4. Compute gradient w.r.t. $b$: $$\frac{\partial J}{\partial b}=\frac{2}{4}[(-1)+(-4)+(-9)+(-16)] = \frac{1}{2}(-30)=-15$$
5. Update parameters: $$w_1=w_0-\alpha\frac{\partial J}{\partial w}=0-0.01(-50)=0.5$$ $$b_1=b_0-\alpha\frac{\partial J}{\partial b}=0-0.01(-15)=0.15$$

6. Substitute updated $(w_1,b_1)$ in the model and repeat the same sequence (prediction, error, gradients, update).

7. Continue until loss reduces and parameter values stabilize.

6) Given Input
  • $x=[1,2,3,4]$
  • $y=[1,4,9,16]$
7) Python Implementation & Output
