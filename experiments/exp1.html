Experiment 1: Gradient Descent for f(x) = x^2

📌 Aim

To understand and implement gradient descent for the function f(x) = x^2. This experiment shows how repeated movement in the direction of the negative gradient leads to the minimum value of the function.

📝 Description

Gradient descent is a first-order optimization method used to minimize a function by repeatedly moving in the direction of steepest decrease. In this experiment, we apply it to the simple function f(x) = x^2 and observe how the value of x moves toward the minimum.

For f(x) = x^2, the gradient is 2x, so each update step pushes x closer to 0, which is the global minimum. This makes Experiment 1 a clear starting point for understanding gradients, learning rate, and convergence in optimization.

  • The gradient tells how the function changes at the current point
  • The learning rate controls how large each update step is
  • Repeated updates gradually reduce the objective value until the algorithm converges
🔧 Algorithm
Step 1: Initialize x to a starting value such as x = 5.0
Step 2: Choose a learning rate α such as 0.1
Step 3: Compute the gradient g = df/dx = 2x
Step 4: Update x using x = x - α * g
Step 5: Repeat the update until the change in x becomes very small
Step 6: Report the final x value and the minimum function value
📐 Formulas
Objective Function:
f(x) = x2
Gradient:
df/dx = 2x
Update Rule:
xn+1 = xn - α(2xn)
📊 Mathematical Procedure

Example: start with x = 5.0 and learning rate α = 0.1.

Iteration 0:
x = 5.0
Gradient = 2(5.0) = 10.0
New x = 5.0 - 0.1(10.0) = 4.0
Iteration 1:
x = 4.0
Gradient = 2(4.0) = 8.0
New x = 4.0 - 0.1(8.0) = 3.2
Iteration 2:
x = 3.2
Gradient = 2(3.2) = 6.4
New x = 3.2 - 0.1(6.4) = 2.56

Continuing this process keeps reducing both x and f(x) until the value approaches 0.

Final Result:
x approaches 0
f(x) approaches 0
💻 Simple Python Program

Python implementation:

# Gradient Descent for f(x) = x^2 def gradient_descent(x_start, learning_rate, iterations): x = x_start for i in range(iterations): gradient = 2 * x x = x - learning_rate * gradient if i % 10 == 0: f_value = x ** 2 print(f"Iteration {i}: x = {x:.4f}, f(x) = {f_value:.4f}") return x final_x = gradient_descent(5.0, 0.1, 100) print(f"\nFinal minimum at x = {final_x:.6f}") print(f"Minimum value f(x) = {final_x**2:.6f}")
