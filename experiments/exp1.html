Experiment 1: Gradient Descent

1) Aim
Understand and implement Gradient Descent for the given data:
  • Input $x=[1,2,3,4]$
  • Target $y=[1,4,9,16]$
2) Description

Gradient Descent is a foundational optimization algorithm that iteratively minimizes a loss function by moving in the direction opposite to the gradient. The gradient represents the direction of steepest increase, so moving opposite to it leads toward minima. This method is the backbone of modern machine learning because it scales well to high-dimensional problems and large datasets.

The core insight is that the gradient encodes both the direction and magnitude of change needed to reduce the loss. At each iteration, parameters are adjusted by a small step (controlled by the learning rate) in the negative gradient direction. This process continues until convergence, meaning the loss stabilizes and significant improvements become negligible.

Gradient Descent's effectiveness depends on problem structure, learning rate selection, and initial conditions. It works well for smooth, differentiable loss functions and is computationally efficient since it only requires first-order derivatives (gradients).

3) Important Formulas

Model:

$$\hat{y}_i=x_i^2$$

Loss (Mean Squared Error):

$$J=\frac{1}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)^2$$

In this case, the function is already defined, so there are no parameters to optimize with Gradient Descent. However, we can analyze the loss:

$$J=\frac{1}{4}[(1-1)^2+(4-4)^2+(9-9)^2+(16-16)^2]$$
4) Algorithm (Step by Step)
  1. Initialize input values $x=[1,2,3,4]$ and target $y=[1,4,9,16]$.
  2. Compute predictions using $\hat{y}=x^2$ for each input.
  3. Compute loss $J$ (Mean Squared Error).
  4. Compare predicted and actual values to understand the fit.
  5. Report predictions, loss, and analysis.
5) Mathematical Procedure (Complete Step-by-Step)

Given: $x=[1,2,3,4]$, $y=[1,4,9,16]$, function $f(x)=x^2$

1. Compute predictions using $\hat{y}=x^2$: $$\hat{y}=[1^2, 2^2, 3^2, 4^2]=[1,4,9,16]$$
2. Compute errors: $$e=\hat{y}-y=[1-1, 4-4, 9-9, 16-16]=[0,0,0,0]$$
3. Compute loss (Mean Squared Error): $$J=\frac{1}{4}(0^2+0^2+0^2+0^2)=0$$

4. The function $f(x)=x^2$ perfectly fits the data, resulting in zero loss.

6) Given Input
  • $x=[1,2,3,4]$
  • $y=[1,4,9,16]$
7) Python Implementation & Output
import numpy as np # Given input x = np.array([1, 2, 3, 4], dtype=float) y = np.array([1, 4, 9, 16], dtype=float) # Function f(x) = x^2 def f(x): return x ** 2 # Compute predictions using f(x) = x^2 y_pred = f(x) error = y_pred - y # Compute loss n = len(x) loss = (1 / n) * np.sum(error ** 2) print("INPUT") print("x =", x) print("y =", y) print("\nOUTPUT") print("Function: f(x) = x^2") print(f"Predicted y: {np.round(y_pred, 4)}") print(f"Errors: {np.round(error, 4)}") print(f"Mean Squared Error (MSE) Loss: {loss:.6f}") print("\nProgram") print("Quadratic function evaluation completed successfully.")
