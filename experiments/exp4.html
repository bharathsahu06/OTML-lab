Experiment 4: Constrained Minimization Using CVXPY

1) Aim
Minimize the function $$f(x,y)=x^2+y^2$$ subject to constraints: $$x+y\ge 1,\quad x\ge 0,\quad y\ge 0$$ using CVXPY.
2) Description

This experiment solves a convex optimization problem with a quadratic objective and linear inequality constraints. The objective $x^2+y^2$ is convex, and all constraints are also convex, so a global minimum exists and can be obtained reliably using CVXPY. CVXPY lets us define decision variables, objective function, and constraints in mathematical form, then solves the problem using an appropriate convex solver.

The method followed is: formulate the optimization model in standard form, apply all constraints exactly as given, and call the solver to compute the optimal feasible point. After solving, we verify that the returned values satisfy $x+y\ge 1$, $x\ge 0$, and $y\ge 0$, and then report the minimum objective value. Because the problem is convex, the obtained result is the true global optimum, not a local approximation.

3) Important Formulas

Objective: $$\min_{x,y}\; f(x,y)=x^2+y^2$$

Constraints: $$x+y\ge 1, \quad x\ge 0, \quad y\ge 0$$

Equivalent CVXPY form: $$\text{Minimize}(x^2+y^2)\quad \text{subject to}\quad [x+y\ge1,\;x\ge0,\;y\ge0]$$

4) Important Functions Used
  • cp.Variable(): creates optimization variables.
  • cp.Minimize(expr): defines minimization objective.
  • cp.Problem(objective, constraints): builds optimization problem.
  • problem.solve(): runs solver and computes optimum.
  • problem.status, problem.value: solver status and optimal objective value.
5) Algorithm (Step by Step)
  1. Import CVXPY.
  2. Define scalar decision variables $x$ and $y$.
  3. Define objective function $x^2+y^2$.
  4. Define constraints $x+y\ge1$, $x\ge0$, $y\ge0$.
  5. Create optimization problem using objective and constraints.
  6. Solve the problem with problem.solve().
  7. Read optimal values of $x$, $y$, and objective.
  8. Verify constraints numerically and print output.
6) Mathematical Procedure

Given: $$\min x^2+y^2\quad \text{s.t.}\quad x+y\ge1,\;x\ge0,\;y\ge0$$

At optimum, smallest Euclidean norm point in feasible set occurs on boundary $x+y=1$ in first quadrant. So set: $$y=1-x,\quad 0\le x\le1$$

Substitute into objective: $$f(x)=x^2+(1-x)^2=2x^2-2x+1$$

Differentiate: $$\frac{df}{dx}=4x-2=0\Rightarrow x=\frac{1}{2}$$

Hence: $$y=\frac{1}{2}$$

Minimum value: $$f\left(\frac12,\frac12\right)=\frac14+\frac14=\frac12=0.5$$

7) Given Input
  • Objective: $f(x,y)=x^2+y^2$
  • Constraints: $x+y\ge1$, $x\ge0$, $y\ge0$
import cvxpy as cp # Decision variables x = cp.Variable() y = cp.Variable() # Objective and constraints objective = cp.Minimize(x**2 + y**2) constraints = [x + y >= 1, x >= 0, y >= 0] # Problem definition and solve problem = cp.Problem(objective, constraints) optimal_value = problem.solve() print("INPUT") print("Objective: minimize f(x, y) = x^2 + y^2") print("Constraints: x + y >= 1, x >= 0, y >= 0") print("\nOUTPUT") print("Status:", problem.status) print(f"Optimal x: {x.value:.6f}") print(f"Optimal y: {y.value:.6f}") print(f"Minimum objective value: {optimal_value:.6f}") print("\nConstraint Check") print(f"x + y = {(x.value + y.value):.6f} (>= 1)") print(f"x >= 0 -> {x.value >= -1e-8}") print(f"y >= 0 -> {y.value >= -1e-8}") print("Program executed successfully.")
