Experiment 4: Minimize f(x,y)=x^2+y^2 (Using CVXPY and Without CVXPY)

1) Aim

a) Minimize $$f(x,y)=x^2+y^2$$ subject to $$x+y\ge 1,\;x\ge0,\;y\ge0$$ using CVXPY.

b) Minimize the same problem without CVXPY.

2) Description

This is a convex optimization problem, so the global minimum is guaranteed. The feasible region is the first-quadrant half-plane above the line $x+y=1$.

Using CVXPY gives a solver-based answer directly. Without CVXPY, we solve analytically by substituting $y=1-x$ on the active boundary and minimizing a 1D quadratic.

Both methods should return the same optimum: $x=y=0.5$ and minimum value $0.5$.

3) Important Formulas

Objective: $$\min\;x^2+y^2$$

Constraints: $$x+y\ge1,\;x\ge0,\;y\ge0$$

Boundary reduction: $$y=1-x,\;0\le x\le1$$

Reduced function: $$g(x)=x^2+(1-x)^2=2x^2-2x+1$$

Minimum at: $$g'(x)=4x-2=0\Rightarrow x=\frac12,\;y=\frac12$$

4A) Python Implementation Using CVXPY
import cvxpy as cp x = cp.Variable() y = cp.Variable() objective = cp.Minimize(x**2 + y**2) constraints = [x + y >= 1, x >= 0, y >= 0] problem = cp.Problem(objective, constraints) value = problem.solve() print("Status:", problem.status) print("x* =", round(float(x.value), 6)) print("y* =", round(float(y.value), 6)) print("f(x*,y*) =", round(float(value), 6))
4B) Python Implementation Without CVXPY
import numpy as np # Analytical solution on active boundary x + y = 1 x_star = 0.5 y_star = 1 - x_star f_star = x_star**2 + y_star**2 print("x* =", x_star) print("y* =", y_star) print("f(x*,y*) =", f_star) print("Constraint check:", (x_star + y_star) >= 1 and x_star >= 0 and y_star >= 0)
x* = 0.5 y* = 0.5 f(x*,y*) = 0.5 Constraint check: True
