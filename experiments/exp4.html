Experiment 4: Quadratic Programming with CVXPY

📌 Aim

To solve a constrained quadratic optimization problem using CVXPY: minimize f(x,y) = x² + y² subject to constraints x+y ≥ 1, x ≥ 0, y ≥ 0. This demonstrates convex optimization using a professional library.

📝 Description

Quadratic programming optimizes a quadratic objective subject to constraints, and CVXPY provides a clear way to express such models in code. The user defines variables, an objective, and constraints, and the solver handles the numerical optimization behind the scenes.

These problems are important in convex optimization because they often have well-structured solution behavior and strong theoretical guarantees. Working with CVXPY also helps separate mathematical modeling from the internal details of the optimization solver.

  • The objective function measures what must be minimized
  • The constraints define the feasible region of allowed solutions
  • CVXPY converts the mathematical model into a solvable optimization program
🔧 Algorithm
Step 1: Define decision variables x and y
Step 2: Define objective function: minimize x² + y²
Step 3: Add constraints: x+y ≥ 1, x ≥ 0, y ≥ 0
Step 4: Create optimization problem
Step 5: Solve the problem
Step 6: Extract and display results

📊 Mathematical Procedure

Problem from Aim: minimize f(x,y)=x²+y² subject to x+y≥1, x≥0, y≥0.

Feasible Region:
First-quadrant half-plane with x+y≥1 (an unbounded convex region).
Optimal Point:
For minimum norm in this feasible set, optimum lies on active boundary x + y = 1.
By symmetry at optimum x = y.
Therefore: x = 0.5, y = 0.5
Verification:
f(0.5, 0.5) = 0.5² + 0.5² = 0.25 + 0.25 = 0.5
Constraint check: 0.5 + 0.5 = 1 ✓ (satisfies ≥ 1)
💻 Simple Python Program

Install CVXPY first: pip install cvxpy

import cvxpy as cp # Define variables x = cp.Variable() y = cp.Variable() # Define objective: minimize x² + y² objective = cp.Minimize(x**2 + y**2) # Define constraints constraints = [ x + y >= 1, # x + y >= 1 x >= 0, # x >= 0 y >= 0 # y >= 0 ] # Create and solve problem problem = cp.Problem(objective, constraints) problem.solve() # Display results print(f"Optimal x = {x.value:.6f}") print(f"Optimal y = {y.value:.6f}") print(f"Minimum value = {objective.value:.6f}") print(f"Status: {problem.status}")
