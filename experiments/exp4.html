Experiment 4: Quadratic Programming with CVXPY

📌 Aim

To solve a constrained quadratic optimization problem using CVXPY: minimize f(x,y) = x² + y² subject to constraints x+y ≥ 1, x ≥ 0, y ≥ 0. This demonstrates convex optimization using a professional library.

CVXPY is a Python library for convex optimization. It allows defining optimization problems in a user-friendly way and automatically solves them using appropriate algorithms. This problem minimizes a quadratic objective function while satisfying linear inequality constraints.

📊 Mathematical Procedure

Optimal Solution Analysis:

Feasible Region:
The region where all constraints are satisfied forms a triangle with vertices at (0,1), (1,0), and (0,0) excluded.
Optimal Point:
Due to symmetry and convexity, the optimal point lies on the line x + y = 1 where x = y
Therefore: x = 0.5, y = 0.5
Verification:
f(0.5, 0.5) = 0.5² + 0.5² = 0.25 + 0.25 = 0.5
Constraint check: 0.5 + 0.5 = 1 ✓ (satisfies ≥ 1)
💻 Simple Python Program

Install CVXPY first: pip install cvxpy

import cvxpy as cp # Define variables x = cp.Variable() y = cp.Variable() # Define objective: minimize x² + y² objective = cp.Minimize(x**2 + y**2) # Define constraints constraints = [ x + y >= 1, # x + y >= 1 x >= 0, # x >= 0 y >= 0 # y >= 0 ] # Create and solve problem problem = cp.Problem(objective, constraints) problem.solve() # Display results print(f"Optimal x = {x.value:.6f}") print(f"Optimal y = {y.value:.6f}") print(f"Minimum value = {objective.value:.6f}") print(f"Status: {problem.status}")
