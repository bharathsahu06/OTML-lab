Quadratic Optimization (CVXPY)

🎯 Aim

To solve a constrained quadratic optimization problem using the CVXPY library, finding the minimum of a circular dish function subject to half-plane constraints.

📐 Mathematical Procedure
Part A: Problem Formulation

Objective function: \( \text{minimize } f(x, y) = x^2 + y^2 \)

Subject to:

  • \( x + y \ge 1 \)
  • \( x \ge 0, y \ge 0 \)
Part B: Analytical Solution

The unconstrained minimum is at \((0,0)\). However, this point violates \( x+y \ge 1 \). The closest point to the origin on the line \( x+y = 1 \) is the intersection with the normal line \( y = x \).

\( x + x = 1 \implies x = 0.5, y = 0.5 \)

Minimum Value: \( f(0.5, 0.5) = 0.5^2 + 0.5^2 = 0.5 \)

💻 Python Implementation
import cvxpy as cp # Define variables x = cp.Variable() y = cp.Variable() # Define objective objective = cp.Minimize(x**2 + y**2) # Define constraints constraints = [x + y >= 1, x >= 0, y >= 0] # Solve problem prob = cp.Problem(objective, constraints) prob.solve() print(f"Status: {prob.status}") print(f"Optimal x: {x.value:.6f}") print(f"Optimal y: {y.value:.6f}") print(f"Minimum value: {prob.value:.6f}")
📊 Expected Output
Terminal Console
Status: optimal
Optimal x: 0.500000
Optimal y: 0.500000
Minimum value: 0.500000
✅ Result & Verification
CVXPY successfully identifies the optimal point at (0.5, 0.5). If we set \( x+y \ge 10 \), the optimal value would increase to 50, showing how constraints shift the equilibrium.
