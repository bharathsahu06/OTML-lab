Experiment 12: Group Lasso for Feature Selection

Home / Experiment 12
Aim

To implement Group Lasso regularization to achieve group-level sparsity in linear models, ensuring that related features are selected or rejected as a single unit.

Description

Standard Lasso promotes individual sparsity (setting specific weights to zero). Group Lasso generalizes this by promoting sparsity among pre-defined groups of variables. If one variable in a group is important, the whole group is kept; otherwise, the whole group is penalized to zero.

\( J(\mathbf{w}) = \frac{1}{n} \sum (y_i - \hat{y}_i)^2 + \lambda \sum_{g \in Groups} \|\mathbf{w}_g\|_2 \)

Groups in this experiment:
Group 1: \(\{w_1, w_2\}\)
Group 2: \(\{w_3, w_4\}\)

Algorithm
  1. Define feature groups and initialize weights.
  2. Compute predictions and Mean Squared Error.
  3. For each group \( g \):
    • Compute the L2-norm of the weights in the group: \( \|\mathbf{w}_g\|_2 = \sqrt{\sum w_j^2} \).
    • Compute the penalty term \( \lambda \|\mathbf{w}_g\|_2 \).
  4. Update weights using Proximal Gradient Descent or a specialized solver.
Mathematical Procedure

The sub-gradient of the group penalty \( \lambda \|\mathbf{w}_g\|_2 \) with respect to a weight \( w_j \) in group \( g \) is:

\( \frac{\partial}{\partial w_j} (\lambda \|\mathbf{w}_g\|_2) = \lambda \frac{w_j}{\|\mathbf{w}_g\|_2} \) (if \( \|\mathbf{w}_g\|_2 \neq 0 \))
Simple Python Program
import numpy as np
from group_lasso import GroupLasso

# Data with 4 features (2 groups)
X = np.random.randn(10, 4)
y = 3*X[:, 0] + 2*X[:, 1] + np.random.randn(10)*0.1

# Define group structure: features 0,1 are Group 0; features 2,3 are Group 1
groups = np.array([0, 0, 1, 1])

model = GroupLasso(
    groups=groups,
    group_reg=0.01,
    l1_reg=0.0,
    n_iter=1000,
    fit_intercept=False
)

model.fit(X, y)
print("Coefficients:", np.round(model.coef_.flatten(), 4))
