Experiment 12: Mini-batch Gradient Descent

📌 Aim

To minimize the loss function of linear regression using mini-batch gradient descent for dataset x=[1,2,3,4], y=[3,5,7,9]. Mini-batch GD is a compromise between batch and stochastic methods.

📝 Description

Mini-batch gradient descent is a compromise between batch gradient descent and stochastic gradient descent. Instead of using the full dataset or a single sample for each update, it computes gradients on small groups of samples called mini-batches.

This gives a useful balance between computational efficiency and stability. The method is widely used in machine learning because batch size can be tuned to control update noise, memory use, speed, and convergence behavior during training.

  • Each mini-batch provides a partial but useful estimate of the full gradient
  • Repeated epochs gradually improve the model parameters
  • Batch size affects computational cost, stability, and convergence behavior
🔧 Algorithm
Step 1: Initialize weights randomly
Step 2: Divide data into mini-batches
Step 3: For each batch, compute gradient
Step 4: Update weights
Step 5: Repeat for multiple epochs
📐 Formulas
Linear Model:
ŷ = wx + b
Loss (MSE):
L = (1/m) × Σ(yᵢ - ŷᵢ)²
Gradient:
∂L/∂w = -(2/m) × Σ(yᵢ - ŷᵢ) × xᵢ
📊 Mathematical Procedure

Model derivation from data: x=[1,2,3,4], y=[3,5,7,9] follows y = 2x + 1.

Check: 2(1)+1=3, 2(2)+1=5, 2(3)+1=7, 2(4)+1=9.
Target parameters: w*=2, b*=1.
Optimization: mini-batch updates (batch size 2) gradually recover w and b from random initialization.
💻 Simple Python Program
import numpy as np # Mini-batch Gradient Descent for Linear Regression X = np.array([1, 2, 3, 4]).reshape(-1, 1) y = np.array([3, 5, 7, 9]) # Initialize w = 0.0 b = 0.0 lr = 0.01 batch_size = 2 epochs = 50 for epoch in range(epochs): # Shuffle data indices = np.random.permutation(len(X)) X_shuffled = X[indices] y_shuffled = y[indices] # Mini-batches for i in range(0, len(X), batch_size): X_batch = X_shuffled[i:i+batch_size] y_batch = y_shuffled[i:i+batch_size] # Predictions y_pred = w*X_batch + b # Gradients dw = -2/batch_size * np.sum((y_batch - y_pred) * X_batch) db = -2/batch_size * np.sum(y_batch - y_pred) # Update w -= lr * dw b -= lr * db # Loss y_pred_all = w*X + b loss = np.mean((y - y_pred_all)**2) if epoch % 10 == 0: print(f"Epoch {epoch}: Loss = {loss:.4f}, w = {w:.4f}, b = {b:.4f}") print(f"\nFinal: y = {w:.4f}x + {b:.4f}")
