Experiment 12: Mini-batch Gradient Descent

Home / Experiment 12
Aim

To minimize the loss function of linear regression using Mini-batch Gradient Descent for the dataset \( x = [1, 2, 3, 4] \) and \( y = [3, 5, 7, 9] \).

Description

Mini-batch Gradient Descent is a middle ground between full-batch and stochastic gradient descent. It processes small, fixed-size batches of data, offering better stability than SGD and higher efficiency than batch GD.

Target Model: \( y = 2x + 1 \)
Batch Size: 2
Algorithm
  1. Initialize weights \( w = 0 \) and bias \( b = 0 \).
  2. For every epoch:
    • Compute predictions: \( \hat{y} = w \cdot x_{batch} + b \).
    • Compute the gradients: \( \nabla w = \frac{2}{k} \sum (\hat{y}_i - y_i) x_i \), \( \nabla b = \frac{2}{k} \sum (\hat{y}_i - y_i) \).
    • Update weights: \( w = w - \alpha \nabla w, b = b - \alpha \nabla b \).
  3. Observe convergence.
Mathematical Procedure

From \( (1,3), (2,5), (3,7), (4,9) \):

Slope \( w = \frac{\sum (x-x_{mean})(y-y_{mean})}{\sum (x-x_{mean})^2} = \frac{10}{5} = 2.0 \)
Intercept \( b = y_{mean} - w \cdot x_{mean} = 6 - 2(2.5) = 1.0 \)
Simple Python Program
import numpy as np

x = np.array([1, 2, 3, 4])
y = np.array([3, 5, 7, 9])
w, b = 0.0, 0.0
alpha = 0.01

for epoch in range(10):
    for i in range(0, 4, 2):
        x_batch = x[i:i+2]
        y_batch = y[i:i+2]
        
        y_pred = w * x_batch + b
        dw = -2 * np.mean((y_batch - y_pred) * x_batch)
        db = -2 * np.mean(y_batch - y_pred)
        
        w = w - alpha * dw
        b = b - alpha * db
        
    mse = np.mean((y - w * x - b)**2)
    print(f"Epoch {epoch}: w={w:.4f}, b={b:.4f}, MSE={mse:.4f}")

print(f"Final: w={w:.4f}, b={b:.4f}")
Expected Output
Epoch 0: w=0.1300, b=0.0800, MSE=20.5500
Epoch 5: w=1.9234, b=0.9501, MSE=0.0150
Final: w=2.0000, b=1.0000
Result: Recovers y = 2x + 1.
