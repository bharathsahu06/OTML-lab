Mini-batch Gradient Descent (MBGD)

🎯 Aim

To implement Mini-batch Gradient Descent, balancing the stability of batch methods with the computational efficiency of stochastic updates.

📐 Mathematical Procedure
Part A: Batch Formulation

Objective: Fit \( y = mx + c \). Divide the full dataset into small clusters (batches).

Dataset: \( x=[1, 2, 3, 4], y=[3, 5, 7, 9] \)

Batch size = 2 (e.g., Batch 1: \((1,3), (2,5)\); Batch 2: \((3,7), (4,9)\))

Part B: Batch-wise Update Rule
\( \nabla L = \frac{1}{B} \sum_{i \in \text{Batch}} \text{grad}(i) \)
💻 Python Implementation
import numpy as np # Full dataset X = np.array([1, 2, 3, 4]) y = np.array([3, 5, 7, 9]) m, c = 0.0, 0.0 alpha = 0.01 batch_size = 2 for epoch in range(5): for i in range(0, len(X), batch_size): xb = X[i:i+batch_size] yb = y[i:i+batch_size] # Batch Gradient error = yb - (m * xb + c) m_grad = -2 * np.mean(error * xb) c_grad = -2 * np.mean(error) m = m - alpha * m_grad c = c - alpha * c_grad print(f"Epoch {epoch+1}: m = {m:.4f}, c = {c:.4f}") print(f"\nFinal Parameters: m = {m:.6f}, c = {c:.6f}")
