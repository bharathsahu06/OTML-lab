Experiment 6: Quadratic Optimization with CVXPY

Home / Experiment 6
Aim

To minimize the quadratic objective function \( f(x) = x_1^2 + x_2^2 \) subject to the linear constraints \( x_1 + x_2 \ge 1 \), \( x_1 \ge 0, x_2 \ge 0 \) using CVXPY.

Description

This experiment demonstrates solving a quadratic program (QP) using the CVXPY library. The objective function is the squared distance from the origin, and the feasible region is defined by a linear inequalities.

Problem: \( \min x_1^2 + x_2^2 \)
Subject to: \( x_1 + x_2 \ge 1, x_1, x_2 \ge 0 \)
Algorithm
  1. Define variables \( x_1, x_2 \) using \( \texttt{cp.Variable()} \).
  2. Construct the quadratic objective function: \( \texttt{cp.Minimize(x1**2 + x2**2)} \).
  3. Define the constraint list: \( [x_1 + x_2 \ge 1, x_1 \ge 0, x_2 \ge 0] \).
  4. Solve the problem using \( \texttt{cp.Problem(objective, constraints).solve()} \).
  5. Verify that the solution matches the analytical point \( (0.5, 0.5) \).
Simple Python Program
import cvxpy as cp

x1 = cp.Variable()
x2 = cp.Variable()

objective = cp.Minimize(x1**2 + x2**2)
constraints = [x1 + x2 >= 1, x1 >= 0, x2 >= 0]

problem = cp.Problem(objective, constraints)
result = problem.solve()

print(f"Optimal x1: {x1.value:.4f}")
print(f"Optimal x2: {x2.value:.4f}")
print(f"Minimum Value: {result:.4f}")
Expected Output
Optimal x1: 0.5000
Optimal x2: 0.5000
Minimum Value: 0.5000
Verification: Point (0.5, 0.5) satisfies x1+x2=1.
