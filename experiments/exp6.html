Experiment 6: Quadratic Objective with Linear Constraints (CVXPY)

📌 Aim

To minimize a quadratic objective function f(x₁,x₂) = x₁² + x₂² subject to linear constraints using CVXPY: x₁+x₂ ≥ 1, x₁ ≥ 0, x₂ ≥ 0.

📝 Description

A convex quadratic optimization problem with linear constraints has a smooth curved objective surface together with a feasible region defined by inequalities. The quadratic terms control the shape of the objective, while the constraints determine where the solution is allowed to lie.

The goal is to find the feasible point that gives the smallest objective value. This shows how optimization is not only about minimizing a function, but also about respecting restrictions that come from the problem setting.

  • Quadratic terms produce a smooth objective with a clear notion of curvature
  • Linear constraints define the boundary of feasible solutions
  • CVXPY solves the numerical problem once the mathematical model is specified
🔧 Algorithm
Step 1: Define variables x₁ and x₂
Step 2: Define quadratic objective
Step 3: Add linear constraints
Step 4: Solve the problem
📐 Formulas
Minimize:
f(x₁, x₂) = x₁² + x₂²
Subject to:
x₁ + x₂ ≥ 1
x₁ ≥ 0
x₂ ≥ 0
📊 Mathematical Procedure
Optimal Point:
By symmetry, x₁ = x₂ = 0.5
This lies on the constraint boundary x₁ + x₂ = 1
Optimal Value:
f(0.5, 0.5) = 0.25 + 0.25 = 0.5
💻 Simple Python Program
import cvxpy as cp # Define variables x1 = cp.Variable() x2 = cp.Variable() # Objective: minimize x₁² + x₂² objective = cp.Minimize(x1**2 + x2**2) # Constraints constraints = [ x1 + x2 >= 1, x1 >= 0, x2 >= 0 ] # Solve problem = cp.Problem(objective, constraints) problem.solve() # Results print(f"x₁ = {x1.value:.6f}") print(f"x₂ = {x2.value:.6f}") print(f"Minimum = {objective.value:.6f}")
