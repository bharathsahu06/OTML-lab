Experiment 6: Minimize f(x1,x2)=x1^2+x2^2 (Using CVXPY and Without CVXPY)

1) Aim

a) Minimize $$f(x_1,x_2)=x_1^2+x_2^2$$ subject to $$x_1+x_2\ge1,\;x_1\ge0,\;x_2\ge0$$ using CVXPY.

b) Minimize the same problem without CVXPY.

2) Description

This is a convex quadratic programming problem. Since the objective is strictly convex and constraints are linear, the solution is unique and global.

Using CVXPY gives the numerical optimum directly. Without CVXPY, we solve analytically by reducing to the active boundary $x_1+x_2=1$ and minimizing a single-variable quadratic.

Both methods return $x_1=x_2=0.5$ with minimum value $0.5$.

3) Important Formulas

Objective: $$\min\;x_1^2+x_2^2$$

Constraints: $$x_1+x_2\ge1,\;x_1\ge0,\;x_2\ge0$$

Boundary: $$x_2=1-x_1,\;0\le x_1\le1$$

Reduced objective: $$g(x_1)=x_1^2+(1-x_1)^2$$

Minimum: $$g'(x_1)=4x_1-2=0\Rightarrow x_1=0.5,\;x_2=0.5$$

4A) Python Implementation Using CVXPY
import cvxpy as cp x1 = cp.Variable() x2 = cp.Variable() objective = cp.Minimize(x1**2 + x2**2) constraints = [x1 + x2 >= 1, x1 >= 0, x2 >= 0] problem = cp.Problem(objective, constraints) value = problem.solve() print("Status:", problem.status) print("x1* =", round(float(x1.value), 6)) print("x2* =", round(float(x2.value), 6)) print("f(x1*,x2*) =", round(float(value), 6))
4B) Python Implementation Without CVXPY
# Analytical solution x1_star = 0.5 x2_star = 1 - x1_star f_star = x1_star**2 + x2_star**2 print("x1* =", x1_star) print("x2* =", x2_star) print("f(x1*,x2*) =", f_star) print("Constraint check:", (x1_star + x2_star) >= 1 and x1_star >= 0 and x2_star >= 0)
x1* = 0.5 x2* = 0.5 f(x1*,x2*) = 0.5 Constraint check: True
