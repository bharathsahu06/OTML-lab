Experiment 06: LP and QP using CVXPY and PuLP

Home / Experiment 06
Aim

To use industry-standard optimization libraries like CVXPY and PuLP to solve linear and quadratic programming problems.

Description

Optimization solvers provide a high-level interface to define variables, objectives, and constraints without manually deriving gradients or simplex tables. CVXPY is excellent for convex optimization (including QP), while PuLP is a popular library specifically for Linear Programming.

Problem 1 (CVXPY): Minimize \( x^2 + y^2 \) s.t. \( x + y \ge 1, x, y \ge 0 \).

Problem 2 (PuLP): Minimize \( 3x + 2y \) s.t. \( x + y \ge 4, x, y \ge 0 \).

Algorithm
  1. Install and import the required library (cvxpy or pulp).
  2. Define decision variables using cp.Variable() or LpVariable().
  3. Define the objective function.
  4. Define constraints as a list.
  5. Call the solve() method.
  6. Extract and print the optimal variable values and objective value.
Simple Python Program
import cvxpy as cp
from pulp import LpProblem, LpMinimize, LpVariable

# 1. Quadratic Program using CVXPY
x_c = cp.Variable()
y_c = cp.Variable()
prob_cvx = cp.Problem(cp.Minimize(x_c**2 + y_c**2), [x_c + y_c >= 1, x_c >= 0, y_c >= 0])
prob_cvx.solve()
print(f"CVXPY: x={x_c.value:.2f}, y={y_c.value:.2f}, val={prob_cvx.value:.2f}")

# 2. Linear Program using PuLP
lp = LpProblem("Minimize_3x_2y", LpMinimize)
x_p = LpVariable("x", lowBound=0)
y_p = LpVariable("y", lowBound=0)
lp += 3*x_p + 2*y_p
lp += x_p + y_p >= 4
lp.solve()
print(f"PuLP: x={x_p.varValue:.2f}, y={y_p.varValue:.2f}, val={lp.objective.value():.2f}")
