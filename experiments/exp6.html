Quadratic Optimization (Vectorized)

🎯 Aim

To demonstrate quadratic optimization using vectorized variables and solving for non-negative iterates under complex linear constraints.

📐 Mathematical Procedure
Part A: Logic

Minimize the squared L2-norm of the weight vector subject to sum constraints.

\( \min ||w||_2^2 \text{ subject to } A^Tw \ge b \)
💻 Python Implementation
import cvxpy as cp # Define variables x1 = cp.Variable() x2 = cp.Variable() # Problem components objective = cp.Minimize(x1**2 + x2**2) constraints = [x1 + x2 >= 1, x1 >= 0, x2 >= 0] # Compute solution prob = cp.Problem(objective, constraints) prob.solve() print(f"Optimal x1: {x1.value:.6f}") print(f"Optimal x2: {x2.value:.6f}") print(f"Min value: {prob.value:.6f}")
