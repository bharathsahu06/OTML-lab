Experiment 6: Quadratic Objective with Linear Constraints Using CVXPY

1) Aim
Minimize the quadratic objective function $$f(x_1,x_2)=x_1^2+x_2^2$$ subject to linear constraints: $$x_1+x_2\ge1,\quad x_1\ge0,\quad x_2\ge0$$ using CVXPY.
2) Description

This experiment solves a convex quadratic optimization problem with linear inequality constraints. The objective $x_1^2+x_2^2$ is strictly convex, and the feasible region defined by the constraints is also convex, so the solution is unique and globally optimal. CVXPY is used to represent the optimization model in a natural mathematical form and solve it using an appropriate convex solver.

The method followed is: define decision variables $x_1$ and $x_2$, construct the quadratic objective, and enforce all linear constraints exactly as given. Then we solve the model and verify that the returned solution satisfies every constraint numerically. Because this is a convex optimization setup, the computed optimum is the true global minimum over the feasible set.

3) Important Formulas

Objective: $$\min_{x_1,x_2}\; x_1^2+x_2^2$$

Constraints: $$x_1+x_2\ge1, \quad x_1\ge0, \quad x_2\ge0$$

Equivalent CVXPY form: $$\text{Minimize}(x_1^2+x_2^2)\quad \text{subject to}\quad [x_1+x_2\ge1,\;x_1\ge0,\;x_2\ge0]$$

4) Important Functions Used
  • cp.Variable(): creates decision variables.
  • cp.Minimize(expr): defines objective for minimization.
  • cp.Problem(objective, constraints): builds optimization model.
  • problem.solve(): solves the optimization problem.
  • problem.status, problem.value: returns solver status and optimum value.
5) Algorithm (Step by Step)
  1. Import CVXPY.
  2. Define scalar variables $x_1$ and $x_2$.
  3. Define objective function $x_1^2+x_2^2$.
  4. Add constraints $x_1+x_2\ge1$, $x_1\ge0$, $x_2\ge0$.
  5. Build Problem(objective, constraints).
  6. Solve with problem.solve().
  7. Read optimal $x_1$, $x_2$, and objective value.
  8. Check constraint satisfaction and print results.
6) Mathematical Procedure

Given: $$\min x_1^2+x_2^2\quad \text{s.t.}\quad x_1+x_2\ge1,\;x_1\ge0,\;x_2\ge0$$

At optimum, closest feasible point to origin lies on boundary $x_1+x_2=1$ in first quadrant.

Let: $$x_2=1-x_1,\quad 0\le x_1\le1$$

Substitute into objective: $$f(x_1)=x_1^2+(1-x_1)^2=2x_1^2-2x_1+1$$

Differentiate and set to zero: $$\frac{df}{dx_1}=4x_1-2=0\Rightarrow x_1=\frac12$$

Hence: $$x_2=\frac12$$

Minimum value: $$f\left(\frac12,\frac12\right)=\frac14+\frac14=\frac12=0.5$$

7) Given Input
  • Objective: $f(x_1,x_2)=x_1^2+x_2^2$
  • Constraints: $x_1+x_2\ge1$, $x_1\ge0$, $x_2\ge0$
8) Python Implementation & Output
import cvxpy as cp # Decision variables x1 = cp.Variable() x2 = cp.Variable() # Objective and constraints objective = cp.Minimize(x1**2 + x2**2) constraints = [x1 + x2 >= 1, x1 >= 0, x2 >= 0] # Build and solve problem problem = cp.Problem(objective, constraints) optimal_value = problem.solve() print("INPUT") print("Objective: minimize f(x1, x2) = x1^2 + x2^2") print("Constraints: x1 + x2 >= 1, x1 >= 0, x2 >= 0") print("\nOUTPUT") print("Status:", problem.status) print(f"Optimal x1: {x1.value:.6f}") print(f"Optimal x2: {x2.value:.6f}") print(f"Minimum objective value: {optimal_value:.6f}") print("\nConstraint Check") print(f"x1 + x2 = {(x1.value + x2.value):.6f} (>= 1)") print(f"x1 >= 0 -> {x1.value >= -1e-8}") print(f"x2 >= 0 -> {x2.value >= -1e-8}") print("Program executed successfully.")
