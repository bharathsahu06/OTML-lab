Experiment 13: Parameter Averaging in Distributed Learning

Home / Experiment 13
Aim

To simulate a distributed learning architecture where multiple workers compute full model updates on local data shards and a central server averages the parameters (weights) instead of just gradients.

Description

Parameter Averaging (or Federated Averaging) reduces communication overhead by allowing workers to perform multiple local updates before synchronizing with the central server. The server then computes the mean of all local weight vectors to form the new global model.

\( \mathbf{w}_{global} = \frac{1}{M} \sum_{j=1}^M \mathbf{w}_j \)
Algorithm
  1. Initialize global weights \( \mathbf{w}_{global} \) at the server.
  2. For each communication round:
    • Distribute \( \mathbf{w}_{global} \) to all \( M \) workers.
    • Workers: Run \( E \) epochs of local SGD to produce local weights \( \mathbf{w}_j \).
    • Server: Receive \( \mathbf{w}_j \) from all workers and compute the average.
    • Update \( \mathbf{w}_{global} = \text{avg}(\mathbf{w}_j) \).
Simple Python Program
import numpy as np

# Data shards for 2 workers
W1 = {'x': np.array([1, 2]), 'y': np.array([3, 5])}
W2 = {'x': np.array([3, 4]), 'y': np.array([7, 9])}
w_global, b_global = 0.0, 0.0
lr = 0.01

for round in range(5):
    local_weights = []
    # Local Training
    for data in [W1, W2]:
        w, b = w_global, b_global
        for _ in range(10): # Local epochs
            err = (w * data['x'] + b) - data['y']
            w -= lr * np.mean(err * data['x'])
            b -= lr * np.mean(err)
        local_weights.append((w, b))
    
    # Parameter Averaging
    w_global = np.mean([lw[0] for lw in local_weights])
    b_global = np.mean([lw[1] for lw in local_weights])
    print(f"Round {round}: w={w_global:.4f}, b={b_global:.4f}")

print(f"Final Global Model: y = {w_global:.2f}x + {b_global:.2f}")
