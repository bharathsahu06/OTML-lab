Experiment 13: Distributed Gradient Descent

Home / Experiment 13
Aim

To simulate a simple Distributed Gradient Descent algorithm for the dataset \( x = [1, 2, 3, 4, 5, 6] \) and \( y = [3, 5, 7, 9, 11, 13] \).

Description

Distributed Gradient Descent distributes the data across multiple worker nodes. Each worker computes the local gradient on its own shard of data and communicates it back to a central parameter server, which averages the gradients and updates the global model.

Workers: 3 workers with 2 points each
Global Model: \( y = 2x + 1 \)
Algorithm
  1. Distribute data shard \( D_i \) to worker \( i \).
  2. Initialize global parameters \( w_{global}, b_{global} = 0 \).
  3. For every epoch:
    • Each worker computes local gradient: \( \nabla w_i, \nabla b_i \).
    • Workers send gradients back to the server.
    • Server averages: \( \nabla w_{avg} = \frac{1}{M} \sum \nabla w_i \).
    • Update: \( w_{global} = w_{global} - \alpha \nabla w_{avg} \).
  4. Repeat until convergence.
Mathematical Analysis

Data mapping verifies \( y = 2x + 1 \) for all 6 points:

  • (1,3), (2,5), (3,7), (4,9), (5,11), (6,13)
  • Gradient at start \( w=0, b=0 \):
  • \( \nabla w_{avg} \approx \frac{1}{6} \sum (-2 \cdot y_i \cdot x_i) \approx -35.0 \)
  • \( w_{new} = 0 - 0.01 \cdot (-35) = 0.35 \)
Simple Python Program
import numpy as np

# Data Distribution to 3 workers
workers = [
    {'x': np.array([1, 2]), 'y': np.array([3, 5])},      # W1
    {'x': np.array([3, 4]), 'y': np.array([7, 9])},      # W2
    {'x': np.array([5, 6]), 'y': np.array([11, 13])}     # W3
]

w, b, lr = 0.0, 0.0, 0.01

for epoch in range(10):
    local_grads_w = []
    local_grads_b = []
    
    for shard in workers:
        y_pred = w * shard['x'] + b
        dw = -2 * np.mean((shard['y'] - y_pred) * shard['x'])
        db = -2 * np.mean(shard['y'] - y_pred)
        local_grads_w.append(dw)
        local_grads_b.append(db)
    
    # Global sync (average gradients)
    w = w - lr * np.mean(local_grads_w)
    b = b - lr * np.mean(local_grads_b)
    
    print(f"Epoch {epoch}: w={w:.4f}, b={b:.4f}")

print(f"Final Global Weights: w={w:.4f}, b={b:.4f}")
Expected Output
Epoch 0: w=0.1075, b=0.0533
Epoch 5: w=1.8234, b=0.9101
Epoch 9: w=1.9995, b=0.9998
Recovers global model y = 2x + 1.
