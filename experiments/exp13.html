Experiment 13: Distributed Gradient Descent

📌 Aim

To simulate a distributed gradient descent algorithm where data is split across multiple workers. Dataset: x=[1,2,3,4,5,6], y=[3,5,7,9,11,13]. Each worker processes a subset and contributes to parameter updates.

Distributed GD divides data among multiple processors/machines. Each processor:

  • Computes local gradients on its data subset
  • Sends gradients to a central server
  • Server aggregates and updates parameters
  • Weights distributed back to workers
🔧 Algorithm
Step 1: Initialize global weights
Step 2: Split data among workers
Step 3: Each worker computes local gradient
Step 4: Aggregate gradients at server
Step 5: Update global weights
Step 6: Broadcast to all workers
Step 7: Repeat until convergence
💻 Simple Python Program
import numpy as np # Distributed Gradient Descent Simulation # Global data X = np.array([1, 2, 3, 4, 5, 6]) y = np.array([3, 5, 7, 9, 11, 13]) num_workers = 3 # Split data among workers def split_data(X, y, num_workers): n = len(X) // num_workers workers_data = [] for i in range(num_workers): start = i * n end = start + n workers_data.append((X[start:end], y[start:end])) return workers_data # Worker computes gradient def worker_gradient(X_local, y_local, w, b): y_pred = w*X_local + b dw = -2/len(X_local) * np.sum((y_local - y_pred) * X_local) db = -2/len(X_local) * np.sum(y_local - y_pred) return dw, db # Initialize w = 0.0 b = 0.0 lr = 0.01 workers_data = split_data(X, y, num_workers) # Training for epoch in range(30): # Collect gradients from workers dw_list, db_list = [], [] for X_local, y_local in workers_data: dw, db = worker_gradient(X_local, y_local, w, b) dw_list.append(dw) db_list.append(db) # Average gradients avg_dw = np.mean(dw_list) avg_db = np.mean(db_list) # Update weights w -= lr * avg_dw b -= lr * avg_db # Loss y_pred = w*X + b loss = np.mean((y - y_pred)**2) if epoch % 10 == 0: print(f"Epoch {epoch}: Loss = {loss:.4f}, w = {w:.4f}") print(f"\nFinal: w = {w:.4f}, b = {b:.4f}")
