Distributed Gradient Averaging

🎯 Aim

To simulate a distributed learning architecture where multiple workers compute gradients on local data shards and a central server averages parameters.

📐 Mathematical Procedure
Part A: Worker Computation

Each worker \( j \) computes a local update \( w_j = w_{old} - \eta \nabla L_j \).

Part B: Server Aggregation
\( w_{new} = \frac{1}{M} \sum_{j=1}^M w_j \)
