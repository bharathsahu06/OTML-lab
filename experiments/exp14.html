Experiment 14: Optimizing Shifted Linear Systems

Home / Experiment 14
Aim

To implement Gradient Descent for linear systems with an explicit bias term (Shifted), ensuring the model correctly handles non-zero intercepts in regression problems.

Description

A "shifted" system is one where the target \( y \) does not pass through the origin. In this experiment, we minimize the Mean Squared Error for the relationship \( y = mx + c \). The shift \( c \) represents the vertical displacement of the line.

Model: \( y_i = m x_i + c + \epsilon \)

Dataset: \( x = [0, 1, 2, 3], y = [1, 3, 5, 7] \).

Algorithm
  1. Initialize parameters \( m \) and \( c \) randomly or at zero.
  2. Set learning rate \( \alpha \) and number of epochs.
  3. Repeat for each epoch:
    • Compute error: \( e_i = \hat{y}_i - y_i = (mx_i + c) - y_i \).
    • Compute partial derivatives for slope: \( \nabla m = \frac{2}{n} \sum e_i x_i \).
    • Compute partial derivatives for bias: \( \nabla c = \frac{2}{n} \sum e_i \).
    • Update: \( m = m - \alpha \nabla m, c = c - \alpha \nabla c \).
  4. Repeat until MSE converges.
Mathematical Procedure
Cost Function: \( J(m,c) = \frac{1}{n} \sum ((mx_i + c) - y_i)^2 \)
Gradient Update: \( \begin{pmatrix} m \\ c \end{pmatrix}_{k+1} = \begin{pmatrix} m \\ c \end{pmatrix}_{k} - \alpha \begin{pmatrix} \nabla m \\ \nabla c \end{pmatrix} \)
Simple Python Program
import numpy as np

x = np.array([0, 1, 2, 3], dtype=float)
y = np.array([1, 3, 5, 7], dtype=float)
m, c, lr = 0.0, 0.0, 0.1

for _ in range(100):
    y_pred = m * x + c
    grad_m = (2/len(x)) * np.sum((y_pred - y) * x)
    grad_c = (2/len(x)) * np.sum(y_pred - y)
    
    m -= lr * grad_m
    c -= lr * grad_c

print(f"Optimal Model: y = {m:.2f}x + {c:.2f}")
Expected Output
Initial Guess: m=0.0, c=0.0
Final Parameters:
  m = 2.00
  c = 1.00
Verification: y = 2x + 1. (Fits x=0, y=1; x=3, y=7 exactly).
