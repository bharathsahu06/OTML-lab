Optimizing Shifted Linear Systems

🎯 Aim

To implement Gradient Descent for a linear system with a non-zero shift (bias), demonstrating how the model handles a non-zero intercept.

📐 Mathematical Procedure
Part A: Model Specification

Relationship: \( y = mx + c \). Dataset: \( x=[0, 1, 2, 3], y=[1, 3, 5, 7] \).

Part B: Gradients
\( \frac{\partial L}{\partial m} = -2\sum (y - (mx+c))x \)
\( \frac{\partial L}{\partial c} = -2\sum (y - (mx+c)) \)
