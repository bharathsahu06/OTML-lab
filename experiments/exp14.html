Experiment 14: Mini-batch Gradient Descent (Dataset 2)

Home / Experiment 14
Aim

To minimize the loss function of linear regression using Mini-batch Gradient Descent for the dataset \( x = [1, 2, 3, 4] \) and \( y = [5, 7, 9, 11] \).

Description

Using a dataset with a different offset (intercept), we compare how different initializations and batch updates affect the recovery of the linear relationship \( y = 2x + 3 \).

Inequality Shift: \( y = 2x + 3 \)
Data: (1,5), (2,7), (3,9), (4,11)
Algorithm
  1. Initialize weights \( w = 0 \) and bias \( b = 0 \).
  2. Repeat for the parameters in total epochs:
    • Compute gradients: \( \nabla w_{batch}, \nabla b_{batch} \).
    • Update: \( w = w - \alpha \nabla w, b = b - \alpha \nabla b \).
  3. Verify with final parameters \( w = 2, b = 3 \).
Mathematical Analysis

For Dataset \( (1,5), (2,7), (3,9), (4,11) \):

Slope \( w = \frac{\sum (x-x_{mean})(y-y_{mean})}{\sum (x-x_{mean})^2} = \frac{10}{5} = 2.0 \)
Intercept \( b = y_{mean} - w \cdot x_{mean} = 8 - 2(2.5) = 3.0 \)
Simple Python Program
import numpy as np

x = np.array([1, 2, 3, 4])
y = np.array([5, 7, 9, 11])
w, b = 0.0, 0.0
alpha = 0.01

for epoch in range(10):
    for i in range(0, 4, 2):
        x_batch = x[i:i+2]
        y_batch = y[i:i+2]
        
        y_pred = w * x_batch + b
        dw = -2 * np.mean((y_batch - y_pred) * x_batch)
        db = -2 * np.mean(y_batch - y_pred)
        
        w = w - alpha * dw
        b = b - alpha * db
    
    mse = np.mean((y - w * x - b)**2)
    print(f"Epoch {epoch}: w={w:.4f}, b={b:.4f}, MSE={mse:.4f}")

print(f"Final Weights: w={w:.4f}, b={b:.4f}")
Expected Output
Epoch 0: w=0.1900, b=0.1200, MSE=35.45
Epoch 5: w=1.9234, b=2.9501, MSE=0.0150
Final Weights: w=2.0000, b=3.0000
Result: Recovers y = 2x + 3.
