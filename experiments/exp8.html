Experiment 8: Sub-gradient Method with L1-Regularization

📌 Aim

To implement the sub-gradient method for linear regression with L1-regularization (Lasso). Uses dataset: (1,2,5), (2,1,6), (3,4,7). L1-regularization promotes sparsity in the solution.

📝 Description

L1-regularized linear regression adds a penalty based on the absolute values of the coefficients. This discourages unnecessary parameters and encourages sparse solutions, which is why the method is closely related to feature selection.

Because the absolute value term is not differentiable at zero, the ordinary gradient is replaced by a sub-gradient. This makes the method a strong example of how optimization techniques are adapted for non-smooth machine learning objectives.

  • The regression term tries to fit the model to observed data
  • The L1 penalty pushes small coefficients toward zero
  • The sub-gradient method makes optimization possible even when the loss is not smooth everywhere
🔧 Algorithm
Step 1: Initialize weights w₁, w₂
Step 2: For each sample, compute prediction error
Step 3: Compute sub-gradient including L1 term
Step 4: Update weights
Step 5: Repeat until convergence
📐 Formulas
Loss Function with L1-Regularization:
L(w) = Σ(yᵢ - ŷᵢ)² + λ||w||₁
where λ is regularization strength
Sub-gradient:
∂L/∂w = -2Σ(yᵢ - ŷᵢ)xᵢ + λ×sign(w)
Update Rule:
w(n+1) = w(n) - α × sub-gradient
💻 Simple Python Program
import numpy as np # Dataset: (x1, x2, y) X = np.array([[1, 2], [2, 1], [3, 4]]) y = np.array([5, 6, 7]) # Initialize weights w = np.array([0.0, 0.0]) learning_rate = 0.01 lambda_reg = 0.1 # L1 regularization strength # Subgradient descent for epoch in range(100): # Predictions y_pred = X @ w # Error error = y_pred - y # Gradient grad = 2 * X.T @ error # Sub-gradient (add L1 penalty) sub_grad = grad + lambda_reg * np.sign(w) # Update w = w - learning_rate * sub_grad if epoch % 20 == 0: loss = np.mean(error**2) + lambda_reg * np.sum(np.abs(w)) print(f"Epoch {epoch}: Loss = {loss:.4f}, w = {w}") print(f"\nFinal w = {w}")
