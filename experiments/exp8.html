Experiment 08: Interior Point Method (Log-Barrier)

Home / Experiment 08
Aim

To implement the Interior Point method using a logarithmic barrier function to solve a constrained optimization problem.

Description

The Interior Point method converts a constrained problem into an unconstrained one by adding a barrier term to the objective function. As the barrier parameter \( \mu \) approaches zero, the solution of the barrier problem converges to the solution of the original constrained problem.

Problem: Minimize \( f(x) = x^2 \) subject to \( x > 1 \).

Barrier Objective: \( \phi(x; \mu) = x^2 - \mu \ln(x - 1) \)
Algorithm
  1. Choose an initial feasible point \( x_0 > 1 \) and initial \( \mu_0 > 0 \).
  2. While not converged:
    • Solve the unconstrained problem \( \min \phi(x; \mu) \) using Newton's method.
      • Grad: \( \nabla \phi = 2x - \frac{\mu}{x-1} \)
      • Hessian: \( \nabla^2 \phi = 2 + \frac{\mu}{(x-1)^2} \)
    • Reduce \( \mu \) in stages (e.g., \( \mu = \mu \times 0.1 \)).
Mathematical Procedure

Newton step for fixed \( \mu \):

\( x_{new} = x - \frac{2x - \frac{\mu}{x-1}}{2 + \frac{\mu}{(x-1)^2}} \)

Ensure \( x_{new} > 1 \) at each step to maintain feasibility inside the "interior".

Simple Python Program
import numpy as np

x = 2.0  # Initial feasible point
mu = 4.0 # Initial barrier parameter
tol = 1e-8

print("Iter | mu | x")
for k in range(12):
    # Newton method for fixed mu
    for _ in range(20):
        grad = 2.0 * x - mu / (x - 1.0)
        hess = 2.0 + mu / (x - 1.0)**2
        x = x - grad / hess
        if x <= 1.0: x = 1.1 # Stay in interior

    print(f"{k} | {mu:.6g} | {x:.8f}")
    
    if x - 1.0 < tol: break
    # Reduce mu
    if mu > 1.0: mu -= 1.0
    else: mu *= 0.1

print(f"Final Solution: {x}")
