Experiment 8: Sub-gradient Method with L1-Regularized Linear Regression

1) Aim
Implementation of the Sub-gradient Method with L1-regularized loss for the linear regression model: $$y = w_1x_1 + w_2x_2$$ using the dataset: $$(1,2,5),\;(2,1,6),\;(3,4,7)$$
2) Description

L1 Regularization and Sub-gradient Methods address the challenge of producing sparse solutions where many model parameters are exactly zero. L1 regularization (also called LASSO) adds penalty $\lambda||w||_1$ to the loss, pushing weights toward zero through a sharp "corner" at the origin. This corner creates non-differentiability—standard gradient descent fails because the gradient is undefined at zero.

Sub-gradient methods generalize gradient descent to handle non-smooth (non-differentiable) functions. Instead of a unique gradient, non-smooth functions have a set of valid gradient-like vectors called sub-gradients. The L1 norm has a simple sub-gradient: the sign function. Sub-gradient updates still descend the objective, maintaining the convergence guarantees of gradient descent while handling non-smoothness.

L1 regularization is crucial in high-dimensional machine learning where feature selection is desired. By encouraging sparsity, L1 helps identify important features and reduces overfitting. The sub-gradient method makes it computationally tractable to optimize L1-regularized objectives, enabling practical sparse learning in applications from text classification to signal processing.

3) Important Formulas

Model: $$\hat{y}_i = w_1x_{1,i} + w_2x_{2,i}$$

MSE part: $$J_{\text{mse}} = \frac{1}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)^2$$

L1-regularized objective: $$J(w_1,w_2)=\frac{1}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)^2 + \lambda\left(|w_1|+|w_2|\right)$$

Sub-gradients: $$\frac{\partial J}{\partial w_1}=\frac{2}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)x_{1,i}+\lambda\,\operatorname{sign}(w_1)$$ $$\frac{\partial J}{\partial w_2}=\frac{2}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)x_{2,i}+\lambda\,\operatorname{sign}(w_2)$$

where $\operatorname{sign}(0)$ is taken as $0$ in this implementation.

Update rules: $$w_1\leftarrow w_1-\alpha\frac{\partial J}{\partial w_1},\qquad w_2\leftarrow w_2-\alpha\frac{\partial J}{\partial w_2}$$

4) Important Functions Used
  • np.sign(): computes sign for L1 sub-gradient term.
  • subgradient_l1_regression(...): performs iterative sub-gradient updates.
  • regularized_loss(...): computes MSE + L1 regularization.
5) Algorithm (Step by Step)
  1. Initialize $w_1=0$, $w_2=0$.
  2. Compute predictions $\hat{y}$ for all samples.
  3. Compute error vector $e=\hat{y}-y$.
  4. Compute MSE gradient terms for $w_1,w_2$.
  5. Add L1 sub-gradient terms $\lambda\operatorname{sign}(w_1)$ and $\lambda\operatorname{sign}(w_2)$.
  6. Update parameters using learning rate $\alpha$.
  7. Repeat for required number of iterations.
  8. Report final weights, predictions, and objective value.
6) Mathematical Procedure (First Iteration)

Given data: $$(x_1,x_2,y)= (1,2,5),(2,1,6),(3,4,7),\quad n=3$$

Initialize $w_1=0, w_2=0$.

Predictions: $$\hat{y}=[0,0,0]$$
Errors: $$e=\hat{y}-y=[-5,-6,-7]$$
MSE gradient part: $$\frac{2}{3}\sum e_i x_{1,i}=\frac{2}{3}[(-5)(1)+(-6)(2)+(-7)(3)] = \frac{2}{3}(-38)=-25.3333$$ $$\frac{2}{3}\sum e_i x_{2,i}=\frac{2}{3}[(-5)(2)+(-6)(1)+(-7)(4)] = \frac{2}{3}(-44)=-29.3333$$

At start, $\operatorname{sign}(w_1)=\operatorname{sign}(w_2)=0$, so L1 term contributes $0$ in first step.

With $\alpha=0.01$:

$$w_1=0-0.01(-25.3333)=0.253333$$

$$w_2=0-0.01(-29.3333)=0.293333$$

7) Given Input
  • Dataset: $(1,2,5),(2,1,6),(3,4,7)$
  • Learning rate: $\alpha=0.01$
  • Regularization parameter: $\lambda=0.1$
  • Iterations: $1000$
8) Python Implementation & Output
import numpy as np # Data X = np.array([ [1.0, 2.0], [2.0, 1.0], [3.0, 4.0], ]) y = np.array([5.0, 6.0, 7.0]) # Parameters w = np.zeros(2) alpha = 0.01 lam = 0.1 iterations = 1000 n = len(y) for _ in range(iterations): # Prediction y_pred = X @ w error = y_pred - y # Gradient (MSE part) grad_mse = (2/n) * (X.T @ error) # Subgradient (L1 part) grad_l1 = lam * np.sign(w) # Total gradient grad = grad_mse + grad_l1 # Update w = w - alpha * grad # Output y_pred = X @ w loss = np.mean((y_pred - y)**2) + lam * np.sum(np.abs(w)) print("function: f(x) = wx + b") print("data points: (1,2,5), (2,1,6), (3,4,7)") print("w1 =", round(w[0], 6)) print("w2 =", round(w[1], 6)) print(f"Final Loss = {loss:.6f}") print("Predictions =", np.round(y_pred, 6))
