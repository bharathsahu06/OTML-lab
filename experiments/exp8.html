Sub-gradient Methods for L1-Regularization

🎯 Aim

To implement the sub-gradient descent algorithm for optimizing non-differentiable objectives, specifically linear regression with an L1-norm penalty (Lasso).

📐 Mathematical Procedure
Part A: Objective Function

We minimize the sum of squared errors plus an L1 penalty:

\( J(w) = \frac{1}{n} \sum_{i=1}^n (y_i - w x_i)^2 + \lambda |w| \)
Part B: Sub-gradient Calculation

The L1 term \( |w| \) is not differentiable at \( w=0 \). We use the sub-gradient:

\( \partial |w| = \text{sign}(w) \) if \( w \neq 0 \)
\( \partial |w| \in [-1, 1] \) if \( w = 0 \)
💻 Python Implementation
import numpy as np # Dataset 1 X = np.array([1, 2, 3]) y = np.array([5, 6, 7]) w = 0.0 alpha = 0.01 lam = 0.1 for i in range(1000): # Gradient of MSE part grad_mse = -2 * np.mean((y - w*X) * X) # Sub-gradient of L1 part subgrad_l1 = lam * np.sign(w) if w != 0 else 0 w = w - alpha * (grad_mse + subgrad_l1) print(f"Optimal weight w: {w:.6f}") print(f"MSE: {np.mean((y - w*X)**2):.6f}")
