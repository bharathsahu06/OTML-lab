Experiment 8: Sub-gradient Method with L1 Regularization

Home / Experiment 8
Aim

To implement the Sub-gradient Method with L1-regularized loss for the linear regression model \( y = w_1 x_1 + w_2 x_2 \) using the dataset \( (1,2,5), (2,1,6), (3,4,7) \).

Description

The Sub-gradient Method is used for non-differentiable convex optimization problems, such as those with L1 regularization (Lasso). Since the absolute value function is not differentiable at zero, we use the "sub-gradient" — any vector that satisfies the sub-gradient inequality.

Model: \( \hat{y} = w_1 x_1 + w_2 x_2 \)
Cost Function: \( J(\mathbf{w}) = \sum (y_i - \hat{y}_i)^2 + \lambda (|w_1| + |w_2|) \)
Sub-gradient of L1: \( g = \lambda \cdot \text{sign}(\mathbf{w}) \)
Algorithm
  1. Initialize weights \( \mathbf{w} = [0, 0] \).
  2. Set the learning rate \( \alpha = 0.01 \) and regularization strength \( \lambda \).
  3. Repeat for iterations \( k \):
    • Compute predictions \( \hat{y} = \mathbf{X}\mathbf{w} \).
    • Compute gradient of squared error: \( \nabla_{MSE} = 2\mathbf{X}^T(\mathbf{X}\mathbf{w} - \mathbf{y}) \).
    • Compute sub-gradient of L1 term: \( \mathbf{g}_{L1} = \lambda \cdot \text{sign}(\mathbf{w}) \).
    • Update weights: \( \mathbf{w} = \mathbf{w} - \alpha (\nabla_{MSE} + \mathbf{g}_{L1}) \).
  4. Repeat until convergence.
Mathematical Procedure

For Dataset \( (1,2,5), (2,1,6), (3,4,7) \):

\( \mathbf{X} = \begin{pmatrix} 1 & 2 \\ 2 & 1 \\ 3 & 4 \end{pmatrix}, \mathbf{y} = \begin{pmatrix} 5 \\ 6 \\ 7 \end{pmatrix} \)
Start with \( w = [0, 0] \):
\( \nabla_{MSE} = 2\mathbf{X}^T(0 - \mathbf{y}) = \begin{pmatrix} -76 \\ -88 \end{pmatrix} \)
\( w_{new} = w_{old} - 0.01(-76, -88) = (0.76, 0.88) \)
Simple Python Program
import numpy as np

X = np.array([[1, 2], [2, 1], [3, 4]])
y = np.array([5, 6, 7])
w = np.zeros(2)
alpha = 0.01
lam = 0.1

for k in range(100):
    grad_mse = 2 * X.T @ (X @ w - y)
    grad_l1 = lam * np.sign(w)
    # Correct for sign(0) if needed, but in simple GD it handles automatically.
    
    w = w - alpha * (grad_mse + grad_l1)
    if k % 10 == 0:
        mse = np.mean((y - X @ w)**2)
        print(f"Iteration {k}: w={np.round(w, 4)}, MSE={mse:.4f}")

print(f"Final Weights: {w}")
