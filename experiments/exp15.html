Experiment 15: Distributed Gradient Descent (Dataset 2)

📌 Aim

To simulate distributed gradient descent on another dataset: x=[1,2,3,4], y=[3,5,7,9]. This shows how the distributed approach generalizes across datasets.

📝 Description

Distributed gradient descent applies optimization across multiple workers that each process only part of the data. Every worker computes a local gradient, and those local updates are then aggregated into a shared global update.

This method shows how optimization and computing systems interact in large-scale learning. Performance depends not only on the mathematics of the update rule but also on communication, synchronization, and how the data is partitioned across workers.

  • Local computation reduces the work done by any single worker
  • Aggregation combines information from all partitions into one update
  • The method illustrates how scalable learning systems coordinate local and global computation
🔧 Algorithm
Step 1: Initialize global parameters
Step 2: Partition data to workers
Step 3: Each worker processes its data partition
Step 4: Workers send gradients to server
Step 5: Server averages gradients
Step 6: Update global model
💻 Simple Python Program
import numpy as np # Distributed GD - Dataset 2 X = np.array([1, 2, 3, 4]) y = np.array([3, 5, 7, 9]) num_workers = 2 n = len(X) // num_workers # Partition data workers = [ (X[:n], y[:n]), (X[n:], y[n:]) ] w, b = 0.0, 0.0 lr = 0.01 for epoch in range(40): # Collect gradients grads_w, grads_b = [], [] for X_Worker, y_worker in workers: y_pred = w*X_Worker + b dw = -2/len(X_Worker) * np.sum((y_worker - y_pred) * X_Worker) db = -2/len(X_worker) * np.sum(y_worker - y_pred) grads_w.append(dw) grads_b.append(db) # Average and update w -= lr * np.mean(grads_w) b -= lr * np.mean(grads_b) if epoch % 10 == 0: y_pred = w*X + b loss = np.mean((y - y_pred)**2) print(f"Epoch {epoch}: Loss={loss:.4f}, w={w:.4f}, b={b:.4f}") print(f"\nFinal model: y = {w:.4f}x + {b:.4f}")
