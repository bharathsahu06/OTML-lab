Experiment 15: Distributed Gradient Descent (Dataset 2)

Home / Experiment 15
Aim

To simulate a simple Distributed Gradient Descent algorithm for the dataset \( x = [1, 2, 3, 4] \) and \( y = [3, 5, 7, 9] \).

Description

Using a smaller distributed dataset, we observe how two workers collaborate to find the same global model \( y = 2x + 1 \). This experiment demonstrates the basic principles of parameter synchronization and local gradient averaging.

Workers: 2 workers with 2 points each
Worker 1: (1,3), (2,5) | Worker 2: (3,7), (4,9)
Algorithm
  1. Assign points (1,3), (2,5) to W1 and (3,7), (4,9) to W2.
  2. Initialize global weights \( w, b = 0 \).
  3. For every epoch:
    • Each worker computes local gradient: \( \nabla w_{local}, \nabla b_{local} \).
    • Server averages: \( \nabla w_{avg} = \frac{\nabla w_1 + \nabla w_2}{2} \).
    • Global update: \( w = w - \alpha \nabla w_{avg} \).
  4. Continue until convergence.
Mathematical Procedure

Verify for points \( 1, 2, 3, 4 \):

Slope \( w = \frac{10}{5} = 2.0 \)
Intercept \( b = y_{mean} - w \cdot x_{mean} = 6 - 2(2.5) = 1.0 \)
Simple Python Program
import numpy as np

# 2 workers setup
workers = [
    {'x': np.array([1, 2]), 'y': np.array([3, 5])},      # W1
    {'x': np.array([3, 4]), 'y': np.array([7, 9])}       # W2
]

w, b, lr = 0.0, 0.0, 0.01

for epoch in range(10):
    local_grads_w = []
    local_grads_b = []
    
    for shard in workers:
        y_pred = w * shard['x'] + b
        dw = -2 * np.mean((shard['y'] - y_pred) * shard['x'])
        db = -2 * np.mean(shard['y'] - y_pred)
        local_grads_w.append(dw)
        local_grads_b.append(db)
    
    w = w - lr * np.mean(local_grads_w)
    b = b - lr * np.mean(local_grads_b)
    
    print(f"Epoch {epoch}: w={w:.4f}, b={b:.4f}")

print(f"Final Weights: w={w:.4f}, b={b:.4f}")
Expected Output
Epoch 0: w=0.1750, b=0.1000
Epoch 5: w=1.8250, b=0.9167
Epoch 9: w=1.9990, b=0.9995
Final Model Result: y = 2x + 1.
