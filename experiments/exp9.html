Experiment 9: Sub-gradient Method with L1-Regularization (Dataset 2)

📌 Aim

Similar to Experiment 8, but with a different dataset: (2,3,8), (1,4,7), (3,2,9). This demonstrates the method's robustness across different datasets.

📝 Description

L1-regularized regression can be applied to different datasets while still using the same sub-gradient optimization principle. The behavior of the method depends on the data as well as the regularization strength, so the learned coefficients change with the structure of the samples.

The algorithm balances two competing goals: fitting the training samples accurately and keeping the parameter vector simple by shrinking unnecessary weights. This shows that optimization is shaped both by the objective function and by the data being modeled.

  • The data term measures how well the model explains the observations
  • The L1 penalty can reduce some coefficients to zero
  • The final parameter values change when the dataset changes
🔧 Algorithm
Step 1: Initialize weights to 0
Step 2: For each iteration, compute predictions
Step 3: Calculate loss including L1 penalty
Step 4: Compute sub-gradient
Step 5: Update weights using learning rate
📐 Formulas
Model:
ŷ = w₁×x₁ + w₂×x₂
L1-Regularized Loss:
L(w) = Σ(yᵢ - ŷᵢ)² + λ||w||₁
💻 Simple Python Program
import numpy as np import matplotlib.pyplot as plt # Dataset 2: (x1, x2, y) X = np.array([[2, 3], [1, 4], [3, 2]]) y = np.array([8, 7, 9]) # Parameters w = np.array([0.0, 0.0]) lr = 0.01 lambda_reg = 0.05 epochs = 150 losses = [] for epoch in range(epochs): # Forward pass y_pred = X @ w error = y_pred - y # Gradient + L1 penalty grad = 2 * X.T @ error l1_penalty = lambda_reg * np.sign(w) # Update weights w = w - lr * (grad + l1_penalty) # Loss loss = np.mean(error**2) + lambda_reg * np.sum(np.abs(w)) losses.append(loss) if epoch % 30 == 0: print(f"Epoch {epoch}: Loss = {loss:.4f}, w = [{w[0]:.4f}, {w[1]:.4f}]") print(f"\nFinal weights: w₁ = {w[0]:.4f}, w₂ = {w[1]:.4f}")
