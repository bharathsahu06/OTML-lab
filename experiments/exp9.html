Experiment 9: Sub-gradient Method with L1 (Dataset 2)

Home / Experiment 9
Aim

To implement the Sub-gradient Method with L1-regularized loss for the linear regression model \( y = w_1 x_1 + w_2 x_2 \) using the dataset \( (2,3,8), (1,4,7), (3,2,9) \).

Description

Using the second dataset, we continue exploring how the sub-gradient method handles non-differentiable regularization terms and pushes small weights toward zero (sparsity).

Model: \( \hat{y} = w_1 x_1 + w_2 x_2 \)
Cost Function: \( J(\mathbf{w}) = \sum (y_i - \hat{y}_i)^2 + \lambda (|w_1| + |w_2|) \)
Algorithm
  1. Initialize weights \( \mathbf{w} = [0, 0] \).
  2. Set the learning rate \( \alpha = 0.01 \) and regularization strength \( \lambda \).
  3. Repeat for iterations \( k \):
    • Compute predictions: \( \hat{y} = \mathbf{X}\mathbf{w} \).
    • Compute the gradient of squared error: \( \nabla_{MSE} = 2\mathbf{X}^T(\mathbf{X}\mathbf{w} - \mathbf{y}) \).
    • Compute the sub-gradient of L1 term: \( \mathbf{g}_{L1} = \lambda \cdot \text{sign}(\mathbf{w}) \).
    • Update: \( \mathbf{w} = \mathbf{w} - \alpha (\nabla_{MSE} + \mathbf{g}_{L1}) \).
  4. Repeat until convergence.
Mathematical Procedure

For Dataset \( (2,3,8), (1,4,7), (3,2,9) \):

\( \mathbf{X} = \begin{pmatrix} 2 & 3 \\ 1 & 4 \\ 3 & 2 \end{pmatrix}, \mathbf{y} = \begin{pmatrix} 8 \\ 7 \\ 9 \end{pmatrix} \)
Start with \( w = [0, 0] \):
\( \nabla_{MSE} = 2\mathbf{X}^T(0 - \mathbf{y}) = \begin{pmatrix} -100 \\ -140 \end{pmatrix} \)
\( w_{new} = w_{old} - 0.01(-100, -140) = (1.0, 1.4) \)
Simple Python Program
import numpy as np

X = np.array([[2, 3], [1, 4], [3, 2]])
y = np.array([8, 7, 9])
w = np.zeros(2)
alpha = 0.01
lam = 0.1

for k in range(100):
    grad_mse = 2 * X.T @ (X @ w - y)
    grad_l1 = lam * np.sign(w)
    
    w = w - alpha * (grad_mse + grad_l1)
    if k % 10 == 0:
        mse = np.mean((y - X @ w)**2)
        print(f"Iteration {k}: w={np.round(w, 4)}, MSE={mse:.4f}")

print(f"Final Weights: {w}")
