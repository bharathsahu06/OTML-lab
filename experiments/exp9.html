L1-Regularization with Specific Datasets

🎯 Aim

To analyze the effect of L1-regularization (Lasso) on a different dataset and observe how the optimal weight \( w \) changes compared to unregularized OLS.

💻 Python Implementation
import numpy as np # Dataset 2: (x, y) = (2, 8), (1, 7), (3, 9) X = np.array([2, 1, 3]) y = np.array([8, 7, 9]) w = 0.0 lam = 0.5 alpha = 0.01 for _ in range(1000): grad_mse = -2 * np.mean((y - w*X) * X) subgrad = lam * np.sign(w) if w != 0 else 0 w = w - alpha * (grad_mse + subgrad) print(f"L1-Regularized Weight: {w:.6f}")
