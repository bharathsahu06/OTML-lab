Experiment 18: Conjugate Gradient Method

1) Aim
Implement the Conjugate Gradient (CG) method to solve the linear system: $$Ax=b$$ with interpreted input: $$A=\begin{bmatrix}4 & 1\\1 & 3\end{bmatrix}, \qquad b=\begin{bmatrix}1\\2\end{bmatrix}$$
2) Description

The Conjugate Gradient method is an iterative algorithm for solving symmetric positive definite (SPD) linear systems. It is efficient because it avoids explicit matrix inversion and converges in at most $n$ steps (in exact arithmetic) for an $n\times n$ SPD matrix. In this experiment, we solve a 2-variable system using CG and verify that the computed solution satisfies $Ax \approx b$.

The method followed is: start with an initial guess, compute initial residual and search direction, then update solution using step size computed from residual and matrix-direction products. After each update, a new conjugate direction is built using the residual ratio, and iterations continue until residual norm is below tolerance. This gives an accurate numerical solution with clear step-by-step updates.

3) Important Formulas

System: $$Ax=b$$

Initialize: $$x_0=0, \quad r_0=b-Ax_0, \quad p_0=r_0$$

Step size: $$\alpha_k=\frac{r_k^Tr_k}{p_k^TAp_k}$$

Solution update: $$x_{k+1}=x_k+\alpha_k p_k$$

Residual update: $$r_{k+1}=r_k-\alpha_kAp_k$$

Direction coefficient: $$\beta_k=\frac{r_{k+1}^Tr_{k+1}}{r_k^Tr_k}$$

Direction update: $$p_{k+1}=r_{k+1}+\beta_k p_k$$

Stopping condition: $$\|r_k\|_2 < \varepsilon$$

4) Important Functions Used
  • conjugate_gradient(A, b, x0, tol, max_iter): implements CG iterations.
  • np.dot(...): inner products for $\alpha$ and $\beta$.
  • np.linalg.norm(...): residual norm for stopping.
5) Algorithm (Step by Step)
  1. Choose initial guess $x_0$.
  2. Compute residual $r_0=b-Ax_0$.
  3. Set search direction $p_0=r_0$.
  4. Compute $\alpha_k$.
  5. Update $x_{k+1}$ and $r_{k+1}$.
  6. If residual norm is small, stop.
  7. Compute $\beta_k$.
  8. Update direction $p_{k+1}$.
  9. Repeat until convergence.
6) Mathematical Procedure (First Iteration)

Take $x_0=[0,0]^T$. Then: $r_0=b-Ax_0=b=\begin{bmatrix}1\\2\end{bmatrix}, \quad p_0=r_0$

Compute: $$Ap_0=\begin{bmatrix}4 & 1\\1 & 3\end{bmatrix} \begin{bmatrix}1\\2\end{bmatrix} = \begin{bmatrix}6\\7\end{bmatrix}$$
Step size: $$\alpha_0=\frac{r_0^Tr_0}{p_0^TAp_0} = \frac{1^2+2^2}{[1,2]\cdot[6,7]} = \frac{5}{20}=0.25$$
Update: $$x_1=x_0+0.25p_0 = \begin{bmatrix}0.25\\0.5\end{bmatrix}$$ $$r_1=r_0-0.25Ap_0 = \begin{bmatrix}1\\2\end{bmatrix}-0.25 \begin{bmatrix}6\\7\end{bmatrix} = \begin{bmatrix}-0.5\\0.25\end{bmatrix}$$

Continue similarly until residual is sufficiently small.

7) Given Input
  • $A=\begin{bmatrix}4&1\\1&3\end{bmatrix}$
  • $b=\begin{bmatrix}1\\2\end{bmatrix}$
  • Initial guess: $x_0=[0,0]^T$
  • Tolerance: $10^{-10}$
8) Python Implementation & Output
import numpy as np # Interpreted input from the aim text A = np.array([[4.0, 1.0], [1.0, 3.0]]) b = np.array([1.0, 2.0]) def conjugate_gradient(A, b, x0=None, tol=1e-10, max_iter=100): n = len(b) x = np.zeros(n) if x0 is None else np.array(x0, dtype=float) r = b - A @ x p = r.copy() rs_old = np.dot(r, r) history = [] for k in range(1, max_iter + 1): Ap = A @ p alpha = rs_old / np.dot(p, Ap) x = x + alpha * p r = r - alpha * Ap rs_new = np.dot(r, r) res_norm = np.sqrt(rs_new) history.append((k, x.copy(), res_norm)) if res_norm < tol: break beta = rs_new / rs_old p = r + beta * p rs_old = rs_new return x, history x0 = np.array([0.0, 0.0]) tol = 1e-10 max_iter = 100 x_star, steps = conjugate_gradient(A, b, x0=x0, tol=tol, max_iter=max_iter) b_est = A @ x_star residual = b - b_est print("INPUT") print("A =") print(A) print("b =", b) print("Initial guess x0 =", x0) print(f"Tolerance = {tol}") print("\nITERATION SUMMARY") for k, xk, rn in steps: print(f"Iter {k}: x = {np.round(xk, 10)}, residual_norm = {rn:.3e}") print(\nOUTPUT") print("Solution x* =", np.round(x_star, 10)) print("A @ x* =", np.round(b_est, 10)) print("Residual (b - A@x*) =", np.round(residual, 12)) print("Program executed successfully.")
