Experiment 18: Conjugate Gradient Method

📌 Aim

To solve a system of linear equations Ax = b using the Conjugate Gradient method, a fundamental algorithm for solving large sparse linear systems.

Given: A = [[4, 1], [1, 3]], b = [[1], [2]]

📝 Description

The Conjugate Gradient (CG) method solves symmetric positive-definite systems. It:

  • Uses conjugate directions (orthogonal in A-norm)
  • Converges in at most n iterations for n×n system
  • Ideal for large sparse matrices
  • More efficient than Gaussian elimination
🔧 Algorithm
Step 1: Start with initial guess x₀
Step 2: Compute residual r₀ = b - Ax₀
Step 3: Set direction p₀ = r₀
Step 4: For each iteration k:
Step 5: Compute step size α
Step 6: Update x and residual
Step 7: Compute new direction using β
📐 Formulas
Linear System:
Ax = b
CG Update (iteration k):
α_k = (r_k^T r_k) / (p_k^T A p_k)
x_{k+1} = x_k + α_k × p_k
r_{k+1} = r_k - α_k × A × p_k
β_k = (r_{k+1}^T r_{k+1}) / (r_k^T r_k)
p_{k+1} = r_{k+1} + β_k × p_k
💻 Simple Python Program
import numpy as np # Conjugate Gradient Method A = np.array([[4, 1], [1, 3]]) b = np.array([1, 2]) def conjugate_gradient(A, b, x0=None, tol=1e-6, max_iter=100): n = len(b) if x0 is None: x = np.zeros(n) else: x = x0.copy() r = b - A @ x p = r.copy() for k in range(max_iter): # Compute alpha Ap = A @ p alpha = np.dot(r, r) / np.dot(p, Ap) # Update x x = x + alpha * p # Update residual r_new = r - alpha * Ap # Check convergence if np.linalg.norm(r_new) < tol: print(f"Converged at iteration {k}") break # Compute beta beta = np.dot(r_new, r_new) / np.dot(r, r) # Update direction p = r_new + beta * p r = r_new if k % 1 == 0: print(f"Iter {k}: x = {x}, residual = {np.linalg.norm(r):.6f}") return x # Solve solution = conjugate_gradient(A, b) print(f"\nSolution x = {solution}") print(f"Verification: Ax = {A @ solution}") print(f"Target b = {b}")
