Experiment 18: Linear Conjugate Gradient Method

Home / Experiment 18
Aim

To implement the Conjugate Gradient (CG) method to solve symmetric, positive-definite linear systems of the form \( \mathbf{A}\mathbf{x} = \mathbf{b} \).

Description

The Conjugate Gradient method is an iterative process that minimizes the quadratic form of the linear system. It is faster than Gradient Descent as it searches in "conjugate" directions, preventing it from searching in the same direction twice. It converges in at most \( n \) iterations for an \( n \times n \) system.

Problem: Solve \( \begin{pmatrix} 4 & 1 \\ 1 & 3 \end{pmatrix} \begin{pmatrix} x_1 \\ x_2 \end{pmatrix} = \begin{pmatrix} 1 \\ 2 \end{pmatrix} \)
Algorithm
  1. Initialize residual \( \mathbf{r}_0 = \mathbf{b} - \mathbf{A}\mathbf{x}_0 \).
  2. Set initial search direction \( \mathbf{p}_0 = \mathbf{r}_0 \).
  3. Repeat for iterations \( k \):
    • Compute step size \( \alpha_k = \frac{\mathbf{r}_k^T \mathbf{r}_k}{\mathbf{p}_k^T \mathbf{A} \mathbf{p}_k} \).
    • Update solution: \( \mathbf{x}_{k+1} = \mathbf{x}_k + \alpha_k \mathbf{p}_k \).
    • Update residual: \( \mathbf{r}_{k+1} = \mathbf{r}_k - \alpha_k \mathbf{A} \mathbf{p}_k \).
    • Compute improvement factor \( \beta_k = \frac{\mathbf{r}_{k+1}^T \mathbf{r}_{k+1}}{\mathbf{r}_k^T \mathbf{r}_k} \).
    • Update direction: \( \mathbf{p}_{k+1} = \mathbf{r}_{k+1} + \beta_k \mathbf{p}_k \).
Mathematical Procedure

Iteration 1 (\( x_0=[0,0] \)):

  • \( r_0 = [1, 2]^T, p_0 = [1, 2]^T \)
  • \( Ap_0 = [6, 7]^T \), \( p_0^T Ap_0 = 20 \)
  • \( \alpha_0 = 5/20 = 0.25 \), \( x_1 = [0.25, 0.5]^T \)
  • \( r_1 = [-0.5, 0.25]^T, \beta_0 = 0.3125/5 = 0.0625 \)
Simple Python Program
import numpy as np

A = np.array([[4, 1], [1, 3]], dtype=float)
b = np.array([1, 2], dtype=float)
x = np.zeros(2)
r = b - A @ x
p = r.copy()
rsold = r @ r

for i in range(2):
    Ap = A @ p
    alpha = rsold / (p @ Ap)
    x = x + alpha * p
    r = r - alpha * Ap
    rsnew = r @ r
    if np.sqrt(rsnew) < 1e-10: break
    p = r + (rsnew / rsold) * p
    rsold = rsnew

print(f"CG Solution: {x}")
print(f"Residual Check: {A @ x - b}")
