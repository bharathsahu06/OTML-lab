Experiment 7: Linear Programming with PuLP

📌 Aim

To implement linear programming both with PuLP library and without it (manual implementation) to understand how optimization solvers work internally.

📝 Description

Linear programming can be understood from two complementary perspectives: modeling the problem with a solver library such as PuLP and understanding the logic behind a manual implementation. PuLP offers a high-level way to define decision variables, objectives, and constraints without dealing directly with solver internals.

At the same time, a manual approach helps reveal what the solver is conceptually doing when it evaluates feasibility and searches for an optimum. Comparing both views builds stronger intuition about how optimization software connects theory with computation.

  • PuLP is useful for formulating practical optimization models quickly
  • A manual method helps explain the internal logic of the solution process
  • Together they connect mathematical modeling, implementation, and interpretation
🔧 Algorithm
With PuLP: Define problem → Add variables → Add constraints → Solve
Without PuLP: Use direct calculation or check corner points
💻 Simple Python Program - With PuLP

Install: pip install pulp

from pulp import * # Create problem prob = LpProblem("Linear_Programming", LpMinimize) # Variables x = LpVariable("x", lowBound=0) y = LpVariable("y", lowBound=0) # Objective: minimize 3x + 2y prob += 3*x + 2*y, "objective" # Constraints: x + y >= 4 prob += x + y >= 4, "constraint1" # Solve prob.solve() # Results print(f"Status: {LpStatus[prob.status]}") print(f"x = {value(x):.6f}") print(f"y = {value(y):.6f}") print(f"Minimum = {value(prob.objective):.6f}")
💻 Manual Implementation (Without PuLP)
# Manual Linear Programming - Check corner points # Minimize: 3x + 2y # Subject to: x + y >= 4, x >= 0, y >= 0 # Corner points of feasible region corners = [ (0, 4), # (0, 4) (4, 0) # (4, 0) ] # Objective function def objective(x, y): return 3*x + 2*y # Evaluate at each corner best_value = float('inf') best_point = None for point in corners: value = objective(point[0], point[1]) print(f"Point {point}: value = {value}") if value < best_value: best_value = value best_point = point print(f"\nOptimal point: {best_point}") print(f"Optimal value: {best_value}")
