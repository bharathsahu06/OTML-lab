Linear Programming with PuLP

🎯 Aim

To implement a linear programming solver using the PuLP library, which provides a high-level Pythonic interface to industry-standard solvers like CBC and GLPK.

💻 Python Implementation
from pulp import * # Initialize the problem prob = LpProblem("Minimization_Problem", LpMinimize) # Define decision variables x = LpVariable("x", lowBound=0) y = LpVariable("y", lowBound=0) # Objective function prob += 3*x + 2*y # Constraints prob += x + y >= 4 # Solve status = prob.solve(PULP_CBC_CMD(msg=0)) print(f"Status: {LpStatus[status]}") print(f"x = {value(x):.6f}") print(f"y = {value(y):.6f}") print(f"Objective = {value(prob.objective):.6f}")
📊 Expected Output
Terminal Console
Status: Optimal
x = 0.000000
y = 4.000000
Objective = 8.000000
