Experiment 07: Sub-gradient Method for LASSO

Home / Experiment 07
Aim

To implement the Sub-gradient method for L1-regularized linear regression (LASSO) to minimize a non-smooth objective function.

Description

The LASSO adds an L1-penalty to the standard Mean Squared Error (MSE) loss. Since the absolute value function is not differentiable at zero, we use sub-gradients rather than standard gradients to update the parameters.

Objective: \( J(m,c) = \frac{1}{2n}\sum_{i=1}^n (y_i-(m x_i+c))^2 + \lambda |m| \)

Dataset: \( x = [1, 2, 3], y = [1, 2, 3] \).
Regularization: \( \lambda = 0.1 \).

Algorithm
  1. Initialize slope \( m \) and intercept \( c \).
  2. For each epoch:
    • Compute the gradient of the MSE part.
    • Compute the sub-gradient of the L1 term \( \lambda |m| \):
      • If \( m > 0 \), sub-gradient is \( \lambda \).
      • If \( m < 0 \), sub-gradient is \( -\lambda \).
      • If \( m = 0 \), sub-gradient can be any value in \( [-\lambda, \lambda] \).
    • Total sub-gradient \( \tilde{g} = \nabla MSE + \text{sub-gradient}(L1) \).
    • Update parameters: \( \theta_{k+1} = \theta_k - \alpha_k \tilde{g} \).
Mathematical Procedure

Parameter Updates:

  • \( m_{k+1} = m_k - \alpha_k (\frac{1}{n}\sum ((mx_i+c)-y_i)x_i + \lambda \cdot \text{sgn}(m_k)) \)
  • \( c_{k+1} = c_k - \alpha_k (\frac{1}{n}\sum ((mx_i+c)-y_i)) \)
Simple Python Program
import numpy as np

x = np.array([1, 2, 3], dtype=float)
y = np.array([1, 2, 3], dtype=float)
lam = 0.1
m, c = 0.0, 0.0
epochs = 1000

for epoch in range(1, epochs + 1):
    n = len(x)
    ri = (m * x + c) - y
    dm = np.sum(ri * x) / n
    dc = np.sum(ri) / n

    # Sub-gradient for m
    if m > 0: dm += lam
    elif m < 0: dm -= lam
    
    alpha = 1.0 / (1.0 + epoch)
    m -= alpha * dm
    c -= alpha * dc

print(f"Final m: {m:.4f}")
print(f"Final c: {c:.4f}")
