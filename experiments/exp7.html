Experiment 7: Linear Programming Using PuLP and Without PuLP

1) Aim

a) Implementation of linear programming using PuLP.

b) Implementation of linear programming without PuLP.

For demonstration, solve: $$\min Z=3x+2y$$ subject to: $$x+y\ge 4,\quad x\ge 0,\quad y\ge 0$$

2) Description

LP Solution Methods encompass different computational approaches to finding optimal solutions. The Simplex Method (used by PuLP) navigates the polytope vertices efficiently, taking larger steps toward the optimum. The Vertex/Corner Point Method directly examines intersection points, making it suitable for low-dimensional problems. Understanding multiple approaches reveals that LP has rich algorithmic variety, each with different strengths in speed, numerical stability, and scalability.

The key insight is that all methods exploit the geometric structure of LP: the optimum always resides at a vertex. Simplex implicitly checks this by visiting promising vertices in sequence. Direct vertex enumeration becomes impractical in high dimensions but perfectly illustrates the principle in 2D. Modern solvers (interior point, barrier methods) take different paths through the feasible region but all converge to the same vertex optimum.

Comparing methods on the same problem demonstrates that despite different computational routes, they find identical optimal solutions. This consistency validates the underlying theory and builds confidence in using high-level solvers like PuLP.

3) Important Formulas

Objective: $$\min Z=3x+2y$$

Constraints: $$x+y\ge4, \quad x\ge0, \quad y\ge0$$

LP principle for 2-variable case: Optimum occurs at a feasible corner (extreme) point.

4) Important Functions Used
  • LpProblem, LpVariable, LpMinimize, value from PuLP.
  • problem.solve() to run LP solver in PuLP.
  • numpy.linalg.solve() for line intersections (without PuLP).
  • Feasibility checks using constraint conditions.
5) Algorithm (Step by Step)

Part A: Using PuLP

  1. Define LP minimization problem.
  2. Create variables $x, y \ge 0$.
  3. Set objective $3x+2y$.
  4. Add constraint $x+y\ge4$.
  5. Solve and read $x^*, y^*, Z^*$.

Part B: Without PuLP (Vertex Method)

  1. Write boundary lines: $x=0$, $y=0$, $x+y=4$.
  2. Compute pairwise intersections.
  3. Keep only feasible points satisfying all constraints.
  4. Evaluate objective $Z=3x+2y$ at feasible points.
  5. Select point with minimum $Z$.
6) Mathematical Procedure

From constraint boundary $x+y=4$, write: $$y=4-x,\quad 0\le x\le4$$

Substitute into objective: $$Z=3x+2(4-x)=x+8$$

Since $Z=x+8$ increases with $x$, minimum occurs at smallest feasible $x$, i.e. $x=0$.

Then: $$y=4$$

Minimum value: $$Z_{\min}=3(0)+2(4)=8$$

7) Given Input
  • Objective: $Z=3x+2y$
  • Constraints: $x+y\ge4$, $x\ge0$, $y\ge0$
8A) Experiment 7A - Using PuLP
import pulp # Create model model = pulp.LpProblem("LP_Problem", pulp.LpMinimize) # Variables x = pulp.LpVariable("x", lowBound=0) y = pulp.LpVariable("y", lowBound=0) # Objective function model += 3*x + 2*y # Constraint model += x + y >= 4 # Solve model.solve(pulp.PULP_CBC_CMD(msg=0)) # Output print("model:", model) print("Status:", pulp.LpStatus[model.status]) print("x =", pulp.value(x)) print("y =", pulp.value(y)) print("Minimum Z =", pulp.value(model.objective))
Status: Optimal x = 0.0 y = 4.0 Minimum Z = 8.0
8B) Experiment 7B - Without PuLP (Vertex Method)
import numpy as np # Objective function def Z(x, y): return 3*x + 2*y # Step 1: Define boundary equations # x = 0, y = 0, x + y = 4 # Step 2: Find intersection points manually points = [] # Intersection of x=0 and y=0 points.append((0, 0)) # Intersection of x=0 and x+y=4 → y=4 points.append((0, 4)) # Intersection of y=0 and x+y=4 → x=4 points.append((4, 0)) # Step 3: Check feasibility (x+y >= 4) feasible_points = [] for x, y in points: if x + y >= 4 and x >= 0 and y >= 0: feasible_points.append((x, y)) # Step 4: Find minimum best_point = min(feasible_points, key=lambda p: Z(p[0], p[1])) min_value = Z(best_point[0], best_point[1]) # Output print("Objective function: Z = 3x + 2y") print("Constraints: x + y >= 4, x >= 0, y >= 0") print("Feasible points:", feasible_points) print("Optimal x:", best_point[0]) print("Optimal y:", best_point[1]) print("Minimum Z:", min_value)
