Experiment 7: Sub-gradient Method with L1-Regularized Loss

1) Aim
Implement the Sub-gradient Method with L1-regularized loss for linear regression. The model is: $$\hat{y}=w_1 x_1+w_2 x_2$$ using the dataset:
  • Sample 1: $(x_1, x_2, y)=(1, 2, 5)$
  • Sample 2: $(x_1, x_2, y)=(2, 1, 6)$
  • Sample 3: $(x_1, x_2, y)=(3, 4, 7)$
2) Description

Sub-gradient Method is an optimization algorithm for non-smooth convex functions. Unlike gradient descent which requires the function to be differentiable everywhere, sub-gradient methods work on a broader class of functions, including those with sharp corners or absolute-value terms (non-smooth functions).

L1 regularization adds a penalty term $\lambda \|w\|_1$ to the loss function, promoting sparsity in learned parameters. This is particularly useful when we want many weights to become exactly zero, leading to simpler, more interpretable models. The L1 norm is non-smooth at the origin, making sub-gradient methods essential for optimization.

The sub-gradient method uses a sub-differential set at non-smooth points instead of a unique gradient. At smooth points, the sub-differential contains only the gradient. This generalization allows optimization algorithms to navigate non-smooth landscapes and find optimal sparse solutions efficiently.

3) Important Formulas

Model:

$$\hat{y}_i=w_1 x_{i1}+w_2 x_{i2}$$

L1-Regularized Loss:

$$J(w_1,w_2)=\frac{1}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)^2+\lambda(|w_1|+|w_2|)$$

Sub-gradient of L1 norm:

$$\partial|w|=\begin{cases}1 & \text{if } w>0 \\ -1 & \text{if } w<0 \\ [-1,1] & \text{if } w=0\end{cases}$$

Sub-gradient update:

$$w_j \leftarrow w_j-\alpha g_j, \quad \text{where } g_j \in \partial J(w)$$
4) Algorithm (Step by Step)
  1. Initialize weights $w_1=0, w_2=0$ and set hyperparameters ($\alpha$, $\lambda$, tolerance, max iterations).
  2. Compute predictions $\hat{y}_i=w_1 x_{i1}+w_2 x_{i2}$.
  3. Compute prediction errors $e_i=\hat{y}_i-y_i$.
  4. Compute MSE loss component: $\frac{1}{n}\sum_i e_i^2$.
  5. Compute gradients of MSE w.r.t. weights.
  6. Compute sub-gradient of L1 norm for each weight.
  7. Update weights using combined gradient and sub-gradient.
  8. Repeat until convergence.
5) Mathematical Procedure (Complete Step-by-Step)

Given data: $(x_1, x_2, y)$: $(1,2,5), (2,1,6), (3,4,7)$

1. Compute predictions: $$\hat{y}_i=w_1 x_{i1}+w_2 x_{i2}$$
2. Compute errors: $$e_i=\hat{y}_i-y_i$$
3. Compute loss: $$J(w_1,w_2)=\frac{1}{n}\sum_{i=1}^{n}e_i^2+\lambda(|w_1|+|w_2|)$$
4. Compute MSE gradients: $$g_{w_1}^{MSE}=\frac{2}{n}\sum_{i=1}^{n}e_i \cdot x_{i1}$$ $$g_{w_2}^{MSE}=\frac{2}{n}\sum_{i=1}^{n}e_i \cdot x_{i2}$$
5. Compute L1 sub-gradients: $$g_{w_1}^{L1}=\lambda \cdot \text{sign}(w_1), \quad g_{w_2}^{L1}=\lambda \cdot \text{sign}(w_2)$$
6. Update weights: $$w_1 \leftarrow w_1-\alpha(g_{w_1}^{MSE}+g_{w_1}^{L1})$$ $$w_2 \leftarrow w_2-\alpha(g_{w_2}^{MSE}+g_{w_2}^{L1})$$
6) Given Input
  • Dataset: $(1,2,5), (2,1,6), (3,4,7)$
  • Initial $w_1=0, w_2=0$
  • Learning rate $\alpha=0.01$
  • L1 penalty $\lambda=0.1$
  • Tolerance $=10^{-6}$
  • Max iterations $=200$
7) Python Implementation & Output
import numpy as np # Data x1 = np.array([1, 2, 3], dtype=float) x2 = np.array([2, 1, 4], dtype=float) y = np.array([5, 6, 7], dtype=float) # Initialize weights w1 = 0.0 w2 = 0.0 # Hyperparameters lr = 0.01 lambda_l1 = 0.1 tol = 1e-6 max_iter = 200 n = len(y) print("Iter\t w1\t\t w2\t\t Loss") for iteration in range(1, max_iter + 1): # Predictions y_pred = w1 * x1 + w2 * x2 errors = y_pred - y # MSE loss mse_loss = np.mean(errors**2) l1_loss = lambda_l1 * (np.abs(w1) + np.abs(w2)) total_loss = mse_loss + l1_loss # MSE gradients grad_w1_mse = 2 * np.mean(errors * x1) grad_w2_mse = 2 * np.mean(errors * x2) # L1 sub-gradients grad_w1_l1 = lambda_l1 * np.sign(w1) if w1 != 0 else 0 grad_w2_l1 = lambda_l1 * np.sign(w2) if w2 != 0 else 0 # Total sub-gradients grad_w1 = grad_w1_mse + grad_w1_l1 grad_w2 = grad_w2_mse + grad_w2_l1 # Update weights w1_new = w1 - lr * grad_w1 w2_new = w2 - lr * grad_w2 if iteration <= 10 or iteration % 20 == 0: print(f"{iteration}\t {w1_new:.6f}\t {w2_new:.6f}\t {total_loss:.6f}") # Check convergence if np.abs(w1_new - w1) < tol and np.abs(w2_new - w2) < tol: w1, w2 = w1_new, w2_new break w1, w2 = w1_new, w2_new print(f"\nOptimal w1 = {round(w1, 6)}") print(f"Optimal w2 = {round(w2, 6)}") print(f"Final Loss = {round(total_loss, 6)}")
