Experiment 19: Quasi-Newton Method (BFGS)

📌 Aim

To minimize a multivariate function f(x₁,x₂) = x₁² + 2x₂² using the Quasi-Newton method (BFGS algorithm). BFGS approximates the Hessian without computing second derivatives explicitly.

📝 Description

Quasi-Newton methods (BFGS) are alternatives to Newton's method:

  • Approximate Hessian (second derivatives) cheaply
  • Converge faster than gradient descent
  • Don't compute full Hessian like Newton's method
  • Widely used in production: implemented in scipy.optimize
🔧 Algorithm
Step 1: Initialize position and Hessian approximation H
Step 2: Compute gradient ∇f
Step 3: Compute Newton-like direction: p = -H ∇f
Step 4: Line search to find step size
Step 5: Update position
Step 6: Update Hessian approximation (BFGS formula)
Step 7: Repeat until convergence
📐 Formulas
Objective Function:
f(x₁, x₂) = x₁² + 2x₂²
Gradient:
∇f = [2x₁, 4x₂]^T
BFGS Update (Hessian approximation):
H_{k+1} = H_k - (H_k s s^T H_k)/(s^T H_k s) + (y y^T)/(y^T s)
where s = x_{k+1} - x_k, y = ∇f(x_{k+1}) - ∇f(x_k)
💻 Simple Python Program

Easy approach using scipy:

import numpy as np from scipy.optimize import minimize # Objective function def objective(x): return x[0]**2 + 2*x[1]**2 # Gradient def gradient(x): return np.array([2*x[0], 4*x[1]]) # Initial point x0 = np.array([5.0, 5.0]) # Optimize using BFGS result = minimize( objective, x0, method='BFGS', jac=gradient, options={'disp': True} ) print(f"\nOptimal point: {result.x}") print(f"Optimal value: {result.fun:.6f}") print(f"Iterations: {result.nit}") print(f"Success: {result.success}")
💻 Manual BFGS Implementation (Educational)
import numpy as np def bfgs_method(x0, max_iter=50): x = x0.copy() n = len(x0) H = np.eye(n) # Identity matrix def f(x): return x[0]**2 + 2*x[1]**2 def grad(x): return np.array([2*x[0], 4*x[1]]) for i in range(max_iter): grad_x = grad(x) # Newton-like direction p = -H @ grad_x # Line search (simple) alpha = 0.1 # Update x_new = x + alpha * p grad_new = grad(x_new) # BFGS update s = x_new - x y = grad_new - grad_x if np.linalg.norm(y) > 1e-8: H = H - (H@s@s.T@H)/(s.T@H@s) + (y@y.T)/(y.T@s) x = x_new if i % 10 == 0: print(f"Iter {i}: x={x}, f(x)={f(x):.6f}") return x x0 = np.array([5.0, 5.0]) result = bfgs_method(x0, 50) print(f"\nFinal x: {result}")
