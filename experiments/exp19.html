Experiment 19: Quasi-Newton Method (BFGS)

📌 Aim

To minimize a multivariate function f(x₁,x₂) = x₁² + 2x₂² using the Quasi-Newton method (BFGS algorithm). BFGS approximates the Hessian without computing second derivatives explicitly.

📝 Description

BFGS is a Quasi-Newton method that builds an approximation to the inverse Hessian using gradient information from previous steps. This gives it much of the speed advantage of Newton's method without requiring explicit second derivatives.

As the iterations proceed, the approximation improves and the search direction becomes more informed by the local curvature of the objective function. That is why BFGS is a popular general-purpose optimizer for smooth unconstrained problems and is widely used in practical optimization libraries.

  • Gradient differences reveal how the curvature is changing
  • The inverse-Hessian approximation improves the search direction over time
  • BFGS is often faster and more stable than basic gradient descent for smooth problems
🔧 Algorithm
Step 1: Initialize position and Hessian approximation H
Step 2: Compute gradient ∇f
Step 3: Compute Newton-like direction: p = -H ∇f
Step 4: Line search to find step size
Step 5: Update position
Step 6: Update Hessian approximation (BFGS formula)
Step 7: Repeat until convergence
📐 Formulas
Objective Function:
f(x₁, x₂) = x₁² + 2x₂²
Gradient:
∇f = [2x₁, 4x₂]^T
BFGS Update (Hessian approximation):
H_{k+1} = H_k - (H_k s s^T H_k)/(s^T H_k s) + (y y^T)/(y^T s)
where s = x_{k+1} - x_k, y = ∇f(x_{k+1}) - ∇f(x_k)
📊 Mathematical Procedure
Objective: minimize f(x₁,x₂)=x₁²+2x₂².
Stationary condition: ∇f=[2x₁,4x₂]ᵀ=0 gives x₁=0, x₂=0.
Optimality check: Hessian [[2,0],[0,4]] is positive definite, so (0,0) is the unique global minimum.
BFGS process: start from x₀=[5,5], use gradient-based quasi-Newton updates to approach (0,0).
Expected result: final point near [0,0] with very small function value and gradient norm.
💻 Simple Python Program

Easy approach using scipy:

import numpy as np from scipy.optimize import minimize # Objective function def objective(x): return x[0]**2 + 2*x[1]**2 # Gradient def gradient(x): return np.array([2*x[0], 4*x[1]]) # Initial point x0 = np.array([5.0, 5.0]) history = [] def callback(xk): history.append(xk.copy()) # Optimize using BFGS result = minimize( objective, x0, method='BFGS', jac=gradient, callback=callback, options={'disp': True} ) print(f"\nOptimal point: {result.x}") print(f"Optimal value: {result.fun:.6f}") print(f"Iterations: {result.nit}") print(f"Gradient norm: {np.linalg.norm(gradient(result.x)):.2e}") print(f"Success: {result.success}")
