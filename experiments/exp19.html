Experiment 19: Quasi-Newton Method (BFGS Update)

Home / Experiment 19
Aim

To implement the Broyden–Fletcher–Goldfarb–Shanno (BFGS) quasi-Newton algorithm to minimize a multi-variable convex function without using the actual Hessian matrix.

Description

BFGS is a quasi-Newton method that approximates the inverse Hessian matrix using gradient information over several steps. Unlike Newton's method which requires second-order derivatives, BFGS only needs the gradient and the previous search history to perform high-speed convergence.

Objective: \( f(x_1, x_2) = x_1^2 + 2x_2^2 \)
Gradient: \( \nabla f = [2x_1, 4x_2]^T \)
Algorithm
  1. Initialize initial solution \( \mathbf{x}_0 \) and an identity matrix as inverse Hessian \( \mathbf{H}_0 = \mathbf{I} \).
  2. For iterations \( k \):
    • Compute search direction \( \mathbf{p}_k = -\mathbf{H}_k \nabla f(\mathbf{x}_k) \).
    • Compute step size \( \alpha_k \) using line search.
    • Update solution: \( \mathbf{x}_{k+1} = \mathbf{x}_k + \alpha_k \mathbf{p}_k \).
    • Compute the change in solution \( \mathbf{s}_k = \mathbf{x}_{k+1} - \mathbf{x}_k \) and change in gradient \( \mathbf{y}_k = \nabla f(\mathbf{x}_{k+1}) - \nabla f(\mathbf{x}_k) \).
    • Update the inverse Hessian \( \mathbf{H}_{k+1} \) using the BFGS update formula.
Mathematical Analysis

BFGS Update Formula for Inverse Hessian:

\( \mathbf{H}_{k+1} = \left( \mathbf{I} - \frac{\mathbf{s}_k \mathbf{y}_k^T}{\mathbf{y}_k^T \mathbf{s}_k} \right) \mathbf{H}_k \left( \mathbf{I} - \frac{\mathbf{y}_k \mathbf{s}_k^T}{\mathbf{y}_k^T \mathbf{s}_k} \right) + \frac{\mathbf{s}_k \mathbf{s}_k^T}{\mathbf{y}_k^T \mathbf{s}_k} \)
Simple Python Program
import numpy as np
from scipy.optimize import minimize

def f(x): return x[0]**2 + 2*x[1]**2
def grad_f(x): return np.array([2*x[0], 4*x[1]])

x0 = np.array([5.0, 5.0])
result = minimize(f, x0, method='BFGS', jac=grad_f)

print("BFGS Result:")
print(f"Final point: {result.x}")
print(f"Iterations: {result.nit}")
print(f"Successful: {result.success}")
