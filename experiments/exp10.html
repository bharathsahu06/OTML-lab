Interior Point Method (IPM)

🎯 Aim

To implement the Interior Point Method for constrained optimization using the logarithmic barrier function approach.

📐 Mathematical Procedure
Part A: Constrained Problem

Minimize \( f(x) = x^2 \) subject to \( x \ge 1 \).

Part B: Logarithmic Barrier

Approximate the indicator function with a smooth barrier:

\( \text{minimize } \phi(x) = x^2 - \mu \ln(x - 1) \)

As \( \mu \to 0 \), the solution to the barrier problem approaches the constrained minimum.

💻 Python Implementation
import numpy as np def barrier_objective(x, mu): return x**2 - mu * np.log(x - 1) def solve_ipm(): x = 2.0 # Start in the feasible interior mu = 1.0 alpha = 0.05 for iteration in range(5): # Update mu mu *= 0.5 # Optimize interior function for _ in range(100): grad = 2*x - mu / (x - 1) x = x - alpha * grad print(f"Mu = {mu:.3f}, x = {x:.6f}") solve_ipm()
