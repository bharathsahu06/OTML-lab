Experiment 10: Interior Point Method

📌 Aim

To solve a non-linear constrained optimization problem using the interior point method: minimize f(x) = x² subject to x ≥ 1. The interior point method is a powerful technique for constrained optimization.

📝 Description

The interior point method, also called the barrier method, solves constrained optimization problems by approaching the optimum from inside the feasible region. Instead of stepping directly onto or beyond a constraint boundary, it modifies the objective with a barrier term that becomes very large near the boundary.

As the barrier parameter is gradually adjusted, the solution moves closer to the true constrained optimum while staying feasible throughout the iterations. This barrier-based approach gives a clear view of how modern constrained optimization methods enforce restrictions without leaving the allowed region.

  • The barrier function discourages constraint violation
  • The search starts and remains inside the feasible set
  • The method is widely used in constrained and large-scale optimization
🔧 Algorithm
Step 1: Start with feasible point inside constraint region (x=2)
Step 2: Add barrier function: f(x) + (1/t)×log(x-1)
Step 3: Minimize modified objective
Step 4: Increase t (reduce barrier weight)
Step 5: Repeat until convergence
📐 Formulas
Original Problem:
minimize f(x) = x²
subject to x ≥ 1
Barrier Problem:
minimize f(x) + (1/t)×log(x-1)
where t → ∞ (penalty weight decreases)
Barrier Gradient:
∇f_barrier = 2x - (1/t)×1/(x-1)
📊 Mathematical Procedure
Optimal Solution:
At the constraint boundary x* = 1, f(1) = 1
This is the minimum value in the feasible region
💻 Simple Python Program
import numpy as np # Interior Point Method # Minimize: f(x) = x² # Subject to: x >= 1 def objective(x): return x**2 def barrier(x, t): # Barrier function: -log(x-1) if x <= 1: return float('inf') return -(1/t) * np.log(x - 1) def barrier_gradient(x, t): return 2*x + (1/t) * 1/(x - 1) # Parameters x = 2.0 # Start inside constraint region t = 1.0 # Initial barrier parameter lr = 0.01 # Iterations for iteration in range(10): # Minimize with current barrier for inner in range(50): grad = barrier_gradient(x, t) x = x - lr * grad if x <= 1.01: x = 1.01 obj_val = objective(x) print(f"Iteration {iteration}: x = {x:.6f}, f(x) = {obj_val:.6f}, t = {t}") t *= 10 # Increase t print(f"\nOptimal x ≈ {x:.6f}") print(f"Optimal f(x) ≈ {objective(x):.6f}")
