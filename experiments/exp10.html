Experiment 10: Interior Point Method for Non-Linear Constrained Optimization

1) Aim
Solve the constrained optimization problem using interior point method: $$\min f(x)=x^2 \quad \text{subject to} \quad x\ge1$$
2) Description

This experiment solves a non-linear constrained optimization problem using the interior point (barrier) approach. The idea is to convert the constrained problem into a sequence of unconstrained problems by adding a logarithmic barrier term that prevents the iterate from crossing the boundary $x=1$. Instead of solving directly on the boundary, we start from a strictly feasible point ($x>1$) and move toward the optimum by minimizing the barrier objective repeatedly.

The method followed is: define a barrier function for the inequality constraint, then use Newton updates to minimize the barrier objective for a fixed barrier parameter $t$. After inner convergence, increase $t$ so that the barrier effect reduces and the solution gets closer to the true constrained optimum. This process is repeated until the duality-gap style stopping condition is satisfied, and final $x$ and $f(x)$ are reported.

3) Important Formulas

Original problem: $$\min_{x}\; x^2 \quad \text{s.t.} \quad x\ge1$$

Write constraint as: $$g(x)=x-1>0$$

Barrier objective (for parameter $t>0$): $$\phi_t(x)=t\,x^2-\log(x-1), \quad x>1$$

Gradient and Hessian: $$\phi_t'(x)=2tx-\frac{1}{x-1}$$ $$\phi_t''(x)=2t+\frac{1}{(x-1)^2}$$

Newton update: $$x_{k+1}=x_k-\frac{\phi_t'(x_k)}{\phi_t''(x_k)}$$

Outer stopping criterion (single inequality, $m=1$): $$\frac{m}{t}<\varepsilon$$

4) Important Functions Used
  • barrier_phi(x, t): barrier objective value.
  • barrier_grad(x, t): gradient of barrier objective.
  • barrier_hess(x, t): Hessian (second derivative).
  • interior_point_1d(...): performs outer barrier loop and inner Newton iterations.
5) Algorithm (Step by Step)
  1. Choose strictly feasible initial point $x_0>1$.
  2. Initialize barrier parameter $t$ and scaling factor $\mu>1$.
  3. For fixed $t$, minimize $\phi_t(x)$ by Newton method.
  4. Use backtracking to keep all iterates feasible ($x>1$).
  5. When inner Newton converges, increase $t\leftarrow\mu t$.
  6. Repeat until $1/t<\varepsilon$.
  7. Return final $x^*$ and objective $f(x^*)=x^{*2}$.
6) Mathematical Procedure (First Inner Newton Step)

Take initial $x_0=2$, $t=1$.

Compute gradient: $$\phi_t'(2)=2(1)(2)-\frac{1}{2-1}=4-1=3$$
Compute Hessian: $$\phi_t''(2)=2(1)+\frac{1}{(2-1)^2}=2+1=3$$
Newton step: $$\Delta x=-\frac{\phi_t'(2)}{\phi_t''(2)}=-\frac{3}{3}=-1$$
Candidate update: $$x_1=x_0+\Delta x=2-1=1$$

Since $x=1$ is on boundary and barrier needs $x>1$, backtracking reduces step size and keeps $x_1>1$.

7) Given Input
  • Objective: $f(x)=x^2$
  • Constraint: $x\ge1$
  • Initial point: $x_0=2.0$
  • Initial barrier parameter: $t_0=1.0$
8) Python Implementation & Output
import numpy as np def f(x): return x**2 def barrier_phi(x, t): return t * (x**2) - np.log(x - 1.0) def barrier_grad(x, t): return 2.0 * t * x - 1.0 / (x - 1.0) def barrier_hess(x, t): return 2.0 * t + 1.0 / ((x - 1.0) ** 2) def interior_point_1d(x0=2.0, t0=1.0, mu=10.0, eps=1e-6, newton_tol=1e-10, max_outer=20, max_inner=50): x = float(x0) t = float(t0) m = 1 # number of inequality constraints history = [] if x <= 1.0: raise ValueError("Initial point must satisfy x > 1.") for outer in range(1, max_outer + 1): for inner in range(1, max_inner + 1): g = barrier_grad(x, t) h = barrier_hess(x, t) if abs(g) < newton_tol: break # Newton step delta = -g / h step = 1.0 # Backtracking to maintain feasibility x > 1 while x + step * delta <= 1.0: step *= 0.5 # Armijo-style sufficient decrease (simple form) while barrier_phi(x + step * delta, t) > barrier_phi(x, t) + 1e-4 * step * g * delta: step *= 0.5 if step < 1e-12: break x = x + step * delta history.append((outer, inner, t, x, f(x), abs(g))) if m / t < eps: break t *= mu return x, f(x), history # Given input parameters x0 = 2.0 t0 = 1.0 mu = 10.0 eps = 1e-6 x_star, f_star, steps = interior_point_1d(x0=x0, t0=t0, mu=mu, eps=eps) print("INPUT") print("Objective: minimize f(x) = x^2") print("Constraint: x >= 1") print(f"Initial x0: {x0}") print(f"Initial barrier parameter t0: {t0}") print(f"Barrier update factor mu: {mu}") print(f"Stopping epsilon: {eps}") print("\nOUTPUT") print(f"Optimal x*: {x_star:.8f}") print(f"Minimum objective value f(x*): {f_star:.8f}") print(f"Constraint check (x* >= 1): {x_star >= 1.0}") print(f"Distance from boundary (x* - 1): {x_star - 1.0:.8e}") print("Program executed successfully.")
