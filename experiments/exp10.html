Experiment 10: Interior Point Method for Constrained Optimization

Home / Experiment 10
Aim

To solve a non-linear constrained optimization problem using the Interior Point Method for the function \( \min f(x) = x^2 \) subject to \( x \ge 1 \).

Description

The Interior Point Method (or Barrier Method) converts inequality constraints into terms part of the objective function using barrier functions such as the logarithmic barrier. This transforms a constrained problem into a sequence of unconstrained problems.

Inequality: \( g(x) = x - 1 \ge 0 \)
Barrier Objective: \( \phi_t(x) = x^2 - \frac{1}{t} \ln(x - 1) \)

As \( t \to \infty \), the solution to the barrier problem approaches the solution of the original constrained problem.

Algorithm
  1. Form the barrier objective function \( \phi_t(x) = f(x) - \mu \ln(g(x)) \).
  2. Initialize parameters \( t = 1 \) and \( \mu = 1/t \).
  3. Repeat for the barrier factor \( t \):
    • Compute the derivative: \( \phi_t'(x) = 2x - \frac{1}{t(x-1)} = 0 \).
    • Solve for \( x \): \( 2x^2 - 2x - 1/t = 0 \).
    • Increase \( t \) and update \( \mu \).
  4. Observe the point \( x \) approaching 1 as \( t \) increases.
Mathematical Analysis

Solving for \( x \) at different values of \( t \):

  • If \( t = 1 \): \( x = \frac{1 + \sqrt{1 + 2}}{2} \approx 1.3660 \)
  • If \( t = 10 \): \( x = \frac{1 + \sqrt{1 + 0.2}}{2} \approx 1.0477 \)
  • If \( \lim t \to \infty \): \( x = \frac{1 + \sqrt{1}}{2} = 1.0 \) (Solution)
Simple Python Program
import numpy as np

def barrier_solve(t):
    # Roots of quadratic 2x^2 - 2x - 1/t = 0
    # From quadratic formula: x = (2 + sqrt(4 - 4*2*(-1/t))) / 4
    # x = (1 + sqrt(1 + 2/t)) / 2
    return (1 + np.sqrt(1 + 2/t)) / 2

print("t | x(t)")
print("-" * 15)

# Sequence of increasing t
for t in [1, 5, 10, 50, 100, 1000]:
    sol = barrier_solve(t)
    print(f"{t:4} | {sol:.6f}")

print("\nResult: Solution converges to x=1.0 as t increases.")
