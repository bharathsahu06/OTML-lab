Experiment 10: Distributed Gradient Descent

Home / Experiment 10
Aim

To simulate the Distributed Gradient Descent algorithm by partitioning data across virtual "workers" and aggregating their gradients to update a global model.

Description

Distributed optimization is essential for large-scale machine learning where data cannot fit on a single machine. In this simulation, data points are distributed among workers. Each worker calculates gradients based on its local subset, and the master node averages these gradients to perform the descent step.

Dataset: \( X = [1, 2, 3, 4, 5, 6], Y = [3, 5, 7, 9, 11, 13] \).
Workers: 3 (2 samples each).

Algorithm
  1. Partition the dataset into \( N \) subsets (one per worker).
  2. Initialize global parameters \( w_{global} \) and \( b_{global} \).
  3. For each iteration:
    • Send current global parameters to all workers.
    • Workers: Compute local gradients \( \nabla w_i, \nabla b_i \) on local data.
    • Master: Aggregate local gradients by averaging: \( \nabla w_{avg} = \frac{1}{N} \sum \nabla w_i \).
    • Update global parameters: \( \theta_{global} = \theta_{global} - \eta \nabla \theta_{avg} \).
Mathematical Procedure

Global update rule:

\( w_{k+1} = w_k - \eta \left[ \frac{1}{N} \sum_{j=1}^N \left( \frac{1}{n_j} \sum_{i \in Batch_j} \nabla L(w_k; x_i, y_i) \right) \right] \)
Simple Python Program
import numpy as np

# Setup
X = np.array([1, 2, 3, 4, 5, 6], dtype=float)
Y = np.array([3, 5, 7, 9, 11, 13], dtype=float)
n_workers = 3
workers = [ (X[i:i+2], Y[i:i+2]) for i in range(0, 6, 2) ]

# Training
w, b, lr = 0.0, 0.0, 0.01

for _ in range(1000):
    gw_list, gb_list = [], []
    for xw, yw in workers:
        err = (w * xw + b) - yw
        gw_list.append( np.mean(2 * xw * err) )
        gb_list.append( np.mean(2 * err) )
    
    w -= lr * np.mean(gw_list)
    b -= lr * np.mean(gb_list)

print(f"Optimal w: {w:.4f}")
print(f"Optimal b: {b:.4f}")
