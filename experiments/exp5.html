Experiment 5: Linear Objective with Linear Constraints Using CVXPY

1) Aim
Minimize the linear objective function $$f(x,y)=3x+2y$$ subject to constraints: $$x+y\ge 4,\quad x\ge 0,\quad y\ge 0$$ using CVXPY.
2) Description

This experiment solves a linear programming problem where both objective and constraints are linear. The goal is to find feasible values of $x$ and $y$ that satisfy all inequalities while giving the smallest possible value of $3x+2y$. CVXPY is used to model the optimization problem in a direct mathematical way and then solve it using a suitable LP solver.

The method followed is: define decision variables, construct the linear objective, add all given constraints, and solve the optimization model. After obtaining the result, we check whether the solution satisfies each constraint and then report the minimum objective value. Since this is a convex linear program, the solver returns a global optimum over the feasible region.

3) Important Formulas

Objective: $$\min_{x,y}\; 3x+2y$$

Constraints: $$x+y\ge 4, \quad x\ge 0, \quad y\ge 0$$

CVXPY form: $$\text{Minimize}(3x+2y)\quad \text{subject to}\quad [x+y\ge4,\;x\ge0,\;y\ge0]$$

4) Important Functions Used
  • cp.Variable(): creates optimization variables.
  • cp.Minimize(expr): defines minimization objective.
  • cp.Problem(objective, constraints): creates optimization problem.
  • problem.solve(): solves problem using available solver.
  • problem.status, problem.value: solution status and optimal value.
5) Algorithm (Step by Step)
  1. Import CVXPY library.
  2. Create decision variables $x$ and $y$.
  3. Define linear objective $3x+2y$.
  4. Add constraints $x+y\ge4$, $x\ge0$, $y\ge0$.
  5. Build optimization problem.
  6. Solve using problem.solve().
  7. Read optimal $x$, $y$, and minimum objective.
  8. Verify constraints and print results.
6) Mathematical Procedure

Given: $$\min 3x+2y\quad \text{s.t.}\quad x+y\ge4,\;x\ge0,\;y\ge0$$

From $x+y\ge4$, write $y=4-x$ on boundary (candidate optimum line), with $0\le x\le4$.

Substitute into objective: $$f(x)=3x+2(4-x)=x+8$$

Since $f(x)=x+8$ increases with $x$, minimum occurs at smallest feasible $x$, i.e. $x=0$.

Then: $$y=4$$

Minimum value: $$f(0,4)=3(0)+2(4)=8$$

7) Given Input
  • Objective: $f(x,y)=3x+2y$
  • Constraints: $x+y\ge4$, $x\ge0$, $y\ge0$
import cvxpy as cp # Decision variables x = cp.Variable() y = cp.Variable() # Objective and constraints objective = cp.Minimize(3 * x + 2 * y) constraints = [x + y >= 4, x >= 0, y >= 0] # Build and solve optimization problem problem = cp.Problem(objective, constraints) optimal_value = problem.solve() print("INPUT") print("Objective: minimize f(x, y) = 3x + 2y") print("Constraints: x + y >= 4, x >= 0, y >= 0") print("\nOUTPUT") print("Status:", problem.status) print(f"Optimal x: {x.value:.6f}") print(f"Optimal y: {y.value:.6f}") print(f"Minimum objective value: {optimal_value:.6f}") print("\nConstraint Check") print(f"x + y = {(x.value + y.value):.6f} (>= 4)") print(f"x >= 0 -> {x.value >= -1e-8}") print(f"y >= 0 -> {y.value >= -1e-8}") print("Program executed successfully.")
