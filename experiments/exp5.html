Experiment 05: Linear and Quadratic Programming

Home / Experiment 05
Aim

To implement and solve Linear Programming Problems (LPP) using Graphical and Simplex methods, and Quadratic Programming Problems (QPP) using KKT conditions.

Description

Part 1: Linear Programming (LPP)

Maximize \( Z = 6A + 5B \)

Subject to:

  • \( A + B \le 5 \)
  • \( 3A + 2B \le 12 \)
  • \( A, B \ge 0 \)

Part 2: Quadratic Programming (QPP)

Minimize \( f(x, y) = x^2 + y^2 \)

Subject to:

  • \( x + y - 4 \le 0 \)
  • \( x - 1 \le 0 \)
Algorithm

Graphical Method:

  1. Convert inequalities into equations.
  2. Plot the lines on a graph.
  3. Identify the feasible region.
  4. Find corner points of the region and evaluate the objective function.

KKT Method for QPP:

  1. Define the Lagrangian: \( L = f(x,y) + \lambda_1 g_1(x,y) + \lambda_2 g_2(x,y) \).
  2. Compute partial derivatives of \( L \) wrt \( x, y, \lambda_1, \lambda_2 \).
  3. Solve the system of equations.
Formulas
Maximize \( Z = \mathbf{c}^T \mathbf{x} \) s.t. \( \mathbf{Ax} \le \mathbf{b} \)
Lagrangian \( L(\mathbf{x}, \lambda) = f(\mathbf{x}) + \sum \lambda_i g_i(\mathbf{x}) \)
Simple Python Program
import numpy as np
import sympy as sp

# 1. Linear Programming (Simplex Table approach)
t = np.array([
    [1, 1, 1, 0, 5],
    [3, 2, 0, 1, 12],
    [-6, -5, 0, 0, 0]
], float)

while np.any(t[-1, :-1] < 0):
    pc = np.argmin(t[-1, :-1])
    ratios = t[:-1, -1] / t[:-1, pc]
    pr = np.argmin(ratios)
    t[pr] /= t[pr, pc]
    for i in range(len(t)):
        if i != pr: t[i] -= t[i, pc] * t[pr]

print(f"LPP Optimal Z: {t[-1, -1]}")

# 2. Quadratic Programming (KKT using Sympy)
x, y, l1, l2 = sp.symbols('x y l1 l2', real=True)
f = x**2 + y**2
L = f + l1*(x + y - 4) + l2*(x - 1)
eqs = [sp.diff(L, x), sp.diff(L, y), x + y - 4, x - 1]
sol = sp.solve(eqs, [x, y, l1, l2])
print(f"QPP Optimal Point: {sol}")
Expected Output
LPP Optimal Z: 27.0
QPP Optimal Point: {l1: -6, l2: 4, x: 1, y: 3}
Minimum Value: 10
