Linear Programming (CVXPY)

🎯 Aim

To solve a standard Linear Programming (LP) problem where both the objective function and constraints are linear in the decision variables.

📐 Mathematical Procedure
Part A: Problem Formulation

Objective function: \( \text{minimize } f(x, y) = 3x + 2y \)

Subject to:

  • \( x + y \ge 4 \)
  • \( x \ge 0, y \ge 0 \)
Part B: Corner Point Analysis

In linear programming, the optimum always lies at a vertex of the feasible region:

  • At \((4, 0)\): \( 3(4) + 2(0) = 12 \)
  • At \((0, 4)\): \( 3(0) + 2(4) = 8 \) (Minimum)
💻 Python Implementation
import cvxpy as cp # Define variables x = cp.Variable() y = cp.Variable() # Define objective objective = cp.Minimize(3*x + 2*y) # Define constraints constraints = [x + y >= 4, x >= 0, y >= 0] # Solve problem prob = cp.Problem(objective, constraints) prob.solve() print(f"Status: {prob.status}") print(f"Optimal x: {x.value:.6f}") print(f"Optimal y: {y.value:.6f}") print(f"Minimum objective: {prob.value:.6f}")
📊 Expected Output
Terminal Console
Status: optimal
Optimal x: 0.000000
Optimal y: 4.000000
Minimum objective: 8.000000
✅ Result & Verification
Since the coefficient of \( y \) (2) is smaller than that of \( x \) (3), the algorithm chooses to maximize \( y \) within the constraint to achieve the lowest overall value.
