Experiment 5: Minimize f(x,y)=3x+2y (Using PuLP and Without PuLP)

1) Aim

a) Minimize $$f(x,y)=3x+2y$$ subject to $$x+y\ge 4,\;x\ge0,\;y\ge0$$ using PuLP.

b) Minimize the same problem without PuLP.

2) Description

This is a linear programming problem. With PuLP, we model variables, objective, and constraints and solve automatically with the LP solver.

Without PuLP, we solve by corner-point analysis: evaluate objective at feasible vertices and choose the minimum.

For this problem, the optimum is at $(x,y)=(0,4)$ with minimum objective value $8$.

3) Important Formulas

Objective: $$\min\;3x+2y$$

Constraints: $$x+y\ge4,\;x\ge0,\;y\ge0$$

Boundary: $$y=4-x,\;0\le x\le4$$

Reduced objective on boundary: $$3x+2(4-x)=x+8$$

Minimum at smallest feasible $x$: $$x=0,\;y=4$$

4A) Python Implementation Using PuLP
import pulp model = pulp.LpProblem("LP_Q5", pulp.LpMinimize) x = pulp.LpVariable("x", lowBound=0) y = pulp.LpVariable("y", lowBound=0) model += 3*x + 2*y model += x + y >= 4 model.solve(pulp.PULP_CBC_CMD(msg=0)) print("Status:", pulp.LpStatus[model.status]) print("x* =", pulp.value(x)) print("y* =", pulp.value(y)) print("f(x*,y*) =", pulp.value(model.objective))
4B) Python Implementation Without PuLP
# Corner-point method candidates = [(0, 4), (4, 0)] # feasible boundary corners in first quadrant def f(x, y): return 3*x + 2*y best = min(candidates, key=lambda p: f(p[0], p[1])) print("x* =", best[0]) print("y* =", best[1]) print("f(x*,y*) =", f(best[0], best[1]))
x* = 0 y* = 4 f(x*,y*) = 8
