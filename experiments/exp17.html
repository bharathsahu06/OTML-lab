Experiment 17: Group Lasso for Grouped Sparsity

📌 Aim

To demonstrate grouped sparsity using Group Lasso regularization. Group Lasso encourages entire groups of features to be selected/eliminated together.

Dataset: x₁=[1,2], x₂=[2,1], x₃=[1,1], x₄=[1,1], y=[9,10]

📝 Description

Group Lasso extends sparse regularization from individual features to predefined groups of features. Instead of penalizing each coefficient independently, it penalizes the norm of an entire group, so related variables tend to be selected or removed together.

This is useful when features naturally belong to the same category, measurement source, or logical block. The method improves interpretability because the model can retain or discard whole groups rather than making isolated coefficient decisions.

  • The grouping structure adds prior knowledge to the optimization problem
  • Whole feature groups can shrink to zero together
  • The method is useful in structured feature selection and multi-task learning
🔧 Algorithm
Step 1: Define feature groups
Step 2: Apply Group Lasso penalty
Step 3: Solve optimization problem
Step 4: Identify selected/eliminated groups
📐 Formulas
Group Lasso Objective:
L(w) = MSE + λ × Σ||w_g||₂
where g indexes feature groups
Group Penalty:
Penalizes L2-norm of each group, not individual weights
📊 Mathematical Procedure

Given data: x₁=[1,2], x₂=[2,1], x₃=[1,1], x₄=[1,1], y=[9,10].

Grouping: define feature groups g₁,...,g₄ (one feature per group in this demo).
Objective: minimize MSE(w) + λΣ||w_g||₂.
Optimization: solve for w using numerical minimization (L-BFGS-B) of the grouped objective.
Interpretation: groups with tiny norms are suppressed; active groups remain selected together.
💻 Simple Python Program
import numpy as np from scipy.optimize import minimize # Simple Group Lasso Implementation X = np.array([[1, 2, 1, 1], [2, 1, 1, 1]]) y = np.array([9, 10]) # Define 4 groups (each feature is a group) groups = [[0], [1], [2], [3]] # Initialize weights w = np.zeros(4) lr = 0.01 lambda_reg = 0.1 def group_lasso_penalty(w, groups, lambda_reg): penalty = 0 for group in groups: group_norm = np.linalg.norm(w[group]) penalty += lambda_reg * group_norm return penalty # Objective for scipy optimizer def objective(w_vec): residual = X @ w_vec - y mse = np.mean(residual**2) penalty = group_lasso_penalty(w_vec, groups, lambda_reg) return mse + penalty result = minimize(objective, w, method='L-BFGS-B') w = result.x loss = np.mean((X @ w - y)**2) penalty = group_lasso_penalty(w, groups, lambda_reg) print(f"Solver success: {result.success}") print(f"MSE: {loss:.4f}, Penalty: {penalty:.4f}, Total: {loss+penalty:.4f}") print(f"Weights: {w}") print(f"\nSelected features: {np.where(np.abs(w) > 0.01)[0]}")
