Experiment 17: Grouped Sparsity Using Group Lasso

1) Aim
Demonstrate grouped sparsity using Group Lasso for:
  • $x_1=[1,2]$, $x_2=[2,1]$, $x_3=[1,1]$, $x_4=[1,1]$
  • $y=[9,10]$
2) Description

In this experiment, we use Group Lasso regularization for linear regression. Group Lasso does not shrink each weight independently; instead, it shrinks entire groups of coefficients together using an $\ell_2$ norm inside each group and an $\ell_1$-style sum across groups. This helps identify relevant feature groups and set less useful groups close to zero.

The linear model is: $\hat{y}=w_1x_1+w_2x_2+w_3x_3+w_4x_4$. The method followed is: define feature groups (here, Group-1: $[w_1,w_2]$, Group-2: $[w_3,w_4]$), compute gradient of MSE, and then apply group-wise soft-thresholding (proximal step). This proximal update shrinks each group by its norm, so an entire group can become exactly zero when its norm is small enough relative to regularization strength. After iterations, we compare group norms to verify grouped sparsity behavior.

3) Important Formulas

Model: $$\hat{y}=Xw$$

Objective (Group Lasso): $$\min_w\; \frac{1}{2n}\|Xw-y\|_2^2 + \lambda\sum_{g\in G}\|w_g\|_2$$

Gradient of smooth part: $$g(w)=\frac{1}{n}X^T(Xw-y)$$

Proximal group update for each group $g$: $$v_g = w_g - \alpha \; g_g, \qquad w_g \leftarrow \max\left(0,1-\frac{\alpha\lambda}{\|v_g\|_2}\right)v_g$$

4) Important Functions Used
  • group_soft_threshold(...): performs group-wise proximal shrinkage.
  • group_lasso_train(...): trains model with proximal gradient descent.
  • group_lasso_objective(...): computes MSE + group penalty.
5) Algorithm (Step by Step)
  1. Build design matrix $X=[x_1,x_2,x_3,x_4]$.
  2. Define groups: $G_1=[0,1]$, $G_2=[2,3]$.
  3. Initialize weights $w=0$.
  4. Compute gradient of MSE part.
  5. Take gradient step: $v=w-\alpha g$.
  6. Apply group soft-threshold to each group.
  7. Repeat for fixed iterations.
  8. Print learned weights, objective, and group norms.
6) Mathematical Procedure (Complete Step-by-Step)

Given: $X=\begin{bmatrix}1 & 2 & 1 & 1 \\ 2 & 1 & 1 & 1\end{bmatrix}, y=\begin{bmatrix}9 \\ 10\end{bmatrix}, n=2$. Groups: $G_1=(w_1,w_2), G_2=(w_3,w_4)$. Initialize: $w^{(0)}=[0,0,0,0]^T$.

Step 1: Compute step size used in code $$\frac{1}{n}X^TX=\begin{bmatrix}2.5 & 2.0 & 1.5 & 1.5 \\ 2.0 & 2.5 & 1.5 & 1.5 \\ 1.5 & 1.5 & 1.0 & 1.0 \\ 1.5 & 1.5 & 1.0 & 1.0\end{bmatrix}$$ Largest eigenvalue is $L\approx 6.5$, so $\alpha=\frac{0.9}{L}\approx 0.13846$. With $\lambda=2$, $\tau=\alpha\lambda\approx 0.27692$.
Step 2: Compute gradient at initial point Prediction $\hat{y}^{(0)}=[0,0]^T$. Error $e^{(0)}=[-9,-10]^T$. $g^{(0)}=\frac{1}{n}X^Te^{(0)} = \begin{bmatrix}-14.5 \\ -14.0 \\ -9.5 \\ -9.5\end{bmatrix}$.
Step 3: Gradient step $v^{(0)}=w^{(0)}-\alpha g^{(0)}=-\alpha g^{(0)} = \begin{bmatrix}2.007692 \\ 1.938462 \\ 1.315385 \\ 1.315385\end{bmatrix}$.
Step 4: Group-wise proximal shrinkage Group 1: $v_{G_1}=[2.007692, 1.938462]$, $\|v_{G_1}\|_2 \approx 2.7910$, scale $\approx 0.9008 \Rightarrow w_{G_1}^{(1)}\approx [1.809,1.747]$. Group 2: $v_{G_2}=[1.315385, 1.315385]$, $\|v_{G_2}\|_2 \approx 1.8602$, scale $\approx 0.8511 \Rightarrow w_{G_2}^{(1)}\approx [1.119,1.119]$.

Step 5: Why grouped sparsity appears. If $\|v_g\|_2 \le \tau$, then the whole group becomes zero. In final output, $[w_3,w_4]=[0,0]$, meaning Group 2 is eliminated together.

7) Given Input
  • $x_1=[1,2], x_2=[2,1], x_3=[1,1], x_4=[1,1]$
  • $y=[9,10]$
  • Groups: $(w_1,w_2)$ and $(w_3,w_4)$
  • Regularization: $\lambda=2.0$
8) Python Implementation & Output
import numpy as np # Given input x1 = np.array([1, 2], dtype=float) x2 = np.array([2, 1], dtype=float) x3 = np.array([1, 1], dtype=float) x4 = np.array([1, 1], dtype=float) y = np.array([9, 10], dtype=float) X = np.column_stack([x1, x2, x3, x4]) # Group definitions: [w1, w2] and [w3, w4] groups = [np.array([0, 1]), np.array([2, 3])] def group_lasso_objective(X, y, w, groups, lam): n = len(y) mse_part = 0.5 / n * np.sum((X @ w - y) ** 2) group_penalty = lam * sum(np.linalg.norm(w[g], ord=2) for g in groups) return mse_part + group_penalty def group_soft_threshold(v_group, thresh): norm_v = np.linalg.norm(v_group, ord=2) if norm_v == 0 or norm_v <= thresh: return np.zeros_like(v_group) scale = 1.0 - thresh / norm_v return scale * v_group def group_lasso_train(X, y, groups, lam=2.0, iterations=2000): n, d = X.shape w = np.zeros(d) # Step size from Lipschitz constant of gradient of 0.5/n ||Xw-y||^2 L = np.linalg.eigvalsh((X.T @ X) / n).max() alpha = 0.9 / (L + 1e-12) history = [] for _ in range(iterations): grad = (X.T @ (X @ w - y)) / n # Gradient step v = w - alpha * grad # Proximal group-wise shrinkage w_new = np.zeros_like(w) for g in groups: w_new[g] = group_soft_threshold(v[g], alpha * lam) w = w_new history.append(group_lasso_objective(X, y, w, groups, lam)) return w, alpha, history lam = 2.0 iterations = 2000 w_opt, alpha, history = group_lasso_train(X, y, groups, lam=lam, iterations=iterations) y_pred = X @ w_opt obj_val = group_lasso_objective(X, y, w_opt, groups, lam) group1_norm = np.linalg.norm(w_opt[groups[0]], ord=2) group2_norm = np.linalg.norm(w_opt[groups[1]], ord=2) print("INPUT") print("x1 =", x1) print("x2 =", x2) print("x3 =", x3) print("x4 =", x4) print("y =", y) print("Groups: G1=(w1,w2), G2=(w3,w4)") print(f"lambda = {lam}") print(f"iterations = {iterations}") print(f"step size (alpha) = {alpha:.6f}") print("\nOUTPUT") print(f"w1 = {w_opt[0]:.6f}") print(f"w2 = {w_opt[1]:.6f}") print(f"w3 = {w_opt[2]:.6f}") print(f"w4 = {w_opt[3]:.6f}") print(f"Final objective value = {obj_val:.6f}") print("Predicted y:", np.round(y_pred, 6)) print("\nGROUP SPARSITY CHECK") print(f"||[w1, w2]||_2 = {group1_norm:.6f}") print(f"||[w3, w4]||_2 = {group2_norm:.6f}") print("Group 1 near zero:", group1_norm < 1e-3) print("Group 2 near zero:", group2_norm < 1e-3) print("Program executed successfully.")
