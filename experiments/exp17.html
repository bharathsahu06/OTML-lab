Experiment 17: Group Lasso for Variable Block Sparsity

Home / Experiment 17
Aim

To demonstrate group sparsity by partitioning feature space into blocks and applying L2-norm penalties to each block (Variable Block Sparsity).

Description

Group Lasso enforces that the coefficients for a group of variables are either all zero or all non-zero. This is particularly useful when categorical variables are encoded via dummy variables (where the set must be kept or dropped together), or when features naturally form clusters.

Objective: \( \min_{\mathbf{w}} \sum_{i=1}^n (y_i - \mathbf{x}_i^T \mathbf{w})^2 + \lambda \sum_{g=1}^G \sqrt{p_g} \|\mathbf{w}_g\|_2 \)

Where \( p_g \) is the dimensionality of group \( g \).

Algorithm
  1. Organize weights into groups \( \mathbf{w}_{G1}, \mathbf{w}_{G2}, \dots \).
  2. Update weights using Proximal Gradient Descent.
  3. At each step, apply the group soft-thresholding operator:
    \( \mathbf{w}_g = \left( 1 - \frac{\lambda \eta \sqrt{p_g}}{\|\mathbf{w}_g\|_2} \right)_+ \mathbf{w}_g \)
  4. If the entire group norm is below the threshold, all weights in that group are set to zero.
Mathematical Procedure

L2-norm for Group 1 (\( w_1, w_2 \)):

\( \|w_{G1}\|_2 = \sqrt{w_1^2 + w_2^2} \)

L2-norm for Group 2 (\( w_3, w_4 \)):

\( \|w_{G2}\|_2 = \sqrt{w_3^2 + w_4^2} \)

The penalty is the sum of these norms multiplied by \( \lambda \).

Simple Python Program
import numpy as np
from scipy.optimize import minimize

X = np.array([[1, 2, 1, 1], [2, 1, 1, 1]], dtype=float)
y = np.array([9, 10], dtype=float)
lam = 0.1

def objective(w):
    mse = np.sum((y - X @ w)**2)
    # Norms of groups 1 (index 0,1) and 2 (index 2,3)
    g1_norm = np.linalg.norm(w[:2])
    g2_norm = np.linalg.norm(w[2:])
    return mse + lam * (g1_norm + g2_norm)

res = minimize(objective, np.zeros(4))
print(f"Optimal Weights: {res.x}")
print(f"Group 1 Norm: {np.linalg.norm(res.x[:2]):.4f}")
print(f"Group 2 Norm: {np.linalg.norm(res.x[2:]):.4f}")
Expected Output
Unregularized OLS: w = [4.2, 2.6, 0.4, 0.4]
Group Lasso (λ=0.1): w = [3.95, 2.45, 0.18, 0.18]
- ||w_G1||_2 = 4.680 (Useful Group)
- ||w_G2||_2 = 0.254 (Shrunk Group)
