Group Lasso extends L1 regularization to feature groups. Instead of penalizing individual weights, it penalizes the L2 norm of weight groups. This causes entire feature groups to be eliminated simultaneously.

Applications: Feature selection, categorical variables, multi-task learning

Step 2: Apply Group Lasso penalty
Step 3: Solve optimization problem
Step 4: Identify selected/eliminated groups

import numpy as np # Simple Group Lasso Implementation X = np.array([[1, 2, 1, 1], [2, 1, 1, 1]]) y = np.array([9, 10]) # Define 4 groups (each feature is a group) groups = [[0], [1], [2], [3]] # Initialize weights w = np.zeros(4) lr = 0.01 lambda_reg = 0.1 def group_lasso_penalty(w, groups, lambda_reg): penalty = 0 for group in groups: group_norm = np.linalg.norm(w[group]) penalty += lambda_reg * group_norm return penalty # Training for epoch in range(100): y_pred = X @ w error = y_pred - y # MSE gradient grad = 2 * X.T @ error # Add group penalty (simplified) for group in groups: group_norm = np.linalg.norm(w[group]) + 1e-8 grad[group] += lambda_reg * w[group] / group_norm w -= lr * grad if epoch % 20 == 0: loss = np.mean(error**2) penalty = group_lasso_penalty(w, groups, lambda_reg) print(f"Epoch {epoch}: Loss={loss:.4f}, Total={loss+penalty:.4f}, w={w}") print(f"\nSelected features: {np.where(np.abs(w) > 0.01)[0]}")
