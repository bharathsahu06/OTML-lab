Experiment 11: Interior Point Method (2D Problem)

Home / Experiment 11
Aim

To solve a non-linear constrained optimization problem using the Interior Point Method for the function \( \min f(x, y) = x + y \) subject to \( x \ge 1 \) and \( y \ge 1 \).

Description

Applying the Interior Point Method to a 2D problem shows how multiple constraints are handled. Each constraint contributes a log-barrier term to the objective function, and the collective force keeps the solution within the feasible region.

Inequalities: \( g_1(x) = x - 1 \ge 0, g_2(y) = y - 1 \ge 0 \)
Barrier Objective: \( \phi_t(x, y) = x + y - \frac{1}{t} [\ln(x - 1) + \ln(y - 1)] \)
Algorithm
  1. Form the collective barrier objective: \( \phi_t(x, y) = f(x, y) - \mu \sum \ln(g_i) \).
  2. Repeat for the barrier factor \( t \):
    • Compute the partial derivatives: \( \frac{\partial \phi}{\partial x} = 1 - \frac{1}{t(x-1)} \).
    • Compute the partial derivatives: \( \frac{\partial \phi}{\partial y} = 1 - \frac{1}{t(y-1)} \).
    • Set derivatives to zero and solve for \( (x, y) \).
    • Increase \( t \).
  3. Observe convergence to the optimal point \( (1, 1) \).
Mathematical Analysis

The solution for the barrier objective is \( x_t = 1 + 1/t \) and \( y_t = 1 + 1/t \):

  • If \( t = 1 \): \( x = 2, y = 2 \)
  • If \( t = 10 \): \( x = 1.1, y = 1.1 \)
  • If \( t = 100 \): \( x = 1.01, y = 1.01 \)
Simple Python Program
import numpy as np

def barrier_solve_2d(t):
    # Partial derivatives show x = 1 + 1/t and y = 1 + 1/t
    return (1 + 1/t), (1 + 1/t)

print("t | x(t) | y(t)")
print("-" * 15)

# Current t sequence
for t in [1, 5, 10, 50, 100]:
    x, y = barrier_solve_2d(t)
    print(f"{t:4} | {x:.4f} | {y:.4f}")

print("\nResult: Solution converges to x=1, y=1 as t increases.")
