Experiment 11: Interior Point Method (2D)

📌 Aim

To solve a 2D non-linear constrained optimization problem using the interior point method: minimize f(x,y) = x + y subject to x ≥ 1, y ≥ 1.

📝 Description

Interior point methods extend naturally to multivariable constrained optimization, where the feasible set becomes a region rather than a single interval. The method still uses barrier terms to guide the search through the interior of the admissible set instead of crossing the boundaries.

In higher dimensions, these methods show how numerical optimization combines geometry, feasibility, and iterative improvement in one framework. They are widely used because they remain stable on constrained problems while steadily moving toward an optimum.

  • The objective determines which direction improves the solution
  • The constraints shape the feasible region in multiple dimensions
  • The barrier term prevents the iterates from leaving the admissible region
🔧 Algorithm
Step 1: Initialize (x,y) inside feasible region
Step 2: Create barrier objective with penalty for constraint violation
Step 3: Perform gradient descent on barrier function
Step 4: Gradually reduce barrier weight (increase t)
Step 5: Repeat until optimal solution found
📐 Formulas
Problem:
minimize f(x,y) = x + y
subject to x ≥ 1, y ≥ 1
Barrier Formulation:
minimize (x + y) - (1/t)[log(x-1) + log(y-1)]
Gradient:
∂f/∂x = 1 - (1/t)×1/(x-1)
∂f/∂y = 1 - (1/t)×1/(y-1)
📊 Mathematical Procedure
Original problem: minimize f(x,y) = x + y with constraints x ≥ 1, y ≥ 1.
Feasibility: the smallest feasible values are x = 1 and y = 1.
Optimal point: (x*, y*) = (1, 1), and f(x*,y*) = 1 + 1 = 2.
Interior-point interpretation: barrier updates keep x > 1, y > 1 and converge toward (1,1) as t increases.
💻 Simple Python Program
import numpy as np # Interior Point Method 2D # min: f(x,y) = x + y # s.t: x >= 1, y >= 1 def objective(x, y): return x + y def barrier_gradient(x, y, t): # Gradient of barrier function gx = 1 - (1/t) / (x - 1) gy = 1 - (1/t) / (y - 1) return np.array([gx, gy]) # Initialize point = np.array([3.0, 3.0]) t = 0.1 lr = 0.01 # Barrier method iterations for iteration in range(8): # Inner minimization for inner in range(100): grad = barrier_gradient(point[0], point[1], t) point = point - lr * grad # Ensure feasibility point = np.maximum(point, 1.01) obj = objective(point[0], point[1]) print(f"Iter {iteration}: x={point[0]:.4f}, y={point[1]:.4f}, f={obj:.4f}, t={t}") t *= 10 print(f"\nOptimal: x={point[0]:.4f}, y={point[1]:.4f}") print(f"Optimal value = {objective(point[0], point[1]):.4f}")
