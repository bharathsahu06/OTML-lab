Experiment 11: Interior Point Method for Non-Linear Constrained Optimization (Two Variables)

1) Aim
Solve the constrained optimization problem using interior point method: $$\min f(x,y)=x+y \quad \text{subject to} \quad x\ge1,\;y\ge1$$
2) Description

This experiment solves a constrained non-linear optimization problem using the interior point (log-barrier) method. The constraints are handled by adding barrier terms, which keep iterates strictly inside the feasible region ($x>1, y>1$) during optimization. Instead of directly working on the boundary, we solve a sequence of unconstrained barrier subproblems and gradually reduce barrier influence.

Interior Point Methods extend easily to multi-dimensional constrained optimization. The barrier approach remains elegant: convert inequality constraints into logarithmic barriers that push the solution away from boundaries. The Hessian grows (becoming more ill-conditioned) near boundaries due to the barrier curvature, but Newton's method adapts automatically through its second-order information. Scaling the barrier parameter $t$ creates a homothopy path from problems far from boundaries toward the true constrained optimum..

The key insight in higher dimensions is that the barrier objective remains smooth and well-behaved as long as we stay inside the feasible region. Newton's method efficiently navigates this smooth landscape, converging rapidly at each temperature level. The outer iteration then adjusts the barrier strength, creating a sequence of problems that converge to the constrained optimum. This method avoids explicit constraint handling and the combinatorial complexity of vertex enumeration.

3) Important Formulas

Original problem: $$\min_{x,y}\;x+y \quad \text{s.t.} \quad x\ge1,\;y\ge1$$

Constraints in barrier form: $$g_1(x,y)=x-1>0,\qquad g_2(x,y)=y-1>0$$

Barrier objective: $$\phi_t(x,y)=t(x+y)-\log(x-1)-\log(y-1)$$

Gradient: $$\nabla\phi_t(x,y)=\begin{bmatrix} t-\frac{1}{x-1}\\ t-\frac{1}{y-1} \end{bmatrix}$$

Hessian: $$\nabla^2\phi_t(x,y)=\begin{bmatrix} \frac{1}{(x-1)^2} & 0 \\ 0 & \frac{1}{(y-1)^2} \end{bmatrix}$$

Newton step: $$\Delta z = -\left(\nabla^2\phi_t\right)^{-1}\nabla\phi_t, \quad z=\begin{bmatrix}x\\y\end{bmatrix}$$

Outer stopping criterion ($m=2$ inequalities): $$\frac{m}{t}<\varepsilon$$

4) Important Functions Used
  • barrier_phi(xy, t): evaluates barrier objective.
  • barrier_grad(xy, t): computes gradient vector.
  • barrier_hess(xy, t): computes Hessian matrix.
  • interior_point_2d(...): runs outer barrier loop and inner Newton iterations.
5) Algorithm (Step by Step)
  1. Start from strictly feasible point $(x_0,y_0)$ with $x_0>1, y_0>1$.
  2. Set initial barrier parameter $t$ and update factor $\mu>1$.
  3. For fixed $t$, compute Newton direction from gradient and Hessian.
  4. Use backtracking line search to keep iterates feasible ($x>1,y>1$).
  5. Repeat inner Newton steps until gradient norm is small.
  6. Increase barrier parameter $t\leftarrow\mu t$.
  7. Repeat until $2/t<\varepsilon$.
  8. Return final $(x^*,y^*)$ and $f(x^*,y^*)$.
6) Mathematical Procedure (Complete Step-by-Step)

Given: Initial point $(x_0,y_0)=(2,2)$, Initial barrier parameter $t=1$

1. Compute gradient at (2,2): $$\nabla\phi_t(2,2)=\begin{bmatrix}1-\frac{1}{2-1}\\1-\frac{1}{2-1}\end{bmatrix}=\begin{bmatrix}0\\0\end{bmatrix}$$
2. Compute Hessian at (2,2): $$\nabla^2\phi_t(2,2)=\begin{bmatrix}1&0\\0&1\end{bmatrix}$$
3. Newton direction: $$\Delta z=-\left(\nabla^2\phi_t\right)^{-1}\nabla\phi_t=-\begin{bmatrix}1&0\\0&1\end{bmatrix}\begin{bmatrix}0\\0\end{bmatrix}=\begin{bmatrix}0\\0\end{bmatrix}$$
4. Update point for this barrier subproblem: $$z_1=z_0+\alpha\Delta z=z_0$$, so $(x,y)$ remains $(2,2)$ for $t=1$.
5. Increase barrier parameter: $t\leftarrow\mu t$ with $\mu=10$.

6. Solve next subproblem with same Newton process.

7. Repeat outer iterations until $2/t<\varepsilon$.

8. Final sequence approaches $(1,1)$ from the interior, giving objective near $f(x^*,y^*)\approx 2$.

7) Given Input
  • Objective: $f(x,y)=x+y$
  • Constraints: $x\ge1, y\ge1$
  • Initial point: $(2.0,2.0)$
  • Initial barrier parameter: $t_0=1.0$
