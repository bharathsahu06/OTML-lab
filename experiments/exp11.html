Experiment 11: Elastic Net Regularization

Home / Experiment 11
Aim

To implement Elastic Net Regularization, which combines L1 (LASSO) and L2 (Ridge) penalties to improve model robustness and handle correlated features.

Description

Elastic Net is useful when there are multiple features which are correlated with one another. Lasso is likely to pick one of these at random, while Elastic Net is likely to pick both. It minimizes the following cost function:

\( J(w) = \frac{1}{n} \sum (y_i - \hat{y}_i)^2 + \lambda_1 \sum |w_j| + \lambda_2 \sum w_j^2 \)

This experiment uses 2 features \( x_1, x_2 \) and optimized coefficients \( w_1, w_2 \).

Algorithm
  1. Initialize weights \( \mathbf{w} \) and hyperparameters \( \lambda_1, \lambda_2 \).
  2. For each epoch:
    • Compute predictions: \( \hat{\mathbf{y}} = \mathbf{X}\mathbf{w} \).
    • Compute MSE gradient: \( \nabla_{MSE} = \frac{2}{n}\mathbf{X}^T(\hat{\mathbf{y}} - \mathbf{y}) \).
    • Compute L1 gradient: \( \nabla_{L1} = \lambda_1 \cdot \text{sgn}(\mathbf{w}) \).
    • Compute L2 gradient: \( \nabla_{L2} = 2\lambda_2 \cdot \mathbf{w} \).
    • Update weights: \( \mathbf{w} = \mathbf{w} - \eta (\nabla_{MSE} + \nabla_{L1} + \nabla_{L2}) \).
Mathematical Procedure

The total gradient used for the update is:

\( \nabla J(\mathbf{w}) = \frac{2}{n}\mathbf{X}^T(\mathbf{X}\mathbf{w} - \mathbf{y}) + \lambda_1 \text{sgn}(\mathbf{w}) + 2\lambda_2 \mathbf{w} \)
Simple Python Program
import numpy as np

# Sample Data (2 features)
X = np.array([[1, 2], [2, 1], [3, 4]], dtype=float)
y = np.array([3, 4, 7], dtype=float)
w = np.array([1.0, 1.0])
l1_ratio, l2_ratio = 0.01, 0.05
lr = 0.01

for _ in range(500):
    pred = X @ w
    grad_mse = (2 / len(y)) * X.T @ (pred - y)
    grad_l1 = l1_ratio * np.sign(w)
    grad_l2 = 2 * l2_ratio * w
    
    w -= lr * (grad_mse + grad_l1 + grad_l2)

print(f"Final Weights: {w}")
