Experiment 2: Newton's Method for Optimization

Home / Experiment 2
Aim

To implement the Newton's Method for optimizing the function \( f(x) = x^4 - 3x^2 + 2 \), identifying critical points where the derivative approaches zero.

Description

Newton's Method is a root-finding algorithm that can also be used for optimization by finding where the first derivative of a function is zero. It uses the second derivative (Hessian) information, making it converge much faster than Gradient Descent for smooth functions.

\( f(x) = x^4 - 3x^2 + 2 \)
First Derivative: \( f'(x) = 4x^3 - 6x \)
Second Derivative: \( f''(x) = 12x^2 - 6 \)
Algorithm
  1. Initialize the point \( x_0 \).
  2. Repeat for iterations \( k \):
    • Compute first derivative \( f'(x_k) \).
    • Compute second derivative \( f''(x_k) \).
    • Update: \( x_{k+1} = x_k - \frac{f'(x_k)}{f''(x_k)} \).
  3. Continue until the residual \( |f'(x)| \) is sufficiently small.
Mathematical Analysis

To find the critical points, we set \( f'(x) = 0 \):

\( 4x^3 - 6x = 0 \implies 2x(2x^2 - 3) = 0 \)
Critical points at: \( x = 0, x = \sqrt{1.5} \approx 1.22, x = -\sqrt{1.5} \approx -1.22 \).
Simple Python Program
import numpy as np

def f_prime(x): return 4*x**3 - 6*x
def f_double_prime(x): return 12*x**2 - 6

# Initial point
x = 2.0
print("k | x_k | f'(x_k)")
print("-" * 15)

for k in range(10):
    grad = f_prime(x)
    hess = f_double_prime(x)
    
    x = x - grad / hess
    print(f"{k} | {x:.6f} | {f_prime(x):.2e}")
    
    if abs(grad) < 1e-10:
        break

print(f"Final Critical Point: x = {x:.4f}")
