Newton's method is a root-finding algorithm that uses both first and second derivatives (Hessian). It's faster than gradient descent because it has quadratic convergence. Each iteration updates using the Newton step formula which considers the curvature of the function.

ðŸ”§ Algorithm
Step 1: Initialize x to a starting point
Step 2: Calculate first derivative: f'(x)
Step 3: Calculate second derivative: f''(x)
Step 4: Update: x = x - f'(x) / f''(x)
Step 5: Repeat until convergence
