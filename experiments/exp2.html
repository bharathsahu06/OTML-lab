Newton's Method for Optimization

🎯 Aim

To implement the second-order Newton-Raphson optimization algorithm to minimize the non-convex objective function \( f(x) = x^4 - 3x^2 + 2 \).

📐 Mathematical Procedure
Part A: Function Analysis

Objective function: \( f(x) = x^4 - 3x^2 + 2 \)

First derivative (Gradient):

\( f'(x) = 4x^3 - 6x \)

Second derivative (Hessian):

\( f''(x) = 12x^2 - 6 \)
Part B: Newton Update Rule

The update step for Newton's method is defined as:

\( x_{k+1} = x_k - \frac{f'(x_k)}{f''(x_k)} \)
Part C: Manual Iteration (Starting point \( x_0 = 1.5 \) )
  • \( f'(1.5) = 4(1.5)^3 - 6(1.5) = 13.5 - 9 = 4.5 \)
  • \( f''(1.5) = 12(1.5)^2 - 6 = 27 - 6 = 21 \)
  • Update: \( x_1 = 1.5 - (4.5 / 21) \approx 1.2857 \)
💻 Python Implementation
# Newton's Method Optimization import numpy as np def f(x): return x**4 - 3*x**2 + 2 def df(x): return 4*x**3 - 6*x def ddf(x): return 12*x**2 - 6 # Starting points x_start = [1.5, 0.1] for x0 in x_start: x = x0 print(f"--- Starting from x0 = {x0} ---") for i in range(5): x = x - df(x)/ddf(x) print(f"Iter {i+1}: x = {x:.6f}, f(x) = {f(x):.6f}") print()

✅ Result & Verification
Newton's method converges extremely fast (quadratically) compared to Gradient Descent. Starting at 1.5, it finds the local minimum \( x^* = \sqrt{1.5} \approx 1.224 \). Starting at 0.1, it slides toward the local maximum/stationary point at 0.
