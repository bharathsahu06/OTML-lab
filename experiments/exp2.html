Experiment 2: Newton's Method for Optimization

📌 Aim

To implement Newton's method (also called Newton-Raphson method) to find the minimum of f(x) = x⁴ - 3x² + 2. This method uses second-order derivatives and converges faster than gradient descent.

Newton's method is a root-finding algorithm that uses both first and second derivatives (Hessian). It's faster than gradient descent because it has quadratic convergence. Each iteration updates using the Newton step formula which considers the curvature of the function.

🔧 Algorithm
Step 1: Initialize x to a starting point
Step 2: Calculate first derivative: f'(x)
Step 3: Calculate second derivative: f''(x)
Step 4: Update: x = x - f'(x) / f''(x)
Step 5: Repeat until convergence
📐 Formulas
Objective Function:
f(x) = x⁴ - 3x² + 2
First Derivative:
f'(x) = 4x³ - 6x
Second Derivative:
f''(x) = 12x² - 6
Newton Update Rule:
x(n+1) = x(n) - f'(x(n)) / f''(x(n))
📊 Mathematical Procedure

Example with x₀ = 2.0:

Iteration 0:
x₀ = 2.0
f(2) = 16 - 12 + 2 = 6
f'(2) = 4(8) - 6(2) = 32 - 12 = 20
f''(2) = 12(4) - 6 = 48 - 6 = 42
x₁ = 2.0 - 20/42 = 2.0 - 0.476 = 1.524
Iteration 1:
x₁ = 1.524
f(1.524) ≈ 5.42
f'(1.524) ≈ 14.05
f''(1.524) ≈ 21.86
x₂ = 1.524 - 14.05/21.86 ≈ 0.880

... (continuing iterations converge to a critical point)

💻 Simple Python Program
# Newton's Method for f(x) = x⁴ - 3x² + 2 def function(x): return x**4 - 3*x**2 + 2 def first_derivative(x): return 4*x**3 - 6*x def second_derivative(x): return 12*x**2 - 6 def newtons_method(x_start, iterations): x = x_start for i in range(iterations): f_prime = first_derivative(x) f_double_prime = second_derivative(x) if abs(f_double_prime) < 1e-10: print("Division by zero!") break x = x - f_prime / f_double_prime if i % 5 == 0: print(f"Iteration {i}: x = {x:.6f}, f(x) = {function(x):.6f}") return x # Run the algorithm final_x = newtons_method(2.0, 20) print(f"\nFinal x = {final_x:.6f}") print(f"Final f(x) = {function(final_x):.6f}")
