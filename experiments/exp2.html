Experiment 2: Newton's Method for Optimization

1) Aim
Implementation of Newton's Method for optimization of: $$f(x)=x^4-3x^2+2$$
2) Description

Newton's Method is an iterative optimization technique that uses first and second derivatives of a function to quickly approach a stationary point.

Newton's Method is a second-order optimization algorithm that uses both first and second derivatives to find function minima. Unlike Gradient Descent which only uses gradient direction, Newton's Method leverages curvature information (the Hessian matrix) to make better-informed steps. This leads to superlinear convergence—typically reaching the optimum in far fewer iterations than first-order methods.

The update rule incorporates the inverse Hessian, which scales the gradient by the local curvature. In regions of high curvature, steps are smaller; where curvature is low, steps are larger. This adaptive scaling allows Newton's Method to navigate non-uniform loss landscapes efficiently. However, computing and inverting the Hessian is expensive, especially in high dimensions, limiting its use to problems where this cost is justified by faster convergence.

Newton's Method requires computing both $f'(x)$ and $f''(x)$, making it more complex but also more powerful than gradient-based methods for well-conditioned problems. It is particularly effective for quadratic functions, where it converges in a single iteration.

3) Important Formulas

Given function:

$$f(x)=x^4-3x^2+2$$

First derivative:

$$f'(x)=4x^3-6x$$

Second derivative:

$$f''(x)=12x^2-6$$

Newton update for optimization:

$$x_{k+1}=x_k-\frac{f'(x_k)}{f''(x_k)}$$

Stopping condition:

$$|x_{k+1}-x_k|<\varepsilon$$
4) Important Functions Used in Program
  • f(x): returns function value.
  • df(x): returns first derivative $f'(x)$.
  • d2f(x): returns second derivative $f''(x)$.
  • newton_optimize(x0, tol, max_iter): performs step-by-step Newton iterations.
5) Algorithm (Step by Step)
  1. Choose initial value $x_0$, tolerance $\varepsilon$, and maximum iterations.
  2. Compute $f'(x_k)$ and $f''(x_k)$.
  3. If $f''(x_k)$ is very close to zero, stop (division risk).
  4. Compute next value: $$x_{k+1}=x_k-\frac{f'(x_k)}{f''(x_k)}$$
  5. If $|x_{k+1}-x_k|<\varepsilon$, stop.
  6. Else set $x_k=x_{k+1}$ and repeat.
  7. Report optimum $x^*$ and $f(x^*)$.
6) Mathematical Procedure (Complete Step-by-Step)

Given function: $$f(x)=x^4-3x^2+2$$

Derivatives: $$f'(x)=4x^3-6x,\qquad f''(x)=12x^2-6$$

Newton update formula for optimization: $$x_{k+1}=x_k-\frac{f'(x_k)}{f''(x_k)}$$

Use $x_0=2$.

1. At $x_0=2$: $$f'(2)=20,\qquad f''(2)=42$$ $$x_1=2-\frac{20}{42}=1.523809524$$
2. At $x_1=1.523809524$: $$f'(x_1)=5.010258,\qquad f''(x_1)=21.863946$$ $$x_2=1.523809524-\frac{5.010258}{21.863946}=1.294653$$
3. At $x_2=1.294653$: $$f'(x_2)=0.912095,\qquad f''(x_2)=14.113528$$ $$x_3=1.294653-\frac{0.912095}{14.113528}=1.230028$$

4. Continue same computation until $|x_{k+1}-x_k|<10^{-6}$.

5. Converged point is near $x^*=1.224745$ with $f(x^*)\approx-0.25$.

7) Given Input
  • Function: $f(x)=x^4-3x^2+2$
  • Initial guess: $x_0=2.0$
  • Tolerance: $10^{-6}$
  • Maximum iterations: $20$
8) Python Implementation & Output
