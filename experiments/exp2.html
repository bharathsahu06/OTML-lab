Experiment 2: Newton's Method for Optimization

1) Aim
Implementation of Newton's Method for optimization of: $$f(x)=x^4-3x^2+2$$
2) Description

Newton's Method is an iterative optimization technique that uses first and second derivatives of a function to quickly approach a stationary point.

For optimization, we solve $f'(x)=0$ using the update rule:

$$x_{k+1}=x_k-\frac{f'(x_k)}{f''(x_k)}$$

In this experiment, we apply Newton's Method to find a minimum of $f(x)=x^4-3x^2+2$.

The method followed is: start with an initial guess $x_0$, calculate $f'(x_k)$ and $f''(x_k)$ at each step, and then update $x_k$ using the Newton formula. After every update, we check the difference $|x_{k+1}-x_k|$ to decide whether convergence has been achieved. This gives faster convergence than basic gradient methods because second-derivative (curvature) information is used in each iteration.

3) Important Formulas

Given function:

$$f(x)=x^4-3x^2+2$$

First derivative:

$$f'(x)=4x^3-6x$$

Second derivative:

$$f''(x)=12x^2-6$$

Newton update for optimization:

$$x_{k+1}=x_k-\frac{f'(x_k)}{f''(x_k)}$$

Stopping condition:

$$|x_{k+1}-x_k|<\varepsilon$$
4) Important Functions Used in Program
  • f(x): returns function value.
  • df(x): returns first derivative $f'(x)$.
  • d2f(x): returns second derivative $f''(x)$.
  • newton_optimize(x0, tol, max_iter): performs step-by-step Newton iterations.
5) Algorithm (Step by Step)
  1. Choose initial value $x_0$, tolerance $\varepsilon$, and maximum iterations.
  2. Compute $f'(x_k)$ and $f''(x_k)$.
  3. If $f''(x_k)$ is very close to zero, stop (division risk).
  4. Compute next value: $$x_{k+1}=x_k-\frac{f'(x_k)}{f''(x_k)}$$
  5. If $|x_{k+1}-x_k|<\varepsilon$, stop.
  6. Else set $x_k=x_{k+1}$ and repeat.
  7. Report optimum $x^*$ and $f(x^*)$.
6) Mathematical Procedure (Complete Step-by-Step)

Given function: $$f(x)=x^4-3x^2+2$$

Derivatives: $$f'(x)=4x^3-6x,\qquad f''(x)=12x^2-6$$

Newton update formula for optimization: $$x_{k+1}=x_k-\frac{f'(x_k)}{f''(x_k)}$$

Use $x_0=2$.

1. At $x_0=2$: $$f'(2)=20,\qquad f''(2)=42$$ $$x_1=2-\frac{20}{42}=1.523809524$$
2. At $x_1=1.523809524$: $$f'(x_1)=5.010258,\qquad f''(x_1)=21.863946$$ $$x_2=1.523809524-\frac{5.010258}{21.863946}=1.294653$$
3. At $x_2=1.294653$: $$f'(x_2)=0.912095,\qquad f''(x_2)=14.113528$$ $$x_3=1.294653-\frac{0.912095}{14.113528}=1.230028$$

4. Continue same computation until $|x_{k+1}-x_k|<10^{-6}$.

5. Converged point is near $x^*=1.224745$ with $f(x^*)\approx-0.25$.

7) Given Input
  • Function: $f(x)=x^4-3x^2+2$
  • Initial guess: $x_0=2.0$
  • Tolerance: $10^{-6}$
  • Maximum iterations: $20$
8) Python Implementation & Output
import numpy as np # Function and derivatives def f(x): return x**4 - 3*x**2 + 2 def df(x): return 4*x**3 - 6*x def d2f(x): return 12*x**2 - 6 def newton_optimize(x0, tol=1e-6, max_iter=20): x = float(x0) history = [] for k in range(1, max_iter + 1): grad = df(x) hess = d2f(x) if abs(hess) < 1e-12: raise ValueError("Second derivative is too close to zero; Newton step is unstable.") x_next = x - grad / hess history.append((k, x, grad, hess, x_next, abs(x_next - x))) if abs(x_next - x) < tol: return x_next, f(x_next), history x = x_next return x, f(x), history # Given input x0 = 2.0 tol = 1e-6 max_iter = 20 x_star, f_star, steps = newton_optimize(x0, tol, max_iter) print("INPUT") print(f"Function: f(x) = x^4 - 3x^2 + 2") print(f"Initial guess (x0): {x0}") print(f"Tolerance: {tol}") print(f"Max iterations: {max_iter}") print("\nITERATION TABLE") print("k\t x_k\t\t f'(x_k)\t f''(x_k)\t x_(k+1)\t |x_(k+1)-x_k|") for k, xk, grad, hess, xn, diff in steps: print(f"{k}\t {xk:.6f}\t {grad:.6f}\t {hess:.6f}\t {xn:.6f}\t {diff:.6e}") print("\nOUTPUT") print(f"Optimal x*: {x_star:.6f}") print(f"Minimum function value f(x*): {f_star:.6f}") print("Program executed successfully.")
