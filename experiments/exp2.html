Experiment 02: Linear Regression using Gradient Descent

Home / Experiment 02
Aim

To implement Linear Regression using the Gradient Descent optimization technique to find the best-fit line by iteratively updating the slope and intercept.

Description

Gradient Descent is an iterative optimization algorithm used to minimize a cost function. In linear regression, it adjusts the slope (m) and intercept (c) by calculating the gradients of the Mean Squared Error with respect to these parameters.

Dataset:

  • X: [8, 18, 24]
  • Y: [15, 30, 25]
Algorithm
  1. Initialize slope \( m \) and intercept \( c \) (e.g., \( m=1.0, c=0.0 \)).
  2. Choose a learning rate \( \eta \) and number of iterations.
  3. For each iteration:
    • Calculate the predictions \( \hat{y} = mx + c \).
    • Calculate the error \( (y - \hat{y}) \).
    • Compute gradients: \( \frac{\partial MSE}{\partial m} \) and \( \frac{\partial MSE}{\partial c} \).
    • Update parameters: \( m = m - \eta \cdot \frac{\partial MSE}{\partial m} \) and \( c = c - \eta \cdot \frac{\partial MSE}{\partial c} \).
  4. Repeat until convergence.
Formulas
Prediction: \( \hat{y}_i = mx_i + c \)
Gradient (m): \( D_m = \frac{-2}{n} \sum_{i=1}^n x_i(y_i - \hat{y}_i) \)
Gradient (c): \( D_c = \frac{-2}{n} \sum_{i=1}^n (y_i - \hat{y}_i) \)
Updates: \( m = m - \eta D_m, \quad c = c - \eta D_c \)
Mathematical Procedure

Initial state: \( m=1.0, c=0.0, \eta=0.001 \). Data: \( (8,15), (18,30), (24,25) \).

Iteration 1:

  • Predictions: \( 1(8)+0=8, 1(18)+0=18, 1(24)+0=24 \)
  • Errors: \( 15-8=7, 30-18=12, 25-24=1 \)
  • Gradient \( D_m = \frac{-2}{3} (8(7) + 18(12) + 24(1)) = \frac{-2}{3} (56+216+24) = -197.33 \)
  • Gradient \( D_c = \frac{-2}{3} (7+12+1) = -13.33 \)
  • New \( m = 1.0 - 0.001(-197.33) \approx 1.197 \)
  • New \( c = 0.0 - 0.001(-13.33) \approx 0.013 \)
Simple Python Program
import numpy as np

x = np.array([8, 18, 24])
y = np.array([15, 30, 25])

m, c = 1.0, 0.0
L = 0.001  # Learning Rate
iterations = 1000

for i in range(iterations):
    y_pred = m * x + c
    Dm = (-2/len(x)) * sum(x * (y - y_pred))
    Dc = (-2/len(x)) * sum(y - y_pred)
    m = m - L * Dm
    c = c - L * Dc

print(f"Final slope: {m:.2f}")
print(f"Final intercept: {c:.2f}")
