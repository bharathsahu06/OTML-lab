Stochastic Gradient Descent (SGD)

🎯 Aim

To implement the Stochastic Gradient Descent algorithm which updates parameters using only a single data point at a time, providing a scalable alternative to batch methods.

📐 Mathematical Procedure
Part A: Model Specification

We assume the relationship \( y = w^2 \). For a given point \( (x_i, y_i) \), the squared error loss is:

\( L(w) = (y_i - w^2)^2 \)
Part B: Stochastic Gradient

The derivative of the loss with respect to weight \( w \) is:

\( \frac{\partial L}{\partial w} = 2(y_i - w^2) \cdot (-2w) = -4w(y_i - w^2) \)
Part C: Iterative Update ( \(\eta = 0.05\), \(w_0 = 5.0\) )

For each sample, perform the update: \( w \leftarrow w - \eta(-4w(y_i-w^2)) \)

  • Sample 1 (\(x=1, y=1\)): \( g = -4(5)(1-25) = 480 \); \( w_{new} = 5 - 0.05(480) = -19 \)
  • (Note: High learning rates cause instability in SGD; proper tuning required)
💻 Python Implementation
import numpy as np # Sample dataset x_data = np.array([1, 2, 3, 4]) y_data = np.array([1, 4, 9, 16]) w = 5.0 alpha = 0.05 epochs = 5 for epoch in range(epochs): for xi, yi in zip(x_data, y_data): y_pred = w**2 grad = -4 * w * (yi - y_pred) w = w - alpha * grad print(f"Epoch {epoch+1}: w = {w:.6f}") print(f"\nFinal weight: {w:.8f}")
