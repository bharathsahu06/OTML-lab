Experiment 03: Newton's Method for Optimization

Home / Experiment 03
Aim

To implement the Newton's Method for finding the minimum of a function using second-order derivatives (Hessian matrix).

Description

Newton's Method is a second-order optimization algorithm. Unlike Gradient Descent, which uses only the first derivative, Newton's method uses both the gradient and the curvature (Hessian) to take more direct steps toward the minimum. It typically converges faster but requires calculating high-order derivatives.

Test Functions:

  • 1D: \( f(x) = (1 - x)^2 \)
  • 2D: Rosenbrock Function \( f(x, y) = (1 - x)^2 + 100(y - x^2)^2 \)
Algorithm
  1. Define the objective function \( f(x) \).
  2. Calculate the first derivative \( f'(x) \) (Gradient) and second derivative \( f''(x) \) (Hessian).
  3. Choose an initial starting point \( x_0 \).
  4. Update the point using the Newton step: \( x_{k+1} = x_k - \frac{f'(x_k)}{f''(x_k)} \).
  5. Repeat until the change \( |x_{k+1} - x_k| \) is within a tolerance.
Formulas

1D Case:

\( x_{k+1} = x_k - [f''(x_k)]^{-1} f'(x_k) \)

Multi-variate Case:

\( \mathbf{x}_{k+1} = \mathbf{x}_k - \mathbf{H}_f(\mathbf{x}_k)^{-1} \nabla f(\mathbf{x}_k) \)

Where \( \mathbf{H}_f \) is the Hessian matrix and \( \nabla f \) is the gradient vector.

Mathematical Procedure

For \( f(x) = (1 - x)^2 \) starting at \( x_0 = 0 \):

  • \( f'(x) = -2(1 - x) = 2x - 2 \)
  • \( f''(x) = 2 \)

Iteration 1:

  • \( f'(0) = 2(0) - 2 = -2 \)
  • \( f''(0) = 2 \)
  • \( \Delta = -(-2)/2 = 1.0 \)
  • \( x_1 = 0 + 1.0 = 1.0 \) (Minimum reached)
Simple Python Program
import sympy as sp

x = sp.symbols('x')
f = (1 - x)**2
f_x = sp.diff(f, x)
f_xx = sp.diff(f_x, x)

# Newton Step
xk = 0.0
for k in range(5):
    grad = float(f_x.subs(x, xk))
    hess = float(f_xx.subs(x, xk))
    delta = -grad / hess
    xk = xk + delta
    print(f"Iteration {k+1}: x = {xk}")
    if abs(delta) < 1e-6: break

print("Optimal x:", xk)
