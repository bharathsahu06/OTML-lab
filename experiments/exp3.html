Experiment 3: Stochastic Gradient Descent for f(x) = x^2

📌 Aim

To implement Stochastic Gradient Descent (SGD) for the function f(x) = x^2. This experiment highlights how stochastic-style updates can still move the variable toward the minimum even when each step is noisy.

📝 Description

Stochastic Gradient Descent (SGD) is a variation of gradient descent in which updates are made using limited information at each step instead of using the full objective in one exact update. For this experiment, we apply that idea to f(x) = x^2 and simulate the stochastic behavior with small random variation in the gradient.

Because each update is based on a noisy estimate, the optimization path is less smooth than standard gradient descent. Even so, the overall trend still moves x toward 0, helping illustrate how SGD behaves while minimizing the function f(x) = x^2.

  • Updates happen more frequently than in batch gradient descent
  • Each step is cheap to compute but less stable
  • Noise in the gradient causes small fluctuations around the minimum
🔧 Algorithm
Step 1: Start with an initial value such as x = 5.0
Step 2: Choose a learning rate and number of epochs
Step 3: Estimate a stochastic gradient using 2x plus a small random variation
Step 4: Update x using the stochastic gradient
Step 5: Repeat the process until x moves close to the minimum
📐 Formulas
Objective Function:
f(x) = x2
True Gradient:
df/dx = 2x
Stochastic Gradient Estimate:
g(x) = 2x + noise
SGD Update Rule:
xn+1 = xn - αg(xn)
📊 Mathematical Procedure

Suppose x = 5.0 and α = 0.05.

Update 1:
True gradient = 2(5.0) = 10.0
Assume random noise = 0.14
Stochastic gradient = 10.14
New x = 5.0 - 0.05(10.14) = 4.493
Update 2:
True gradient is recomputed at the new x value
A new random noise value is added
x is updated again using the noisy gradient estimate

The updates do not move smoothly like standard gradient descent, but they still trend toward x = 0 over time.

💻 Simple Python Program
import random random.seed(42) # Stochastic Gradient Descent for f(x) = x^2 # Random noise is added to simulate a stochastic gradient. def stochastic_gradient_descent(x_start, learning_rate, epochs): x = x_start for epoch in range(epochs): stochastic_gradient = 2 * x + random.uniform(-0.5, 0.5) x = x - learning_rate * stochastic_gradient if epoch % 10 == 0: loss = x ** 2 print(f"Epoch {epoch}: x = {x:.6f}, Loss = {loss:.6f}") return x final_x = stochastic_gradient_descent(5.0, 0.05, 100) print(f"\nFinal x = {final_x:.6f}") print(f"Final Loss = {final_x**2:.6f}")
