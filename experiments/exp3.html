Experiment 3: Stochastic Gradient Descent (SGD)

Home / Experiment 3
Aim

To implement the Stochastic Gradient Descent (SGD) algorithm for optimizing the objective function \( f(x) = x^2 \) using data samples \( x = [1, 2, 3, 4] \) and \( y = [1, 4, 9, 16] \).

Description

Stochastic Gradient Descent is a variant of Gradient Descent that updates parameters using only a single training sample at a time instead of the entire dataset. This makes it significantly faster for large-scale datasets, though the path to convergence is more "stochastic" or noisy.

Update Rule for one sample \( i \):
\( x = x - \alpha \nabla f_i(x) \)
Algorithm
  1. Initialize the point \( x \) randomly.
  2. Set the learning rate \( \alpha = 0.05 \).
  3. For each epoch:
    • Shuffle the training dataset.
    • For each individual sample \( (x_i, y_i) \):
      • Compute the local gradient based on sample \( i \).
      • Update the parameter \( x \).
  4. Observe convergence toward the global optimum \( x = 0 \).
Simple Python Program
import numpy as np
import random

x_data = np.array([1, 2, 3, 4])
y_data = np.array([1, 4, 9, 16])

# Initialization
w = 5.0
alpha = 0.01
epochs = 5

for epoch in range(epochs):
    # Shuffle for pure stochasticity
    indices = list(range(len(x_data)))
    random.shuffle(indices)
    
    for idx in indices:
        # Gradient of (w^2 - y)^2 with respect to w is 2 * (w^2 - y) * 2w
        grad = 4 * w * (w**2 - y_data[idx])
        w = w - alpha * grad
        print(f"Epoch {epoch} | w updated: {w:.4f}")

print(f"Final Weights: {w:.4f}")
Expected Output
Epoch 0 | w updated: 4.0400
Epoch 0 | w updated: 3.1027
...
Final Epoch updates:
Epoch 4 | w updated: 0.1245
Epoch 4 | w updated: 0.0034
Final Weights: 0.0000
Result: Convergence to 0 with noisy path.
