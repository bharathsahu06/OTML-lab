Experiment 3: Stochastic Gradient Descent (SGD)

1) Aim
Implementation of Stochastic Gradient Descent for the given data:
  • Input $x=[1,2,3,4]$
  • Target $y=[1,4,9,16]$
2) Description

Stochastic Gradient Descent (SGD) is an optimization method where model parameters are updated using one training sample at a time, instead of using the full dataset in every update. This makes each update faster and helps in learning from data incrementally.

In this experiment, we fit a linear model $\hat{y}=wx+b$ to the given data using SGD. Even though the target values follow a quadratic relation, SGD is used here to demonstrate the parameter update mechanism clearly.

The method followed is: initialize parameters, pick one sample $(x_i,y_i)$, compute prediction error, and immediately update $w$ and $b$. This sample-wise update is repeated for all points across multiple epochs until the loss decreases and parameters stabilize. Because updates happen more frequently than batch gradient descent, SGD often reaches a good solution quickly for small and large datasets.

3) Important Formulas

Model: $$\hat{y}_i=wx_i+b$$

Sample loss (squared error): $$L_i=(\hat{y}_i-y_i)^2$$

Sample gradients: $$\frac{\partial L_i}{\partial w}=2(\hat{y}_i-y_i)x_i, \qquad \frac{\partial L_i}{\partial b}=2(\hat{y}_i-y_i)$$

SGD update rules: $$w\leftarrow w-\alpha\frac{\partial L_i}{\partial w}, \qquad b\leftarrow b-\alpha\frac{\partial L_i}{\partial b}$$

Final MSE (for reporting): $$J(w,b)=\frac{1}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)^2$$

4) Important Functions Used in Program
  • sgd_linear_regression(x, y, alpha, epochs, seed): performs SGD updates for $w$ and $b$.
  • mse_loss(x, y, w, b): computes Mean Squared Error over full dataset.
5) Algorithm (Step by Step)
  1. Initialize $w=0$, $b=0$, learning rate $\alpha$, and number of epochs.
  2. For each epoch, shuffle the training indices.
  3. Pick one sample $(x_i,y_i)$.
  4. Compute prediction $\hat{y}_i=wx_i+b$.
  5. Compute error $e_i=\hat{y}_i-y_i$.
  6. Compute gradients $2e_ix_i$ and $2e_i$.
  7. Update $w$ and $b$ immediately.
  8. Repeat for all samples and all epochs.
  9. Report final $w$, $b$, predictions, and MSE.
6) Mathematical Procedure (First Two SGD Updates)

Given: $x=[1,2,3,4]$, $y=[1,4,9,16]$, initial $w=0$, $b=0$, learning rate $\alpha=0.01$.

For first sample $(x_1,y_1)=(1,1)$: $$\hat{y}_1=0(1)+0=0, \quad e_1=0-1=-1$$ $$\frac{\partial L_1}{\partial w}=2(-1)(1)=-2, \quad \frac{\partial L_1}{\partial b}=2(-1)=-2$$ $$w=0-0.01(-2)=0.02, \quad b=0-0.01(-2)=0.02$$
For second sample $(x_2,y_2)=(2,4)$: $$\hat{y}_2=0.02(2)+0.02=0.06, \quad e_2=0.06-4=-3.94$$ $$\frac{\partial L_2}{\partial w}=2(-3.94)(2)=-15.76, \quad \frac{\partial L_2}{\partial b}=2(-3.94)=-7.88$$ $$w=0.02-0.01(-15.76)=0.1776, \quad b=0.02-0.01(-7.88)=0.0988$$

SGD continues in this way for all samples and epochs.

7) Given Input
  • $x=[1,2,3,4]$
  • $y=[1,4,9,16]$
  • Learning rate $\alpha=0.01$
  • Epochs $=500$
8) Python Implementation & Output
