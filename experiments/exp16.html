Experiment 16: L1/L2 and Elastic Net Regularization

📌 Aim

To implement linear regression with different regularization techniques:

  • L1 regularization (Lasso): λ = 0.1
  • L2 regularization (Ridge): λ = 0.1
  • Elastic Net (combination): λ₁ = 0.05, λ₂ = 0.05
📝 Description

Three major regularization strategies used in linear regression are L1, L2, and Elastic Net. Regularization changes the objective function by adding a penalty on the weights so the model does not become unnecessarily complex.

These penalties help reduce overfitting and improve generalization, especially when data is noisy or features are correlated. Comparing the three approaches shows that different penalties produce different coefficient patterns and different modeling behavior.

  • L1 regularization encourages sparsity and can remove unimportant features
  • L2 regularization shrinks weights smoothly and often improves numerical stability
  • Elastic Net combines both effects, which is useful when features are correlated
📐 Formulas
L1 Regularization (Lasso):
L(w) = MSE + λ||w||₁
L2 Regularization (Ridge):
L(w) = MSE + λ||w||₂²
Elastic Net:
L(w) = MSE + λ₁||w||₁ + λ₂||w||₂²
💻 Simple Python Program
import numpy as np from sklearn.linear_model import Lasso, Ridge, ElasticNet # Sample data X = np.array([[1], [2], [3], [4]]) y = np.array([2, 4, 6, 8]) # L1 Regularization (Lasso) lasso = Lasso(alpha=0.1) lasso.fit(X, y) # L2 Regularization (Ridge) ridge = Ridge(alpha=0.1) ridge.fit(X, y) # Elastic Net elastic_net = ElasticNet(alpha=0.05, l1_ratio=0.5) elastic_net.fit(X, y) # Results print("L1 (Lasso):") print(f" Coefficient: {lasso.coef_[0]:.4f}") print(f" Intercept: {lasso.intercept_:.4f}") print("\nL2 (Ridge):") print(f" Coefficient: {ridge.coef_[0]:.4f}") print(f" Intercept: {ridge.intercept_:.4f}") print("\nElastic Net:") print(f" Coefficient: {elastic_net.coef_[0]:.4f}") print(f" Intercept: {elastic_net.intercept_:.4f}")
