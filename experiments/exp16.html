Elastic Net Regularization

🎯 Aim

To implement the Elastic Net algorithm which combines the advantages of both L1 (sparsity) and L2 (stability) regularizations.

📐 Mathematical Procedure
Part A: Objective Function
\( J(w) = \frac{1}{n} \text{SSE}(w) + \lambda_1 |w| + \lambda_2 w^2 \)
Part B: Update Step
\( w_{new} = w_{old} - \eta ( \nabla \text{SSE} + \lambda_1 \text{sign}(w) + 2\lambda_2 w ) \)
