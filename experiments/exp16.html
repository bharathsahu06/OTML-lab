Experiment 14: Linear Regression with L1, L2, and Elastic Net Regularization

1) Aim
Implement linear regression model: $$y = w_1x_1 + w_2x_2$$ with:
  • L1 regularization, $\lambda=0.1$
  • L2 regularization, $\lambda=0.1$
  • Elastic Net regularization, $\lambda_1=0.1$, $\lambda_2=0.2$
2) Description

Regularization Techniques (L1, L2, and their combination) are essential tools for controlling overfitting in machine learning. L1 regularization encourages sparsity through absolute value penalties, L2 regularization smooths weights through quadratic penalties, and Elastic Net combines both to balance their strengths. Choosing the right regularization reveals trade-offs: L1 selects few important features, L2 uses all features uniformly, Elastic Net compromises between these extremes.

The core principle is adding penalty terms to the loss function that penalize large parameter values. The regularization strength $\lambda$ controls this trade-off: higher $\lambda$ prefers smaller/sparser weights, lower $\lambda$ allows larger weights to fit data more closely. L1's non-smoothness creates exact zeros, enabling feature selection. L2's smoothness encourages cooperation among correlated features. Elastic Net adjusts both simultaneously to handle different problem structures.

Understanding regularization is critical for practical machine learning. These techniques directly address the bias-variance trade-off: they introduce bias (pushing toward smaller weights) to reduce variance (instability from overfitting). Modern machine learning almost always includes regularization, making these techniques foundational knowledge.

3) Important Formulas

Model: $$\hat{y}_i = w_1x_{1,i} + w_2x_{2,i}$$

MSE: $$J_{\text{mse}} = \frac{1}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)^2$$

L1 (Lasso)

$$J_{L1}=J_{\text{mse}} + \lambda\left(|w_1|+|w_2|\right)$$

$$\nabla J_{L1}=\frac{2}{n}X^T(Xw-y)+\lambda\,\operatorname{sign}(w)$$

L2 (Ridge)

$$J_{L2}=J_{\text{mse}} + \lambda(w_1^2+w_2^2)$$

$$\nabla J_{L2}=\frac{2}{n}X^T(Xw-y)+2\lambda w$$

Elastic Net

$$J_{EN}=J_{\text{mse}}+\lambda_1\left(|w_1|+|w_2|\right)+\lambda_2(w_1^2+w_2^2)$$

$$\nabla J_{EN}=\frac{2}{n}X^T(Xw-y)+\lambda_1\operatorname{sign}(w)+2\lambda_2 w$$

Update rule: $$w \leftarrow w - \alpha\nabla J$$

4) Important Functions Used
  • train_regularized(...): trains model using selected regularization.
  • regularized_loss(...): computes total loss for selected method.
  • np.sign(): sub-gradient for L1 term.
5) Algorithm (Step by Step)
  1. Prepare input features $x_1, x_2$ and target $y$.
  2. Initialize weight vector $w=[0,0]^T$.
  3. Compute predictions $\hat{y}=Xw$.
  4. Compute base gradient of MSE.
  5. Add regularization gradient/sub-gradient based on method (L1/L2/Elastic).
  6. Update weights using learning rate.
  7. Repeat for fixed iterations.
  8. Report weights, final loss, and predictions for each method.
6) Mathematical Procedure (Complete Step-by-Step)

Model: $\hat{y}=w_1x_1+w_2x_2$, initialize $w^{(0)}=[0,0]^T$. For first iteration, $\hat{y}=0$, so error vector is $e=-y$.

1. Compute base MSE gradient: $$g_{mse}=\frac{2}{n}X^Te$$
2. For L1 ($\lambda=0.1$): $$g_{L1}=g_{mse}+\lambda\,\text{sign}(w^{(0)})=g_{mse}$$ $$w^{(1)}_{L1}=w^{(0)}-\alpha g_{L1}$$
3. For L2 ($\lambda=0.1$): $$g_{L2}=g_{mse}+2\lambda w^{(0)}=g_{mse}$$ $$w^{(1)}_{L2}=w^{(0)}-\alpha g_{L2}$$
4. For Elastic Net ($\lambda_1=0.1,\lambda_2=0.2$): $$g_{EN}=g_{mse}+\lambda_1\text{sign}(w^{(0)})+2\lambda_2w^{(0)}=g_{mse}$$ $$w^{(1)}_{EN}=w^{(0)}-\alpha g_{EN}$$

5. From second iteration onward, regularization terms differ because $w\neq0$.

6. Repeat updates independently for L1, L2, and Elastic Net until convergence.

7. Compare final weights and losses of all three methods.

7) Given Input
  • $x_1=[1,2,3]$
  • $x_2=[2,1,4]$
  • $y=[5,6,7]$
  • L1 and L2 regularization: $\lambda=0.1$
  • Elastic Net: $\lambda_1=0.1,\;\lambda_2=0.2$
  • Learning rate $\alpha=0.01$, Iterations $=2000$
8) Python Implementation & Output
import numpy as np # Data X = np.array([ [1.0, 2.0], [2.0, 1.0], [3.0, 4.0] ]) y = np.array([5.0, 6.0, 7.0]) alpha = 0.01 iterations = 2000 # Initialize w_l1 = np.zeros(2) w_l2 = np.zeros(2) w_en = np.zeros(2) lam = 0.1 lam1 = 0.1 lam2 = 0.2 n = len(y) for _ in range(iterations): # Common prediction errors e_l1 = X @ w_l1 - y e_l2 = X @ w_l2 - y e_en = X @ w_en - y # ----- L1 ----- grad_l1 = (2/n) * (X.T @ e_l1) + lam * np.sign(w_l1) w_l1 -= alpha * grad_l1 # ----- L2 ----- grad_l2 = (2/n) * (X.T @ e_l2) + 2*lam * w_l2 w_l2 -= alpha * grad_l2 # ----- Elastic ----- grad_en = (2/n) * (X.T @ e_en) + lam1*np.sign(w_en) + 2*lam2*w_en w_en -= alpha * grad_en # Predictions pred_l1 = X @ w_l1 pred_l2 = X @ w_l2 pred_en = X @ w_en # Losses loss_l1 = np.mean((pred_l1 - y)**2) + lam*np.sum(np.abs(w_l1)) loss_l2 = np.mean((pred_l2 - y)**2) + lam*np.sum(w_l2**2) loss_en = np.mean((pred_en - y)**2) + lam1*np.sum(np.abs(w_en)) + lam2*np.sum(w_en**2) # Output print("function: f(x) = w1*x1 + w2*x2") print("data points: (1,2,5), (2,1,6), (3,4,7)") print("L1 -> w:", np.round(w_l1,6), " Loss:", round(loss_l1,6)) print("L2 -> w:", np.round(w_l2,6), " Loss:", round(loss_l2,6)) print("Elastic -> w:", np.round(w_en,6), " Loss:", round(loss_en,6))
