Experiment 16: L1/L2 and Elastic Net Regularization

📌 Aim

To implement linear regression with different regularization techniques:

  • L1 regularization (Lasso): λ = 0.1
  • L2 regularization (Ridge): λ = 0.1
  • Elastic Net (combination): λ₁ = 0.05, λ₂ = 0.05
📝 Description

Three major regularization strategies used in linear regression are L1, L2, and Elastic Net. Regularization changes the objective function by adding a penalty on the weights so the model does not become unnecessarily complex.

These penalties help reduce overfitting and improve generalization, especially when data is noisy or features are correlated. Comparing the three approaches shows that different penalties produce different coefficient patterns and different modeling behavior.

  • L1 regularization encourages sparsity and can remove unimportant features
  • L2 regularization shrinks weights smoothly and often improves numerical stability
  • Elastic Net combines both effects, which is useful when features are correlated
📐 Formulas
L1 Regularization (Lasso):
L(w) = MSE + λ||w||₁
L2 Regularization (Ridge):
L(w) = MSE + λ||w||₂²
Elastic Net:
L(w) = MSE + λ₁||w||₁ + λ₂||w||₂²
📊 Mathematical Procedure

Goal: compare L1, L2, and Elastic Net on the same regression dataset.

Dataset: X has two features (x1, x2) and target y.
L1 (Lasso): optimize MSE + λ||w||₁, encouraging sparse/shrunk coefficients.
L2 (Ridge): optimize MSE + λ||w||₂², smoothing coefficient magnitudes.
Elastic Net: optimize MSE + λ₁||w||₁ + λ₂||w||₂², balancing sparsity and stability.
Comparison: fit all three models and compare (w1, w2, intercept) to observe regularization effects.
💻 Simple Python Program
import numpy as np from sklearn.linear_model import Lasso, Ridge, ElasticNet # Two-feature dataset for comparison X = np.array([[1, 2], [2, 4], [3, 5], [4, 3], [5, 1]]) y = np.array([5, 10, 14, 11, 7]) # L1 Regularization (Lasso) lasso = Lasso(alpha=0.1) lasso.fit(X, y) # L2 Regularization (Ridge) ridge = Ridge(alpha=0.1) ridge.fit(X, y) # Elastic Net elastic_net = ElasticNet(alpha=0.05, l1_ratio=0.5) elastic_net.fit(X, y) # Results print("L1 (Lasso):") print(f" Coefficients: w1={lasso.coef_[0]:.4f}, w2={lasso.coef_[1]:.4f}") print(f" Intercept: {lasso.intercept_:.4f}") print("\nL2 (Ridge):") print(f" Coefficients: w1={ridge.coef_[0]:.4f}, w2={ridge.coef_[1]:.4f}") print(f" Intercept: {ridge.intercept_:.4f}") print("\nElastic Net:") print(f" Coefficients: w1={elastic_net.coef_[0]:.4f}, w2={elastic_net.coef_[1]:.4f}") print(f" Intercept: {elastic_net.intercept_:.4f}")
