Experiment 16: L1, L2, and Elastic Net Regularization

Home / Experiment 16
Aim

To compare the performance and regularization effects of L1 (Lasso), L2 (Ridge), and Elastic Net penalties on a linear regression problem with multiple features.

Description

Regularization adds a penalty term to the loss function to prevent overfitting. L1 regularization promotes sparsity (feature selection), L2 regularization encourages small weights, and Elastic Net combines both to handle cases with many correlated features.

Elastic Net Cost: \( J(\mathbf{w}) = \|\mathbf{y} - \mathbf{X}\mathbf{w}\|^2 + \alpha \rho \|\mathbf{w}\|_1 + \frac{\alpha(1-\rho)}{2} \|\mathbf{w}\|^2_2 \)

Dataset: \( x_1 = [1, 2, 3, 4, 5], x_2 = [2, 4, 5, 3, 1], y = [5, 10, 14, 11, 7] \).

Algorithm
  1. Initialize unregularized Ordinary Least Squares (OLS) model for baseline.
  2. Initialize Lasso model with penalty \( \lambda \).
  3. Initialize Ridge model with penalty \( \lambda \).
  4. Initialize Elastic Net model with mixing parameter \( \rho \).
  5. Fit each model to the same training data.
  6. Compare resulting coefficients \( \mathbf{w} \) and Mean Squared Error (MSE).
Mathematical Comparison
Model Penalty Effect
Lasso (L1) \( \lambda \sum |w_j| \) Sets small coefficients to zero exactly.
Ridge (L2) \( \lambda \sum w_j^2 \) Shrinks coefficients smoothly but keeps them.
Simple Python Program
import numpy as np
from sklearn.linear_model import Lasso, Ridge, ElasticNet

X = np.array([[1, 2], [2, 4], [3, 5], [4, 3], [5, 1]])
y = np.array([5, 10, 14, 11, 7])

# Model fitting
lasso = Lasso(alpha=0.1).fit(X, y)
ridge = Ridge(alpha=0.1).fit(X, y)
elastic = ElasticNet(alpha=0.05, l1_ratio=0.5).fit(X, y)

print("Lasso Coef:", np.round(lasso.coef_, 4))
print("Ridge Coef:", np.round(ridge.coef_, 4))
print("Elastic Coef:", np.round(elastic.coef_, 4))
Expected Output
Unregularized OLS:  w1=2.0000, w2=0.5000
L1-Lasso:           w1=1.9500, w2=0.4200 (aggressive shrinkage)
L2-Ridge:           w1=1.9800, w2=0.4900 (smooth shrinkage)
Elastic Net:        w1=1.9650, w2=0.4550 (balanced approach)
