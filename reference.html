Common Terms
  • Objective Function: The function \( f(\mathbf{x}) \) that we aim to minimize or maximize.
  • Gradient (\( \nabla f \)): The vector of partial derivatives, indicating the direction of steepest increase.
  • Hessian (\( \mathbf{H} \)): The matrix of second-order partial derivatives, representing curvature.
  • Learning Rate (\( \eta \)): A scalar factor controlling the step size in iterative updates.
  • Convergence: The state where an algorithm approaches a stable, optimal solution.
Key Formulas
Gradient Descent: \( \mathbf{x}_{k+1} = \mathbf{x}_k - \eta \nabla f(\mathbf{x}_k) \)
Newton's Update: \( \mathbf{x}_{k+1} = \mathbf{x}_k - \mathbf{H}^{-1} \nabla f(\mathbf{x}_k) \)
MSE Loss: \( J = \frac{1}{n} \sum (y_i - \hat{y}_i)^2 \)
